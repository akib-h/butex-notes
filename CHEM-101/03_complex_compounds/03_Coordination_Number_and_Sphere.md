# 03 — Co-ordination Number & Co-ordination Sphere

> **Last Updated:** April 10, 2025
> **Reference:** Hand notes (Sayim, 2023) | LibreTexts | Wikipedia

---

## 1. Co-ordination Number

### Definition

The **co-ordination number (CN)** of the central metal atom in a complex compound is defined as:

> The **maximum number of ligands** (more precisely, **donor atoms**) that are **directly bonded** to the central metal atom or ion.

**Alternative definition:**
> The number of ligands present in the **co-ordination sphere** of the central metal.

---

### Key Points

- It equals the **number of coordinate bonds** formed by the metal.
- For **monodentate** ligands: CN = number of ligands.
- For **bidentate** ligands: CN = 2 × (number of ligands).
- For **polydentate** ligands: CN = denticity × (number of ligands).
- Common co-ordination numbers: **2, 4, 6** (most common: 6).

---

### Examples with Calculation

**Example 1:**
```
[FeCl₄]⁻

Ligands: 4 × Cl⁻ (each monodentate)
Co-ordination Number = 4
Geometry: Tetrahedral
```

**Example 2:**
```
[Fe(CN)₆]⁴⁻  (2018 exam question)

Ligands: 6 × CN⁻ (each monodentate)
Co-ordination Number of Fe = 6
Geometry: Octahedral
```

**Example 3:**
```
[Co(en)₃]³⁺

Ligands: 3 × en (ethylenediamine, each bidentate)
CN = 3 × 2 = 6
Geometry: Octahedral
```

**Example 4:**
```
[Co(EDTA)]⁻

Ligands: 1 × EDTA (hexadentate)
CN = 1 × 6 = 6
Geometry: Octahedral
```

**Example 5 (from notes):**
```
[FeCl₄]⁻  → CN of Fe³⁺ = 4
[FeCl₆]³⁻ → CN of Fe³⁺ = 6
```

---

### Co-ordination Numbers and Geometries

| Co-ordination Number | Geometry | Example |
|----------------------|----------|---------|
| 2 | Linear | [Ag(NH₃)₂]⁺ |
| 4 | Tetrahedral | [NiCl₄]²⁻ |
| 4 | Square Planar | [Pt(NH₃)₂Cl₂] |
| 6 | Octahedral | [Co(NH₃)₆]³⁺ |
| 8 | Cubic / Dodecahedral | [Mo(CN)₈]⁴⁻ |

> **Rule of thumb:** Greater the size of the metal ion, higher the co-ordination number it can accommodate (more ligands can fit around a larger ion).

---

### Greater Metal Size → Higher Co-ordination Number (from notes)

This is illustrated in the notes:

| Metal Ion | Relative Size | Common CN |
|-----------|---------------|-----------|
| Fe²⁺ | Smaller | 6 |
| Co³⁺ | Medium | 6 |
| Pt⁴⁺ | Larger | 6 |
| La³⁺ | Largest (lanthanide) | 9–12 |

Larger ions have more space around them, so more ligands can coordinate.

## 2. Co-ordination Sphere

### Definition

> In a complex compound, the **central atom** and the **ligands directly attached** to it are together enclosed within a **square bracket `[ ]`**, and this entire unit is called the **co-ordination sphere**.

The co-ordination sphere is the "inner sphere" of the complex.

**Everything outside the square bracket** (counter ions) belongs to the **outer sphere** (ionic atmosphere).

---

### Illustration

```
[Fe(CN)₆]⁴⁻              — coordination sphere is the whole anion
K₄[Fe(CN)₆]              — K⁺ ions are outside (counter ions)

[Co(NH₃)₆]³⁺  Cl₃⁻       — [Co(NH₃)₆]³⁺ is coordination sphere
                             Cl⁻ × 3 is outside

[Cu(H₂O)₆]²⁺             — coordination sphere
```

---

### Diagram of Co-ordination Sphere vs Outer Sphere

```
        ┌──────────────────────────────┐
        │                              │
        │   ┌──────────────────────┐   │
        │   │   CO-ORDINATION      │   │
        │   │       SPHERE         │   │
        │   │                      │   │
        │   │   Central Metal      │   │
        │   │        Co³⁺          │   │
        │   │      ↑↑↑↑↑↑          │   │
        │   │  NH₃ NH₃ NH₃         │   │
        │   │  NH₃ NH₃ NH₃         │   │
        │   │                      │   │
        │   └──────────────────────┘   │
        │           3 Cl⁻              │
        │       (outer sphere)         │
        └──────────────────────────────┘

        [Co(NH₃)₆]Cl₃
```

---

### Properties of the Co-ordination Sphere

| Property | Inner Sphere (Co-ordination Sphere) | Outer Sphere (Counter Ions) |
|----------|-------------------------------------|-----------------------------|
| Bonding | Coordinate (covalent) bonds | Ionic / electrostatic |
| Dissociation | Does **not** dissociate in solution | **Does** dissociate in solution |
| Conductivity | Contributes complex ion | Contributes to ion count |
| Precipitation | **Not** precipitated by AgNO₃ (if Cl⁻ inside) | Precipitated immediately by AgNO₃ |

**Werner's Precipitation Test (historical proof):**

```
CoCl₃·6NH₃  ≡  [Co(NH₃)₆]Cl₃   → 3 Cl⁻ outside → 3 AgCl precipitate
CoCl₃·5NH₃  ≡  [Co(NH₃)₅Cl]Cl₂ → 2 Cl⁻ outside → 2 AgCl precipitate
CoCl₃·4NH₃  ≡  [Co(NH₃)₄Cl₂]Cl → 1 Cl⁻ outside → 1 AgCl precipitate
CoCl₃·3NH₃  ≡  [Co(NH₃)₃Cl₃]   → 0 Cl⁻ outside → 0 AgCl precipitate
```

This proved that the Cl⁻ ions inside the coordination sphere do not dissociate!

```
   Counter ions: 3 Cl⁻

   Coordination sphere: [Co(NO₂)₆]³⁻

   Coordination sphere: [Fe(CN)₆]⁴⁻
   Counter ions: 4 K⁺
```

## 3. Determining the Co-ordination Number: Step-by-Step

### Method

1. Identify the central metal atom.
2. Identify all ligands **inside** the square bracket.
3. Determine the denticity of each ligand.
4. **CN = Σ (denticity × number of that ligand)**

**Worked Example:**
```
[Cr(en)₂(H₂O)Cl]²⁺

Ligands inside bracket:
- 2 × en (ethylenediamine) — bidentate (denticity = 2) → contributes 4
- 1 × H₂O — monodentate (denticity = 1) → contributes 1
- 1 × Cl⁻ — monodentate (denticity = 1) → contributes 1

Co-ordination Number = 4 + 1 + 1 = 6
Geometry: Octahedral
```

---

## 4. Summary

| Term | Definition |
|------|-----------|
| Co-ordination Number | Total donor atoms directly bonded to central metal |
| Co-ordination Sphere | Central metal + ligands inside `[ ]` |
| Counter Ions | Ions outside `[ ]`; dissociate freely in solution |
| Inner Sphere | = Co-ordination sphere |
| Outer Sphere | = Counter ions' region |
