## 🔖 Quick Reference

- **Complex Compound:** Central metal atom surrounded by ligands
- **Ligand:** Ion or neutral molecule donating lone pair to metal
- **Co-ordination Number:** Total number of donor atoms bonded to central metal
- **EAN Formula:** `EAN = Z - charge + electrons from ligands`
- **Werner's Primary Valency** = Oxidation state (ionisable)
- **Werner's Secondary Valency** = Co-ordination number (non-ionisable)
