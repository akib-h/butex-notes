## Introduction

When an acid and a base are combined, they undergo a **neutralization reaction** — one of the most fundamental and important reactions in chemistry. The term "neutralization" comes from the idea that the acidic and basic properties **cancel each other out**, often producing a neutral solution.

However, it is a misconception that neutralization always produces a neutral (pH = 7) solution. The actual pH of the product depends on the **relative strengths** of the acid and base involved.

---

## Definition of Neutralization

**Neutralization** is the chemical reaction between an acid and a base to form a **salt** and, usually, **water**.

### General Equation

```
Acid  +  Base  →  Salt  +  Water
```

From the **Arrhenius perspective:**

```
H⁺(aq)  +  OH⁻(aq)  →  H₂O(l)       ΔH = −57.3 kJ/mol
```

From the **Brønsted-Lowry perspective:**

```
HA  +  B  →  A⁻  +  BH⁺
(acid)  (base)  (conjugate base)  (conjugate acid)
```

> 📌 In neutralization, the acid donates its proton to the base, forming a conjugate pair system and (in aqueous solutions) water.

---

## Types of Neutralization Reactions

### Case 1: Strong Acid + Strong Base → Neutral Salt (pH = 7)

```
HCl(aq)  +  NaOH(aq)  →  NaCl(aq)  +  H₂O(l)
HNO₃(aq) +  KOH(aq)   →  KNO₃(aq)  +  H₂O(l)
H₂SO₄(aq)+  2NaOH(aq) →  Na₂SO₄(aq)+  2H₂O(l)
```

- The salt formed (e.g., NaCl) does **not hydrolyse** — it has no acidic/basic character
- Final solution is **neutral (pH ≈ 7)** at equivalence point
- Reaction is highly **exothermic**

**Net ionic equation:**

```
H⁺(aq)  +  OH⁻(aq)  →  H₂O(l)     ΔH = −57.3 kJ/mol
```

---

### Case 2: Strong Acid + Weak Base → Acidic Salt (pH < 7)

```
HCl(aq)  +  NH₃(aq)   →  NH₄Cl(aq)
HCl(aq)  +  NH₄OH(aq) →  NH₄Cl(aq) + H₂O(l)
H₂SO₄   +  2NH₃(aq)  →  (NH₄)₂SO₄
```

- The **NH₄⁺ cation** (conjugate acid of weak base NH₃) **undergoes hydrolysis**:

```
NH₄⁺(aq)  +  H₂O(l)  ⇌  NH₃(aq)  +  H₃O⁺(aq)
```

- Final solution is **acidic (pH < 7)**
- Enthalpy of neutralization is **less exothermic** than strong + strong

---

### Case 3: Weak Acid + Strong Base → Basic Salt (pH > 7)

```
CH₃COOH(aq) + NaOH(aq) → CH₃COONa(aq) + H₂O(l)
HF(aq)       + NaOH(aq) → NaF(aq)       + H₂O(l)
HCN(aq)      + KOH(aq)  → KCN(aq)       + H₂O(l)
```

- The **acetate (CH₃COO⁻) anion** (conjugate base of acetic acid) hydrolyses:

```
CH₃COO⁻(aq)  +  H₂O(l)  ⇌  CH₃COOH(aq)  +  OH⁻(aq)
```

- Final solution is **basic (pH > 7)**
- Enthalpy of neutralization is **slightly less exothermic** (energy consumed in ionizing weak acid)

---

### Case 4: Weak Acid + Weak Base → Variable pH Salt

```
CH₃COOH(aq)  +  NH₃(aq)  →  CH₃COONH₄(aq)
HCN(aq)      +  NH₃(aq)  →  NH₄CN(aq)
```

- Both ions can hydrolyse, and final pH depends on **Ka vs Kb**:
  - If **Ka > Kb**: solution is **acidic**
  - If **Ka < Kb**: solution is **basic**  
  - If **Ka = Kb**: solution is **neutral** (pH ≈ 7)

For CH₃COONH₄:

```
Ka(CH₃COOH) = 1.8 × 10⁻⁵
Kb(NH₃)     = 1.8 × 10⁻⁵

Since Ka = Kb → solution is approximately neutral (pH ≈ 7)
```

---

### Case 5: Acid + Metal Carbonate/Bicarbonate

```
2HCl  +  Na₂CO₃   →  2NaCl  +  H₂O  +  CO₂↑
HCl   +  NaHCO₃   →  NaCl   +  H₂O  +  CO₂↑
```

These are neutralization reactions where CO₂ gas is produced — used in baking (CO₂ causes dough to rise).

---

### Case 6: Acid + Metal Oxide (Lewis Neutralization)

```
CaO  +  H₂SO₄  →  CaSO₄  +  H₂O
MgO  +  2HCl   →  MgCl₂  +  H₂O
Na₂O +  H₂O    →  2NaOH
```

---

### Case 7: Non-Aqueous Neutralization (Lux-Flood)

```
CaO  +  SO₃  →  CaSO₄    (in high-temperature melt, no water)
```

## Net Ionic Equations

For a strong acid + strong base reaction, we can write:

**Full molecular equation:**

```
NaOH(aq) + HCl(aq) → NaCl(aq) + H₂O(l)
```

**Full ionic equation:**

```
Na⁺(aq) + OH⁻(aq) + H⁺(aq) + Cl⁻(aq) → Na⁺(aq) + Cl⁻(aq) + H₂O(l)
```

**Net ionic equation** (spectator ions Na⁺ and Cl⁻ removed):

```
H⁺(aq)  +  OH⁻(aq)  →  H₂O(l)
```

> 📌 The net ionic equation for **all strong acid + strong base** reactions is the same: `H⁺ + OH⁻ → H₂O`

## Enthalpy of Neutralization

### Definition

The **standard enthalpy of neutralization (ΔH°_neut)** is the **heat released** when one mole of water is formed from the neutralization of an acid and a base under standard conditions (298 K, 1 atm).

```
ΔH°_neut = Energy released per mole of H₂O formed
```

> Neutralization reactions are always **exothermic** (heat is released), so ΔH is always **negative**.

### For Strong Acid + Strong Base

Since both are fully ionised, the only reaction is:

```
H⁺(aq)  +  OH⁻(aq)  →  H₂O(l)      ΔH = −57.3 kJ/mol
```

This value is **constant** for all strong acid + strong base combinations, because the net reaction is always the same.

**Examples:**

| Reaction | ΔH (kJ/mol) |
|----------|------------|
| HCl + NaOH | −57.9 |
| HNO₃ + NaOH | −57.6 |
| HClO₄ + NaOH | −57.1 |
| H₂SO₄ + 2NaOH | −114.6 (2 × 57.3) |

### For Weak Acid + Strong Base

Some of the heat must be used to **ionise the weak acid**. Therefore:

```
ΔH_neut(weak acid) = ΔH_neut(strong) + ΔH_dissociation(acid)
```

Since ΔH_dissociation > 0 (endothermic), the total is **less exothermic**:

| Reaction | ΔH (kJ/mol) |
|----------|------------|
| HCl + NaOH (strong + strong) | −57.9 |
| CH₃COOH + NaOH | −55.2 |
| HF + NaOH | −68.6 (unusually high due to H-bond breaking) |
| HCN + KOH | −11.7 (very weak acid) |

### For Weak Acid + Weak Base

The enthalpy is further reduced because energy is consumed in ionising **both** the weak acid and the weak base:

```
CH₃COOH + NH₃  →  CH₃COONH₄      ΔH ≈ −50 kJ/mol (approximately)
```

### Enthalpy of Neutralization — Diagram

```
Strong acid + Strong base:
  ΔH = −57.3 kJ/mol  ████████████████████████████

Weak acid + Strong base:
  ΔH = −55.2 kJ/mol  ███████████████████████████

Very weak acid + Strong base:
  ΔH = −11.7 kJ/mol  ██████

Weak acid + Weak base:
  ΔH ≈ −50 kJ/mol    █████████████████████████
```

## Calculations — Worked Examples

### Calculation 1: Heat Released in Neutralization

**Given:** 50.0 mL of 1.0 M HCl is mixed with 50.0 mL of 1.0 M NaOH. The temperature rises from 22.5°C to 29.3°C. Calculate the enthalpy of neutralization.

**Step 1: Calculate heat evolved (q)**

```
q = m × c × ΔT
m = 100.0 g (assuming density = 1 g/mL for dilute solution)
c = 4.18 J g⁻¹°C⁻¹ (specific heat of water)
ΔT = 29.3 − 22.5 = 6.8°C

q = 100.0 × 4.18 × 6.8 = 2842.4 J = 2.842 kJ
```

**Step 2: Calculate moles of water formed**

```
n(HCl) = 0.0500 L × 1.0 mol/L = 0.050 mol
n(NaOH) = 0.0500 L × 1.0 mol/L = 0.050 mol
n(H₂O) formed = 0.050 mol (1:1 ratio)
```

**Step 3: Calculate molar enthalpy**

```
ΔH = −q / n(H₂O)
ΔH = −2.842 kJ / 0.050 mol = −56.8 kJ/mol

(close to theoretical −57.3 kJ/mol)
```

---

### Calculation 2: Using Standard Enthalpies of Formation

The enthalpy of neutralization can be calculated using Hess's Law:

```
HCl(aq) + NaOH(aq) → NaCl(aq) + H₂O(l)

ΔH = ΣΔH°f(products) − ΣΔH°f(reactants)
   = [ΔH°f(NaCl(aq)) + ΔH°f(H₂O(l))] − [ΔH°f(HCl(aq)) + ΔH°f(NaOH(aq))]
   = [−407.3 + (−285.8)] − [(−167.2) + (−469.6)]
   = −693.1 − (−636.8)
   = −56.3 kJ/mol
```

---

### Calculation 3: Enthalpy of Neutralization (Weak Acid)

For acetic acid + NaOH:

```
CH₃COOH(aq) + NaOH(aq) → CH₃COONa(aq) + H₂O(l)

Experimental ΔH = −55.2 kJ/mol

Energy needed to ionize acetic acid:
ΔH_ionization = ΔH(strong-strong) − ΔH(weak-strong)
               = −57.3 − (−55.2) = −2.1 kJ/mol

∴ About 2.1 kJ/mol of the heat is "used up" in ionizing the acetic acid.
```

---

### Calculation 4: Finding Volume for Neutralization

**Problem:** What volume of 0.250 M NaOH is required to neutralise 30.0 mL of 0.150 M H₂SO₄?

```
H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O

Moles of H₂SO₄ = 0.0300 L × 0.150 mol/L = 0.00450 mol
Moles of NaOH needed = 2 × 0.00450 = 0.00900 mol

Volume of NaOH = 0.00900 mol / 0.250 mol/L = 0.0360 L = 36.0 mL
```

## Titration and Equivalence Point

**Titration** is the process of carefully adding a solution of known concentration (the **titrant**) to a solution of unknown concentration until neutralization is complete.

The **equivalence point** is the point at which the moles of acid exactly equal the moles of base (or stoichiometrically equivalent amounts have been added).

### Titration Curves

**Strong Acid + Strong Base** (e.g., HCl titrated with NaOH):

```
pH
14 |                          ┌─────────────────
   |                         /
12 |                        /
10 |                       /
 8 |              Equivalence Point (pH ≈ 7)
 7 |·····················●·····················
 6 |                    /
 4 |                   /
 2 |──────────────────/
 0 +──────────────────────────────────────────→
                  Volume NaOH added
```

- pH at equivalence point = **7**
- Curve shows a sharp, vertical jump at equivalence point

**Weak Acid + Strong Base** (e.g., CH₃COOH titrated with NaOH):

```
pH
14 |                          ┌─────────────────
   |                        /
12 |                      /
10 |              /───────/
 9 |      ●      (Buffer region)
 7 |            ●  Equivalence Point (pH ≈ 9)
 5 |──────/
 2 |
 0 +──────────────────────────────────────────→
           Volume NaOH added
```

- pH at equivalence point > 7 (basic, because of CH₃COO⁻ hydrolysis)
- There is a **buffer region** (flat part) before equivalence

**Strong Acid + Weak Base** (e.g., HCl titrated with NH₃):

- pH at equivalence point < 7 (acidic, because of NH₄⁺ hydrolysis)

## Acid/Base Indicators

An **indicator** is a weak acid (HIn) that changes colour depending on pH:

```
HIn  ⇌  H⁺  +  In⁻
(colour 1)      (colour 2)
```

| Indicator | pH Range | Colour (Acid) | Colour (Base) |
|-----------|---------|---------------|---------------|
| Methyl Orange | 3.1 – 4.4 | Red | Yellow |
| Methyl Red | 4.4 – 6.2 | Red | Yellow |
| Bromothymol Blue | 6.0 – 7.6 | Yellow | Blue |
| Phenolphthalein | 8.2 – 10.0 | Colourless | Pink |
| Alizarin Yellow | 10.1 – 12.0 | Yellow | Red |

**Choosing an Indicator:**

| Titration Type | pH at Equivalence | Suitable Indicator |
|---------------|------------------|-------------------|
| Strong acid + Strong base | ~7 | Bromothymol blue, Methyl orange, Phenolphthalein |
| Weak acid + Strong base | ~9 | Phenolphthalein |
| Strong acid + Weak base | ~5 | Methyl orange, Methyl red |
| Weak acid + Weak base | Variable | Not straightforward; universal indicator |

## Real-World Applications

### 1. Antacids

Stomach acid (HCl, pH ~2) is neutralised by antacids containing bases like:

```
Mg(OH)₂  +  2HCl  →  MgCl₂  +  2H₂O   (Milk of Magnesia)
CaCO₃    +  2HCl  →  CaCl₂  +  H₂O + CO₂↑  (Tums)
NaHCO₃   +  HCl   →  NaCl   +  H₂O + CO₂↑  (Baking Soda)
Al(OH)₃  +  3HCl  →  AlCl₃  +  3H₂O   (Maalox)
```

### 2. Wastewater Treatment

Industrial acidic/basic effluents must be neutralised before release:

```
Ca(OH)₂  +  H₂SO₄  →  CaSO₄↓  +  2H₂O
```

Lime (CaO or Ca(OH)₂) is commonly used to neutralise acidic wastewater.

### 3. Agriculture — Soil pH Correction

Acidic soils are treated with **lime (CaO or CaCO₃)**:

```
CaO  +  H₂O  →  Ca(OH)₂  (quick lime slaking)
Ca(OH)₂ neutralises soil acids to raise pH
```

### 4. Food Industry — Baking

```
NaHCO₃  +  CH₃COOH  →  CH₃COONa  +  H₂O  +  CO₂↑
(baking soda)  (vinegar)               (CO₂ causes rise)
```

### 5. Blood Buffering

Blood pH is maintained at 7.35–7.45 using bicarbonate buffering:

```
H⁺  +  HCO₃⁻  ⇌  H₂CO₃  ⇌  H₂O  +  CO₂
OH⁻ +  H₂CO₃  ⇌  HCO₃⁻  +  H₂O
```

### 6. Manufacture of Salts

Many commercially important salts are produced by neutralization:

```
HNO₃ + NH₃ → NH₄NO₃         (ammonium nitrate — fertiliser)
H₂SO₄ + 2NH₃ → (NH₄)₂SO₄   (ammonium sulphate — fertiliser)
H₃PO₄ + 3NaOH → Na₃PO₄ + 3H₂O
```

## Summary of Salt Types by Neutralization

| Acid | Base | Salt | Salt pH | Example |
|------|------|------|---------|---------|
| Strong | Strong | Neutral | = 7 | NaCl, KNO₃ |
| Strong | Weak | Acidic | < 7 | NH₄Cl, (NH₄)₂SO₄ |
| Weak | Strong | Basic | > 7 | CH₃COONa, NaCN, NaF |
| Weak | Weak | Depends on Ka/Kb | ≈ 7, < 7 or > 7 | CH₃COONH₄ |

## Practice Problems

**Problem 1:** Write a balanced equation for the neutralization of phosphoric acid (H₃PO₄) with calcium hydroxide (Ca(OH)₂). What type of salt is formed?

**Answer:**

```
2H₃PO₄  +  3Ca(OH)₂  →  Ca₃(PO₄)₂  +  6H₂O
```

Salt: Ca₃(PO₄)₂ — formed from a weak acid and a strong base → basic salt (solution pH > 7).

---

**Problem 2:** A student mixes 100 mL of 2.0 M HCl with 100 mL of 2.0 M NaOH. The temperature rises by 13.6°C. Calculate ΔH_neutralization in kJ/mol.

```
q = m × c × ΔT = 200 g × 4.18 J/g°C × 13.6°C = 11,370 J = 11.37 kJ
n(H₂O) = 0.100 L × 2.0 mol/L = 0.200 mol
ΔH = −11.37 / 0.200 = −56.85 kJ/mol
```

---

**Problem 3:** Predict the pH (acidic, basic, neutral) of aqueous solutions of the following salts:

1. `NaCl` → Neutral (strong acid + strong base)
2. `NH₄Cl` → Acidic (strong acid + weak base)
3. `CH₃COONa` → Basic (weak acid + strong base)
4. `CH₃COONH₄` → ~Neutral (Ka ≈ Kb)
5. `KCN` → Basic (strong base + very weak acid HCN)
