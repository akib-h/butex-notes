## Introduction

Oxides, hydroxides, and salts are the most common inorganic compounds. Their acid-base behaviour:

- Depends on the **nature of the central atom** (metallic vs. non-metallic)
- Is influenced by the **oxidation state** and **electronegativity** of the atom
- Can be predicted from the **position in the periodic table**

A unifying principle: **If the M–O bond is more ionic → compound is basic; if the M–O bond is more covalent → compound is acidic.**

---

## Acid-Base Properties of Oxides

**Classification of Oxides by Acid-Base Character:**

```
Basic Oxides        Amphoteric Oxides       Acidic Oxides       Neutral Oxides
─────────────        ─────────────────       ─────────────        ─────────────
Na₂O, K₂O           Al₂O₃, ZnO             CO₂, SO₂, SO₃        CO, NO, N₂O
MgO, CaO            SnO, PbO               NO₂, P₄O₁₀            (no salt formed)
FeO, CuO            Sb₂O₃, Bi₂O₃          Cl₂O₇, Mn₂O₇
(metal oxides)       (metalloid oxides)     (nonmetal oxides)
```

## 1. Basic Oxides (Metallic Oxides)

### Definition

**Basic oxides** are metallic oxides that:
- React with water to form **metal hydroxides (bases)**
- React with acids to form **salts and water**
- **Do not react with bases**

### Formation

Basic oxides result from **ionic M–O bonds** in Group 1 and Group 2 metals — the oxygen carries the oxide ion O²⁻, which is a strong Brønsted base.

### Reactions

**With Water (to form base):**

```
Na₂O  +  H₂O  →  2NaOH            (sodium oxide → sodium hydroxide)
K₂O   +  H₂O  →  2KOH             (potassium oxide → potassium hydroxide)
CaO   +  H₂O  →  Ca(OH)₂          (quicklime → slaked lime)
BaO   +  H₂O  →  Ba(OH)₂          (barium oxide → barium hydroxide)
MgO   +  H₂O  →  Mg(OH)₂   (slow/limited; Mg(OH)₂ is sparingly soluble → pH ≈ 9)
```

**With Acids (to form salts):**

```
MgO   +  2HCl  →  MgCl₂  +  H₂O
CaO   +  H₂SO₄ →  CaSO₄  +  H₂O
Na₂O  +  H₂SO₄ →  Na₂SO₄ +  H₂O
CuO   +  2HNO₃ →  Cu(NO₃)₂ + H₂O
FeO   +  H₂SO₄ →  FeSO₄  +  H₂O
```

**Note:** Basic oxides do NOT react with other bases.

### Group 1 Oxides: Special Products

Group 1 metals can form different oxide types depending on conditions:

| Oxide Type | Formula | Example | Formed when |
|-----------|---------|---------|------------|
| Normal oxide | M₂O | Li₂O, Na₂O | O²⁻ |
| Peroxide | M₂O₂ | Na₂O₂ | O₂²⁻ |
| Superoxide | MO₂ | KO₂, RbO₂, CsO₂ | O₂⁻ |

```
4Li  +  O₂  →  2Li₂O    (normal oxide)
2Na  +  O₂  →  Na₂O₂   (peroxide)
K   +  O₂   →  KO₂     (superoxide)
```

## 2. Acidic Oxides (Non-Metallic Oxides)

### Definition

**Acidic oxides** (also called **acid anhydrides**) are non-metallic oxides that:
- React with water to form **acids**
- React with bases to form **salts and water**
- Do not react with acids

### Mechanism

Non-metals form **covalent M–O bonds** where the electronegative oxygen carries a partial negative charge. When these oxides dissolve in water, the oxygen reacts as a base to form oxyacids.

### Reactions

**With Water (to form acid):**

```
CO₂   +  H₂O  →  H₂CO₃              (carbonic acid, pKa = 6.37)
SO₂   +  H₂O  →  H₂SO₃              (sulfurous acid, pKa = 1.81)
SO₃   +  H₂O  →  H₂SO₄              (sulfuric acid — strong)
NO₂   +  H₂O  →  HNO₃ + HNO₂       (nitric + nitrous acid)
P₄O₁₀ + 6H₂O →  4H₃PO₄             (phosphoric acid, pKa₁ = 2.15)
Cl₂O₇ + H₂O  →  2HClO₄             (perchloric acid — very strong)
SiO₂  (no reaction with H₂O due to network structure)
```

**With Bases (to form salts):**

```
CO₂   +  2NaOH  →  Na₂CO₃  +  H₂O
SO₂   +  2NaOH  →  Na₂SO₃  +  H₂O
SO₃   +  2KOH   →  K₂SO₄   +  H₂O
SiO₂  +  2NaOH  →  Na₂SiO₃ +  H₂O     (requires hot concentrated NaOH)
P₂O₅  +  6NaOH  →  2Na₃PO₄ +  3H₂O
```

### Acidic Oxides by Group

| Group | Element | Oxide | Acid Formed | Strength |
|-------|---------|-------|------------|---------|
| 14 | C | CO₂ | H₂CO₃ | Weak |
| 14 | Si | SiO₂ | H₂SiO₃ | Very weak |
| 15 | N | N₂O₃ | HNO₂ | Weak |
| 15 | N | N₂O₅ | HNO₃ | Strong |
| 15 | P | P₄O₁₀ | H₃PO₄ | Moderate |
| 16 | S | SO₂ | H₂SO₃ | Moderate |
| 16 | S | SO₃ | H₂SO₄ | Strong |
| 17 | Cl | Cl₂O | HClO | Weak |
| 17 | Cl | Cl₂O₇ | HClO₄ | Very strong |
| 25 | Mn | Mn₂O₇ | HMnO₄ | Strong |
| 24 | Cr | CrO₃ | H₂CrO₄ | Strong |

### Lux-Flood Reactions (High Temperature)

Acidic and basic oxides can react directly without water (relevant in metallurgy and geology):

```
CaO    +  CO₂  →  CaCO₃     (CaO = base, CO₂ = acid)
CaO    +  SiO₂ →  CaSiO₃    (slag formation in blast furnace)
MgO    +  SO₃  →  MgSO₄
Na₂O   +  P₂O₅ →  2Na₃PO₄
```

## 3. Amphoteric Oxides

### Definition

**Amphoteric oxides** can react with **both acids and bases** to form salts and water. They sit at the boundary between metallic and non-metallic character — typically **metalloids** and some transition metals in intermediate oxidation states.

### Examples and Reactions

**Aluminum Oxide (Al₂O₃) — the most common example:**

```
As a BASE (reacts with acid):
Al₂O₃  +  3H₂SO₄  →  Al₂(SO₄)₃  +  3H₂O

As an ACID (reacts with base):
Al₂O₃  +  2NaOH(conc.)  +  3H₂O  →  2NaAl(OH)₄   (sodium tetrahydroxoaluminate)
```

**Zinc Oxide (ZnO):**

```
ZnO  +  2HCl   →  ZnCl₂   +  H₂O    (basic behaviour)
ZnO  +  2NaOH  →  Na₂ZnO₂ + H₂O    (acidic behaviour → sodium zincate)
```

**Tin(II) Oxide (SnO):**

```
SnO  +  2HCl   →  SnCl₂  +  H₂O    (basic)
SnO  +  2NaOH  →  Na₂SnO₂ + H₂O   (acidic → sodium stannite)
```

**Lead(II) Oxide (PbO):**

```
PbO  +  2HNO₃  →  Pb(NO₃)₂ + H₂O   (basic)
PbO  +  NaOH   →  NaPbO₂  + H₂O    (acidic → sodium plumbite)
```

**Beryllium Oxide (BeO):**

```
BeO  +  H₂SO₄  →  BeSO₄  +  H₂O    (basic)
BeO  +  2NaOH  →  Na₂BeO₂ + H₂O    (acidic)
```

**Chromium(III) Oxide (Cr₂O₃):**

```
Cr₂O₃  +  3H₂SO₄  →  Cr₂(SO₄)₃  +  3H₂O    (basic)
Cr₂O₃  +  2NaOH(fused) →  2NaCrO₂  +  H₂O   (acidic → chromite)
```

### Other Amphoteric Oxides

| Oxide | Metal | Notes |
|-------|-------|-------|
| Al₂O₃ | Al | Most classic example |
| ZnO | Zn | Common example |
| SnO, SnO₂ | Sn | Tin (II and IV) |
| PbO, PbO₂ | Pb | Lead (II and IV) |
| Cr₂O₃ | Cr(III) | Chromium(III) oxide |
| Fe₂O₃ | Fe(III) | Feebly amphoteric |
| MnO₂ | Mn(IV) | Very weakly amphoteric |
| As₂O₃ | As | Arsenic(III) oxide |
| Sb₂O₃ | Sb | Antimony(III) oxide |
| Bi₂O₃ | Bi | Mostly basic |

---

## 4. Neutral Oxides

**Neutral oxides** react with neither acids nor bases under normal conditions — they form no salts.

| Oxide | Notes |
|-------|-------|
| CO (carbon monoxide) | Does not react with H₂O, acids, or bases under normal conditions |
| NO (nitric oxide) | Neutral, radical species |
| N₂O (nitrous oxide) | "Laughing gas" — neutral oxide |
| NO (nitric oxide) | Radical, does not form salts |

```
CO + H₂O → does not occur under normal conditions
NO + NaOH → no reaction
```

## Periodic Trends in Oxide Acid-Base Character

### Across a Period (Left to Right)

Going from left to right across Period 3:

| Oxide | Na₂O | MgO | Al₂O₃ | SiO₂ | P₄O₁₀ | SO₃ | Cl₂O₇ |
|-------|------|-----|-------|------|-------|-----|-------|
| **Character** | Basic | Basic | Amphoteric | Weakly acidic | Acidic | Acidic | Very acidic |
| Reaction with water | NaOH | Mg(OH)₂ | No direct rxn | No rxn | H₃PO₄ | H₂SO₄ | HClO₄ |

```
Na₂O  MgO  Al₂O₃  SiO₂   P₄O₁₀  SO₃  Cl₂O₇
 ←────────────────────────────────────→
  Basic                              Acidic
(ionic bonding)               (covalent bonding)
```

**Reason:** As electronegativity increases left to right, the M–O bond becomes more **covalent**, favouring acidic character.

### Down a Group (Increasing Atomic Number)

For the same group, higher oxidation states tend to be **more acidic**:

```
Cr(II) oxide  CrO    →  mostly basic
Cr(III) oxide Cr₂O₃  →  amphoteric
Cr(VI) oxide  CrO₃   →  acidic (chromic acid precursor)
```

Also, moving down a group the character can shift:

```
Group 14:
CO₂ (acidic) → SiO₂ (weakly acidic) → SnO₂ (amphoteric) → PbO₂ (amphoteric)

Group 15:
N₂O₅ (strongly acidic) → P₂O₅ (acidic) → As₂O₃ (amphoteric) → Bi₂O₃ (basic)
```

## Acid-Base Properties of Hydroxides

### Metal Hydroxides (Bases)

Metal hydroxides are typically **bases**. Their basicity depends on the **strength of the M–OH bond**:

- If M–OH is easily broken → releases OH⁻ → **strong base** (Group 1)
- If M–OH is harder to break → **weak base** (Group 2 and higher)
- If OH is more easily lost as H⁺ than OH⁻ → **acidic hydroxide** (nonmetals)

**Strong Base Hydroxides:**

```
NaOH → Na⁺ + OH⁻    (complete dissociation)
KOH  → K⁺  + OH⁻    (complete dissociation)
Ba(OH)₂ → Ba²⁺ + 2OH⁻
```

**Weak Base Hydroxides:**

```
Mg(OH)₂ ⇌ Mg²⁺ + 2OH⁻    (sparingly soluble, Ksp = 5.61 × 10⁻¹²)
Ca(OH)₂ ⇌ Ca²⁺ + 2OH⁻    (sparingly soluble, Ksp = 4.68 × 10⁻⁶)
Fe(OH)₃ ⇌ Fe³⁺ + 3OH⁻    (very insoluble)
Al(OH)₃ ⇌ Al³⁺ + 3OH⁻    (very insoluble; amphoteric)
```

### Amphoteric Hydroxides

Al(OH)₃, Zn(OH)₂, Sn(OH)₂, Pb(OH)₂, and Cr(OH)₃ are **amphoteric**:

**Aluminium hydroxide Al(OH)₃:**

```
Al(OH)₃ acts as BASE:
  Al(OH)₃  +  3HCl  →  AlCl₃  +  3H₂O

Al(OH)₃ acts as ACID:
  Al(OH)₃  +  NaOH  →  NaAl(OH)₄    [Al(OH)₄]⁻ = tetrahydroxoaluminate

The OH in Al(OH)₃ can lose H⁺:
  Al(OH)₃  →  H⁺  +  [Al(OH)₄]⁻    (acting as weak acid, pKa ≈ 12.4)
```

**Zinc hydroxide Zn(OH)₂:**

```
Zn(OH)₂ + 2HCl    → ZnCl₂    + 2H₂O   (base)
Zn(OH)₂ + 2NaOH   → Na₂[Zn(OH)₄]      (acid → zincate)
```

### Why Amphoteric?

The balance between:
- Losing OH⁻ → basic character (ionic M–OH, or M with low charge density)
- Gaining OH⁻ (or losing H⁺ from OH) → acidic character (covalent M–OH, or M with high charge density)

Metals at the "diagonal" of the periodic table (Al, Be, Zn, Ge, Sn, Pb, Sb, As) typically have amphoteric hydroxides.

## Acid-Base Properties of Salts — Hydrolysis

**Salt hydrolysis** is the reaction of a salt's ions with water. The pH of a salt solution depends on the strengths of the acid and base that formed it.

### General Rule

| Salt type | Cation | Anion | Solution pH | Example |
|-----------|--------|-------|------------|---------|
| Strong acid + Strong base | neutral | neutral | = 7 | NaCl, KNO₃ |
| Weak acid + Strong base | neutral | basic (hydrolyses) | > 7 | CH₃COONa |
| Strong acid + Weak base | acidic (hydrolyses) | neutral | < 7 | NH₄Cl |
| Weak acid + Weak base | both hydrolyse | both hydrolyse | depends on Ka/Kb | CH₃COONH₄ |

### Case 1: Salt of Strong Acid + Strong Base → pH = 7

**Example: NaCl**

```
NaCl → Na⁺ + Cl⁻

Na⁺: conjugate acid of NaOH (strong base) → does NOT hydrolyse
Cl⁻: conjugate base of HCl (strong acid) → does NOT hydrolyse

Solution is NEUTRAL (pH = 7)
```

Other examples: KNO₃, KBr, NaClO₄

### Case 2: Salt of Weak Acid + Strong Base → pH > 7

**Example: Sodium acetate (CH₃COONa)**

```
CH₃COONa → CH₃COO⁻ + Na⁺

Na⁺: does not hydrolyse (spectator)
CH₃COO⁻ (conjugate base of weak acid): DOES hydrolyse:

CH₃COO⁻ + H₂O ⇌ CH₃COOH + OH⁻

This produces OH⁻ → solution is BASIC (pH > 7)

Kb(CH₃COO⁻) = Kw/Ka = 1.0×10⁻¹⁴ / 1.8×10⁻⁵ = 5.6×10⁻¹⁰
```

Other examples: NaF, NaCN, NaHCO₃, Na₂CO₃, K₃PO₄

### Case 3: Salt of Strong Acid + Weak Base → pH < 7

**Example: Ammonium chloride (NH₄Cl)**

```
NH₄Cl → NH₄⁺ + Cl⁻

Cl⁻: does not hydrolyse (spectator)
NH₄⁺ (conjugate acid of weak base NH₃): DOES hydrolyse:

NH₄⁺ + H₂O ⇌ NH₃ + H₃O⁺

This produces H₃O⁺ → solution is ACIDIC (pH < 7)

Ka(NH₄⁺) = Kw/Kb = 1.0×10⁻¹⁴ / 1.8×10⁻⁵ = 5.6×10⁻¹⁰
```

Other examples: (NH₄)₂SO₄, AlCl₃, FeCl₃, CuSO₄

**Note:** Small, highly charged metal cations like Al³⁺, Fe³⁺, Cr³⁺ also hydrolyse:

```
Al³⁺ + 6H₂O ⇌ [Al(H₂O)₅(OH)]²⁺ + H₃O⁺      (pKa ≈ 5)

This is why AlCl₃ solutions are acidic.
```

### Case 4: Salt of Weak Acid + Weak Base → pH depends on Ka vs Kb

**Example: Ammonium acetate (CH₃COONH₄)**

Both ions hydrolyse:

```
NH₄⁺    + H₂O ⇌ NH₃  + H₃O⁺    (Ka = 5.6×10⁻¹⁰)
CH₃COO⁻ + H₂O ⇌ CH₃COOH + OH⁻  (Kb = 5.6×10⁻¹⁰)

Ka(NH₄⁺) = Kb(CH₃COO⁻) → equal hydrolysis → neutral (pH ≈ 7)
```

**General formula for weak acid + weak base salt:**

$$\text{pH} = 7 + \frac{1}{2}(\text{pK}_a - \text{pK}_b)$$

Wait — more precisely:

$$\text{pH} = 7 + \frac{\text{pK}_a(\text{acid}) - \text{pK}_b(\text{base})}{2}$$

For CH₃COONH₄: pH = 7 + (4.74 − 4.74)/2 = 7 ✓

## Calculations for Salt Hydrolysis

### Calculation 1: pH of Weak Acid Salt (CH₃COONa)

**Given:** 0.10 M sodium acetate solution. Ka(CH₃COOH) = 1.8 × 10⁻⁵. Find pH.

```
CH₃COO⁻ + H₂O ⇌ CH₃COOH + OH⁻

Kb = Kw/Ka = 1.0×10⁻¹⁴ / 1.8×10⁻⁵ = 5.56×10⁻¹⁰

[OH⁻] = √(Kb × C) = √(5.56×10⁻¹⁰ × 0.10) = √(5.56×10⁻¹¹)
      = 7.46×10⁻⁶ M

pOH = −log(7.46×10⁻⁶) = 5.13
pH = 14 − 5.13 = 8.87  ✓ (basic, as expected)
```

### Calculation 2: pH of Weak Base Salt (NH₄Cl)

**Given:** 0.10 M ammonium chloride. Kb(NH₃) = 1.8 × 10⁻⁵. Find pH.

```
NH₄⁺ + H₂O ⇌ NH₃ + H₃O⁺

Ka = Kw/Kb = 1.0×10⁻¹⁴ / 1.8×10⁻⁵ = 5.56×10⁻¹⁰

[H₃O⁺] = √(Ka × C) = √(5.56×10⁻¹⁰ × 0.10) = 7.46×10⁻⁶ M

pH = −log(7.46×10⁻⁶) = 5.13  ✓ (acidic, as expected)
```

### Calculation 3: pH of Salt of Weak Acid + Weak Base

**Given:** 0.10 M ammonium cyanide (NH₄CN).  
Ka(HCN) = 6.2 × 10⁻¹⁰, Kb(NH₃) = 1.8 × 10⁻⁵

```
pKa(HCN) = 9.21
pKb(NH₃) = 4.74

pH = 7 + (pKa − pKb)/2 = 7 + (9.21 − 4.74)/2 = 7 + 2.24 = 9.24

Since pKa > pKb → the acid (HCN) is weaker → CN⁻ is a stronger base than NH₄⁺ is an acid → solution is basic (pH > 7) ✓
```

## Degree of Hydrolysis

The **degree of hydrolysis (h)** is the fraction of salt ions that have hydrolysed:

$$h = \frac{[\text{hydrolysed ions}]}{[\text{initial salt}]}$$

For a salt of weak acid + strong base at concentration C:

$$h = \sqrt{\frac{K_h}{C}} = \sqrt{\frac{K_w}{K_a \cdot C}}$$

Where Kh = Kw/Ka is the hydrolysis constant.

**Example:** Degree of hydrolysis of 0.1 M CH₃COONa:

```
Kh = Kw/Ka = 1.0×10⁻¹⁴ / 1.8×10⁻⁵ = 5.56×10⁻¹⁰

h = √(5.56×10⁻¹⁰ / 0.10) = √(5.56×10⁻⁹) = 7.46×10⁻⁵ = 0.00746%
```

Very small degree of hydrolysis, as expected for a moderately weak acid.

## Real-World Applications

### 1. Blood pH and Bicarbonate Buffering

Blood pH (7.35–7.45) is maintained by the **carbonate buffer system**:

```
H₂CO₃  ⇌  H⁺  +  HCO₃⁻       (pKa = 6.37)
HCO₃⁻  ⇌  H⁺  +  CO₃²⁻       (pKa = 10.33)
```

The kidneys and lungs regulate [HCO₃⁻] and pCO₂ to keep pH in range.

### 2. Hard Water and Limestone

```
CaCO₃ (limestone)  +  H₂O  +  CO₂  ⇌  Ca(HCO₃)₂    (hard water)
                                     (calcium bicarbonate, soluble)
```

When heated: `Ca(HCO₃)₂ → CaCO₃↓ + H₂O + CO₂` (limescale/kettle scale)

### 3. Acid Rain and Oxide Reactions

```
SO₂ + H₂O → H₂SO₃    (acid rain component)
NO₂ + H₂O → HNO₃ + HNO₂   (acid rain)
CO₂ + H₂O → H₂CO₃         (natural rain acidity, pH ~ 5.6)
```

### 4. Industrial Use of Amphoteric Al(OH)₃

Aluminum hydroxide is used as:
- **Antacid:** neutralises stomach acid (basic behaviour)
- **Water purification:** flocculation agent
- **Mordant in dyeing:** forms bonds with both fabric and dye
- **Fire retardant:** endothermic decomposition releases water

### 5. Concrete and Portland Cement

Portland cement contains CaO (basic oxide), Al₂O₃ (amphoteric), and SiO₂ (acidic oxide). They react:

```
CaO + SiO₂ → CaSiO₃     (basic + acidic oxide → salt)
CaO + Al₂O₃ → Ca(AlO₂)₂  (basic + amphoteric oxide → calcium aluminate)
```

(a) Na₂O  (b) SO₃  (c) CO  (d) ZnO  (e) Fe₂O₃

```
(a) Na₂O   → Basic (Group 1 metal)
(b) SO₃    → Acidic (non-metal)
(c) CO     → Neutral (no salt with acid or base)
(d) ZnO    → Amphoteric
(e) Fe₂O₃  → Predominantly basic (Fe³⁺, but very weakly amphoteric)
```

**Problem 2:** Write the balanced equation for:  
(a) SO₃ reacting with water  
(b) Al₂O₃ reacting with NaOH  
(c) CaO reacting with CO₂ (Lux-Flood reaction)

```
(a) SO₃ + H₂O → H₂SO₄

(b) Al₂O₃ + 2NaOH + 3H₂O → 2NaAl(OH)₄    (or Al₂O₃ + 2NaOH → 2NaAlO₂ + H₂O in concentrated)

(c) CaO + CO₂ → CaCO₃    (Lux-Flood: O²⁻ donor + O²⁻ acceptor → salt)
```

---

**Problem 3:** Calculate the pH of a 0.050 M solution of KCN.  
Given: Ka(HCN) = 6.2 × 10⁻¹⁰

```
KCN → K⁺ + CN⁻
K⁺ does not hydrolyse. CN⁻ hydrolyses (conjugate base of weak acid HCN):

CN⁻ + H₂O ⇌ HCN + OH⁻

Kb(CN⁻) = Kw/Ka = 1.0×10⁻¹⁴ / 6.2×10⁻¹⁰ = 1.61×10⁻⁵

[OH⁻] = √(Kb × C) = √(1.61×10⁻⁵ × 0.050) = √(8.06×10⁻⁷) = 8.98×10⁻⁴ M

pOH = −log(8.98×10⁻⁴) = 3.05
pH = 14 − 3.05 = 10.95    (basic, as expected for salt of weak acid + strong base)
```

---

**Problem 4:** Identify the type of oxide and write its reaction with water:  
(a) K₂O  (b) N₂O₅  (c) SiO₂

```
(a) K₂O: Basic oxide → K₂O + H₂O → 2KOH
(b) N₂O₅: Acidic oxide → N₂O₅ + H₂O → 2HNO₃
(c) SiO₂: Acidic oxide → SiO₂ does not react directly with H₂O (insoluble network structure)
          But reacts with hot NaOH: SiO₂ + 2NaOH → Na₂SiO₃ + H₂O
```
