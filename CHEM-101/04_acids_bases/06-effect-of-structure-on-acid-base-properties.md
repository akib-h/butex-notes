## Introduction

The acid or base strength of a compound is fundamentally determined by how **stable its conjugate base (or conjugate acid)** is.

> **Core Principle:** The more stable the conjugate base A⁻ (after HA loses H⁺), the weaker that conjugate base is, and therefore the **stronger the acid HA** is.

Equivalently: anything that **stabilises the negative charge** on the conjugate base **increases acidity**.

The structural factors that affect acid-base properties fall into four main categories — often remembered by the acronym **ARIO**:

```
A — Atom / Element effect (electronegativity and atomic size)
R — Resonance (charge delocalisation)
I — Inductive effect (electron withdrawal/donation through σ bonds)
O — Orbital hybridisation (s-character effects)
```

Understanding these factors allows us to **rank and predict** acid/base strengths without needing to look up every pKa.

> 📌 **Remember:** When comparing two acids, compare the **stability of their conjugate bases**. The more stable the conjugate base, the stronger the acid.

---

## The ARIO Approach

### Priority of Effects (General Guide)

1. **Atom effect** — Is the charge on different elements? (Compare atoms first)
2. **Resonance** — Can the charge be delocalised? (Very powerful stabiliser)
3. **Inductive effect** — Are there electronegative groups nearby? (Moderate effect, distance-dependent)
4. **Orbital hybridisation** — What is the s-character of the orbital? (Applies when same atom)

```
Priority: A > R > I > O  (in general, but exceptions exist)
```

However, **resonance effects are generally stronger than inductive effects.**

## 1. Atomic / Element Effect

### A. Same Period — Electronegativity Dominates

Moving across a period (left to right), **electronegativity increases**, which increases the ability to stabilise a negative charge.

**Example: C–H, N–H, O–H bonds in Period 2:**

```
   Ethane      Methylamine    Methanol    HF
   CH₃CH₃        CH₃NH₂       CH₃OH
  pKa ≈ 50      pKa ≈ 40     pKa ≈ 16   pKa = 3.17
```

The conjugate bases are:
- CH₃CH₂⁻: charge on **C** (EN = 2.5) — least stable
- CH₃NH⁻: charge on **N** (EN = 3.0) — intermediate
- CH₃O⁻: charge on **O** (EN = 3.5) — most stable

```
Trend (increasing acid strength):
  Ethane < Methylamine < Methanol < HF
  CH₃CH₃ << CH₃NH₂  << CH₃OH  << HF

  (C < N < O < F in electronegativity → O and F stabilise anions better)
```

**General Rule:** For same-period elements, the more electronegative the atom bearing the negative charge, the more stable the conjugate base → stronger acid.

---

### B. Same Group — Atomic Radius (Size) Dominates

Moving down a group, **atomic size increases**. Although electronegativity decreases, larger atoms can **spread the charge over a larger volume** → more stable anion (lower charge density).

**Example: Haloacids HX (Group 17):**

| Acid | Conjugate Base | Atomic Radius of X | pKa |
|------|---------------|-------------------|-----|
| HF | F⁻ | 64 pm | 3.17 |
| HCl | Cl⁻ | 99 pm | −7 |
| HBr | Br⁻ | 114 pm | −9 |
| HI | I⁻ | 133 pm | −10 |

```
Acidity: HF << HCl << HBr << HI
(increases going DOWN the group despite decreasing electronegativity)
```

**Reason:** The iodide ion (I⁻) has its charge spread over a **much larger volume** than fluoride (F⁻). The lower charge density of I⁻ makes it more stable. This **size effect overrides the electronegativity effect** for same-group comparisons.

**Same trend for Group 16:**

```
H₂O < H₂S < H₂Se < H₂Te
pKa: 15.7   7.0    3.9    2.6
```

O–H bond is the shortest and strongest (and O holds the charge most tightly); therefore H₂O is the weakest acid of the group.

## 2. Resonance Effect

**Resonance** stabilises a conjugate base by **delocalising the negative charge over multiple atoms**, reducing the charge density on any single atom.

> 📌 **Key concept:** A negative charge that is spread over multiple atoms is more stable than one concentrated on a single atom.

### A. Carboxylic Acids vs. Alcohols

The most classic example: why is acetic acid (pKa = 4.74) so much more acidic than ethanol (pKa ≈ 16)?

Both have an **O–H bond**, so the element effect is the same. The difference is **resonance**.

**Ethanol conjugate base (ethoxide, CH₃CH₂O⁻):**

```
CH₃CH₂O⁻   →   charge is LOCALISED on the single oxygen
```

**Acetate ion (CH₃COO⁻):**

```
  O              O⁻
  ‖              |
CH₃-C-O⁻  ↔  CH₃-C=O

Charge is delocalised over BOTH oxygens → each carries −½ charge
```

Both oxygens share the negative charge equally (as confirmed by X-ray crystallography — both C–O bond lengths in acetate are equal: 1.26 Å, intermediate between C=O and C–O).

The **more delocalised** charge → more stable → weaker conjugate base → **stronger acid.**

```
pKa comparison:
Ethanol:      pKa ≈ 16   (no resonance in conjugate base)
Acetic acid:  pKa ≈ 4.74 (resonance over 2 oxygens in conjugate base)

~10¹¹ times more acidic due to resonance!
```

### B. Phenol vs. Cyclohexanol

```
Cyclohexanol: pKa ≈ 16
Phenol:       pKa ≈ 10

Phenoxide ion (C₆H₅O⁻) is resonance-stabilised:
Negative charge delocalised into the aromatic ring (ortho and para positions)
```

Resonance structures of phenoxide:

```
C₆H₅O⁻  →  multiple resonance structures where O⁻ charge delocalises to ring carbons

O⁻       O        O        O
|        ‖        ‖        ‖
[benzene] ↔ [...]  ↔ [...]  ↔ [...]
```

### C. Effect of Nitro Groups on Phenol (Electron-Withdrawing Resonance)

The **nitro group (–NO₂)** is a **resonance-withdrawing group** (–M effect):

| Compound | pKa |
|----------|-----|
| Phenol | 10.0 |
| 2-Nitrophenol (ortho) | 7.2 |
| 3-Nitrophenol (meta) | 8.4 |
| 4-Nitrophenol (para) | 7.1 |

Para and ortho positions allow resonance delocalization of the charge into the nitro group → extra stabilisation → more acidic.

The **meta** nitro group can only exert an **inductive effect** (no resonance path to meta position through the ring) → less acidic than ortho/para.

```
Ortho/para NO₂: both inductive + resonance withdrawal → strongest acid
Meta NO₂: inductive only → intermediate acidity
No NO₂: weakest acid (phenol)
```

### D. Carbanion Stabilisation

In carbon chemistry, resonance stabilisation of carbanions makes adjacent C–H bonds more acidic:

```
  O                O⁻
  ‖                ‖
R-C-CH₂-H  →  R-C=CH₂   (enolate, resonance-stabilised)

pKa of α-H in ketone ≈ 20 (much lower than alkane C–H at ~50)
```

## 3. Inductive Effect

The **inductive effect** is the **transmission of partial charges through σ (sigma) bonds** due to electronegativity differences between atoms. It weakens with distance.

### A. Electron-Withdrawing Groups (–I effect)

Electronegative groups pull electron density **away from** the ionisation site, stabilising the conjugate base → **increase acidity**.

**Haloacetic acids:**

| Acid | Substituent | pKa |
|------|------------|-----|
| Acetic acid | –H | 4.74 |
| Fluoroacetic acid | –F | 2.66 |
| Chloroacetic acid | –Cl | 2.86 |
| Bromoacetic acid | –Br | 2.87 |
| Iodoacetic acid | –I | 3.12 |

Fluorine is the most electronegative → greatest inductive withdrawal → most acidic fluoroacetic acid.

**Multiple substituents (additive effect):**

| Acid | pKa |
|------|-----|
| CH₃COOH | 4.74 |
| ClCH₂COOH | 2.86 |
| Cl₂CHCOOH | 1.48 |
| Cl₃CCOOH | 0.64 |

Each additional Cl further withdraws electrons from the carboxylate → further stabilisation → lower pKa.

Fluorinated version (trifluoroacetic acid CF₃COOH):  
pKa = −0.25 → a **superacid** compared to acetic acid!

### B. Distance Dependence of Inductive Effect

The inductive effect operates through σ bonds and **decreases dramatically with distance**:

```
ClCH₂CH₂COOH   vs   ClCH₂COOH   vs   Cl₃CCOOH
pKa = 4.05           pKa = 2.86        pKa = 0.64
(Cl 2C away)         (Cl 1C away)      (3×Cl at 1C)
```

As the Cl moves further from the COOH group, its electron-withdrawing effect is diminished.

**Rule:** Inductive effect decreases by ~50% with each additional carbon between the substituent and the reacting centre.

### C. Electron-Donating Groups (+I effect)

Alkyl groups (methyl, ethyl, etc.) are **electron-donating** through induction → **increase electron density** on conjugate base → **destabilise** conjugate base → **decrease acidity**:

```
Methanol vs. Trifluoroethanol:
CH₃OH:      pKa = 15.5   (CH₃ is electron-donating, destabilises O⁻)
CF₃CH₂OH:  pKa = 12.4   (CF₃ is strongly electron-withdrawing, stabilises O⁻)
```

**Effect on amine basicity:**

Alkyl groups donate electrons to N → increase electron density on N → N is a **better proton acceptor** → more basic:

```
NH₃        <  CH₃NH₂     <  (CH₃)₂NH  (in gas phase)
pKb = 4.74     pKb = 3.34     pKb = 3.27
(weakest base)               (strongest base in gas phase)
```

Note: In water, solvation effects complicate this trend.

## 4. Orbital Hybridisation Effect

The **hybridisation** of the carbon (or other atom) directly bonded to the acidic hydrogen affects acidity through the **s-character** of the hybrid orbital.

### s-Character and Electronegativity

| Hybridisation | s-character | Effective electronegativity of C | C–H pKa |
|--------------|------------|----------------------------------|---------|
| sp³ | 25% | ~2.5 (Pauling) | ~50 |
| sp² | 33% | ~2.75 | ~44 |
| sp | 50% | ~3.2 | ~25 |

**Higher s-character → electrons held closer to nucleus → carbon acts more electronegative → stabilises conjugate base better → stronger acid.**

### Examples

**Alkane vs. Alkene vs. Alkyne:**

```
sp³ C:  CH₃–CH₃    pKa ≈ 50   (weakest acid, lowest s-character)
sp² C:  CH₂=CH₂    pKa ≈ 44
sp  C:  HC≡CH       pKa ≈ 25   (strongest C–H acid, highest s-character)
```

Acetylene (HC≡CH) is acidic enough that it reacts with strong bases:

```
HC≡CH  +  NaNH₂  →  HC≡C⁻Na⁺  +  NH₃

The sp carbon stabilises the negative charge better than sp² or sp³.
```

**Propiolic acid (sp C acid):**

```
HC≡C–COOH:  pKa = 1.84   (very strong for organic acid, sp hybrid makes it stronger)
```

### Effect on Nitrogen Bases

| Compound | N hybridisation | pKb | More/less basic |
|----------|---------------|-----|----------------|
| Triethylamine | sp³ | 3.25 | Strong base |
| Aniline (C₆H₅NH₂) | sp² (adjacent ring) | 9.4 | Weak base |
| Pyrrole (N in ring) | sp² | ~13 | Very weak base |

The lone pair on N in **aniline** is partially delocalised into the π system of the benzene ring (resonance) → less available for protonation → much weaker base than aliphatic amines.

In **pyridine** (sp² N, but lone pair in sp² orbital, *not* involved in ring π system):
```
Pyridine pKb = 8.75   (moderate base — lone pair in sp² orbital, perpendicular to ring)
```

## Periodic Trends in Acid Strength

### Binary Hydrides (H₂O, HF, H₂S, HCl, etc.)

**Across period (electronegativity dominates):**

```
Period 2: H₂O < HF         (O is less electronegative than F → HF is stronger acid)
Period 3: H₂S < HCl
```

But O–H is stronger acid than N–H which is stronger than C–H (across period):

```
NH₃ << H₂O << HF     (left to right: electronegativity increases)
pKa: ~38      ~16      3.17
```

**Down group (atomic size dominates):**

```
Group 17: HF << HCl << HBr << HI
Group 16: H₂O << H₂S << H₂Se << H₂Te
Group 15: NH₃ << PH₃ << AsH₃ (gas-phase basicity reversal)
```

### Oxyacids: E(=O)ₙ(OH)ₘ (Pauling's Rules)

For oxyacids of general formula: **EO_n(OH)_m** where n = number of non-hydroxo oxygen atoms (=O groups):

**Pauling's Acidity Rules:**

```
n = 0 (no extra =O):   pKa ≈ 7–9    (very weak acid, e.g., HOCl, pKa = 7.5)
n = 1 (one =O):        pKa ≈ 1–2    (weak acid, e.g., H₂SO₃, pKa₁ = 1.81; HNO₂, pKa = 3.4)
n = 2 (two =O):        pKa < 0      (strong acid, e.g., H₂SO₄, HNO₃)
n = 3 (three =O):      pKa << 0     (very strong, e.g., HClO₄, HMnO₄)
```

**Reasoning:** Each additional **non-hydroxo oxygen** (=O) withdraws electron density from the E–O–H unit, making it easier to lose H⁺.

```
Chlorine oxyacids:
HOCl    (HClO,  n=0):  pKa = 7.5    ← weakest
HOClO   (HClO₂, n=1):  pKa = 2.0
HOClO₂  (HClO₃, n=2):  pKa = −1.0
HOClO₃  (HClO₄, n=3):  pKa = −10   ← strongest (superacid)
```

**Same central atom, different oxidation state:**

More oxidised (higher oxidation state) → more =O groups → pulls more electron density → stronger acid:

```
H₂SO₃ (S+4, n=1): pKa₁ = 1.81    < H₂SO₄ (S+6, n=2): pKa₁ ≈ −3
HNO₂ (N+3, n=1):  pKa = 3.37     < HNO₃ (N+5, n=2):  pKa = −1.4
H₃PO₃ (P+3, n=1): pKa₁ = 2.0    < H₃PO₄ (P+5, n=2): pKa₁ = 2.15
```

**Different central atom, same oxidation state/n:**

The more electronegative the central atom E → better withdrawal of electrons from E–OH → stronger acid:

```
HClO₄ (Cl) > HBrO₄ (Br) > HIO₄ (I)
H₂SO₄ (S)  > H₂SeO₄ (Se)
HNO₃ (N)   > H₃PO₄ (P)    (N more electronegative than P)
```

## Effect of Structure on Base Strength

### What Makes a Better Base?

A stronger base:
- Has a **higher electron density** on the donor atom (makes it a better proton acceptor)
- Has a **less stable lone pair** (more available for donation)
- Has a **less electronegative donor atom** (more willing to share)

### Factors Decreasing Base Strength

**1. Resonance delocalisation of lone pair:**

If the lone pair is delocalised into a π system, it is less available for protonation:

```
NH₃ (pKb = 4.74, strong) >> Aniline C₆H₅NH₂ (pKb = 9.38, weak)

In aniline: N lone pair delocalised into benzene ring → weaker base
```

**2. Electron-withdrawing groups near N (or O):**

```
Ammonia:       pKb = 4.74  (most basic)
Aniline:       pKb = 9.38  (EW aromatic ring)
p-Nitroaniline:pKb = 13.0  (EW –NO₂ group makes it almost non-basic)
```

**3. sp hybridisation — lone pair held in tighter orbital:**

```
In R₃N (sp³), the lone pair is in a large sp³ orbital → readily donated
In pyridine (sp²), lone pair in sp² orbital → held more tightly → less basic
In nitriles R–C≡N (sp), lone pair on N in sp orbital → weakest base
```

### Factors Increasing Base Strength

**1. Electron-donating alkyl groups (+I effect):**

```
(CH₃)₃N > (CH₃)₂NH > CH₃NH₂ > NH₃  (in gas phase)

Alkyl groups donate electrons to N → N is more electron-rich → stronger base
```

**2. Larger, more polarisable donor atom:**

For the same group, larger atoms are softer Lewis bases:

```
In HSAB terms: soft bases (S, P donors) are better Lewis bases for soft acids
               hard bases (F, O donors) are better for hard acids
```

**3. Negative charge:**

```
CH₃O⁻ > CH₃OH    (methoxide is stronger base than methanol)
OH⁻    > H₂O     (hydroxide is stronger base than water)
```

## Effect of Structure on Oxyacids

### General Oxyacid Structure

Most oxyacids have the form **E–O–H**, and their acidity depends on:

1. **Identity of E** (electronegativity)
2. **Number of additional =O groups** attached to E
3. **Other substituents** on E

### Inductive Withdrawal by E

The more electronegative E is, the more it withdraws electron density from the O–H bond, weakening it and facilitating loss of H⁺:

```
Compare HF (pKa = 3.17) vs H₂O (pKa = 15.7):
F is more electronegative than O → HF is more acidic
```

For a series of acids with the same structure but different E:

```
HIO₃ < HBrO₃ < HClO₃   (Cl most electronegative → strongest acid)
```

### Effect of Oxidation State

Higher oxidation state of E → more positive charge → pulls more electron density from O–H:

```
Manganese oxyacids:
Mn(VII): HMnO₄ (permanganic acid) — very strong, pKa << 0
Mn(VI):  H₂MnO₄ — strong
```

**Chromium:**

```
Cr(VI): H₂CrO₄ (chromic acid) — strong
Cr(III): Cr(OH)₃ — amphoteric, very weak acid
```

## Practice Problems

**Problem 1:** Arrange in order of increasing acidity:  
FCH₂COOH, BrCH₂COOH, CH₃COOH, ClCH₂COOH

**Answer:**

Based on electronegativity (F > Cl > Br, so inductive effect: F > Cl > Br):

```
CH₃COOH < BrCH₂COOH < ClCH₂COOH < FCH₂COOH
pKa: 4.74     2.87         2.86        2.66

Increasing acidity (decreasing pKa) →
```

---

**Problem 2:** Explain why acetic acid (pKa = 4.74) is more acidic than ethanol (pKa = 16).

**Answer:**

Both have an O–H bond, so we compare their conjugate bases:
- Ethoxide (CH₃CH₂O⁻): negative charge **localised** on one oxygen
- Acetate (CH₃COO⁻): negative charge **delocalised** over two oxygens by resonance

The resonance stabilisation of the acetate ion makes it a much weaker base, so acetic acid is a much stronger acid.

---

**Problem 3:** Which is more acidic: HC≡CH or CH₂=CH₂? Explain.

**Answer:**

HC≡CH (pKa ≈ 25) is more acidic than CH₂=CH₂ (pKa ≈ 44).

The sp-hybridised carbon in acetylene has **50% s-character**, making it effectively more electronegative than sp² carbon (33% s-character). This means the sp carbon holds the electron pair of the carbanion more tightly and in a more stabilised orbital → acetylide is more stable → acetylene is more acidic.

---

**Problem 4:** Rank in order of increasing base strength:  
NH₃, NF₃, N(CH₃)₃, C₆H₅NH₂

**Answer:**

```
NF₃ < C₆H₅NH₂ < NH₃ < N(CH₃)₃

NF₃: F is electron-withdrawing (–I effect) → N lone pair pulled away → very weak base (pKb ≈ undetermined, essentially non-basic)
C₆H₅NH₂: resonance of lone pair into ring → weaker than NH₃ (pKb = 9.38)
NH₃: reference (pKb = 4.74)
N(CH₃)₃: alkyl +I effect → stronger base than NH₃ (pKb ≈ 4.19, gas phase)
```

---

**Problem 5:** Using Pauling's rules, rank these in order of acid strength:  
HClO, HClO₂, HClO₃, HClO₄

**Answer:**

Number of non-hydroxo oxygens (n):
- HClO (HOCl): n = 0, pKa ≈ 7.5
- HClO₂: n = 1, pKa ≈ 2.0
- HClO₃: n = 2, pKa ≈ −1.0
- HClO₄: n = 3, pKa ≈ −10

```
Increasing acid strength: HClO < HClO₂ < HClO₃ < HClO₄
```

Each additional =O group on Cl withdraws more electron density through induction, making the O–H bond progressively more acidic.

---

**Problem 6:** Explain why 4-nitrophenol (pKa = 7.15) is more acidic than phenol (pKa = 10.0) but less acidic than 2,4-dinitrophenol (pKa = 4.09).

**Answer:**

The **nitro group (–NO₂)** stabilises the phenoxide ion by:
1. **Inductive effect (–I):** NO₂ is electron-withdrawing through σ bonds
2. **Resonance effect (–M):** At the para position, NO₂ can directly delocalise the negative charge of the phenoxide via extended resonance

With **two nitro groups** (at ortho and para), both inductive and resonance effects are doubled → even more stable phenoxide → more acidic.
