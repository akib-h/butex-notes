## How to Use These Notes

1. **Sequential study:** Files 01–03 provide context on methods; 04 covers sampling; 05–08 cover statistical foundations.
2. **Exam prep:** Each file ends with worked examples and references for deeper reading.
3. **Cross-referencing:** Error propagation (File 07) directly connects to sig. figs. (File 08) and precision/accuracy (File 05).
