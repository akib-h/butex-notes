## 🔬 Important Formulas

### Metallic Bonding
```
Lattice Energy: U = -NZe²α/r₀(1 - 1/n)
Fermi Energy: E_F = (h²/2m*)(3π²n)^(2/3)
Wiedemann-Franz Law: κ/σT = L
```

### VSEPR Theory
```
Steric Number = bonded atoms + lone pairs
Tetrahedral angle: cos⁻¹(-1/3) = 109.47°
```

### VBT and Hybridization
```
Bond strength ∝ Overlap integral: S = ∫ ψ₁ψ₂ dτ
Hybrid orbital: Ψ(hybrid) = cᵢΨ(s) + c₂Ψ(p) + ...
```

### Intermolecular Forces
```
Dipole-dipole: E ∝ μ₁μ₂/r³
London dispersion: E ∝ -C/r⁶
Hydrogen bonding: E ≈ 10-40 kJ/mol
```

## 🎓 Study Tips

### Understanding Bonding Theories

1. **Don't memorize, understand**: Focus on WHY each theory works
2. **Compare theories**: Know when to use VBT vs. MOT
3. **Practice visualization**: Draw orbital diagrams frequently
4. **Connect to properties**: Always link theory to observable properties

### Problem-Solving Strategy

1. **Identify what's asked**: Geometry? Bonding type? Properties?
2. **Choose appropriate theory**: VSEPR for shapes, VBT for localized bonds, MOT for delocalization
3. **Draw structures**: Lewis, orbital diagrams, MO diagrams as needed
4. **Check reasonableness**: Do results match known trends?

### Common Pitfalls to Avoid

❌ Confusing electron-pair geometry with molecular geometry  
❌ Forgetting lone pairs in VSEPR predictions  
❌ Treating multiple bonds as multiple electron domains  
❌ Mixing up σ and π bond characteristics  
❌ Assuming d-orbitals always participate in sp³d hybridization
