# VSEPR Theory: Molecular Geometries

## Complete Guide to Molecular Shapes

## Steric Number 2: Linear Geometry

### Electron-Pair Geometry: Linear

#### AX₂ - Linear Molecular Geometry

**Structure**:
```
X --- A --- X
```

**Bond Angle**: 180°

**Examples**:
1. **BeCl₂** (Beryllium chloride)
   - Lewis: Cl-Be-Cl
   - Valence e⁻: 2 + 7(2) = 16
   - Be has only 4 e⁻ (incomplete octet)
   
2. **CO₂** (Carbon dioxide)
   - Lewis: O=C=O
   - Note: Double bonds count as ONE domain
   - 2 bonding domains, 0 lone pairs
   
3. **HCN** (Hydrogen cyanide)
   - Lewis: H-C≡N
   - Triple bond counts as ONE domain

**Characteristics**:
- Most symmetric geometry
- No dipole moment if terminal atoms identical
- Minimum repulsion

## Steric Number 3: Trigonal Planar Base

### Electron-Pair Geometry: Trigonal Planar

#### AX₃ - Trigonal Planar Molecular Geometry

**Structure**:
```
      X
     /
    A --- 120°
     \
      X     X
```

**Bond Angle**: 120°

**Examples**:
1. **BF₃** (Boron trifluoride)
   - Valence e⁻: 3 + 7(3) = 24
   - B has only 6 e⁻ (electron deficient)
   
2. **CH₂O** (Formaldehyde)
   ```
      O
      ║
   H-C-H
   ```
   - C=O double bond = 1 domain
   - 2 C-H single bonds = 2 domains
   - Total: 3 domains
   
3. **NO₃⁻** (Nitrate ion)
   - Resonance structures
   - All O-N-O angles = 120°

**Characteristics**:
- All atoms in same plane
- sp² hybridization
- Non-polar if terminal atoms identical

#### AX₂E - Bent (Angular) Molecular Geometry

**Structure**:
```
    X
   / \  <120°
  A   X
  (E)
```

**Bond Angle**: <120° (typically 117-119°)

**Examples**:
1. **SO₂** (Sulfur dioxide)
   - Lewis: O=S=O with lone pair on S
   - Bond angle: ~119°
   
2. **O₃** (Ozone)
   - Central O has lone pair
   - O-O-O angle: ~117°
   
3. **SnCl₂** (Tin(II) chloride)
   - Sn has lone pair
   - Cl-Sn-Cl angle: ~95° (larger deviation due to larger Sn)

**Characteristics**:
- Lone pair causes compression of bond angle
- Polar molecules
- Not to be confused with AX₂E₂ (bent from tetrahedral)

## Steric Number 4: Tetrahedral Base

### Electron-Pair Geometry: Tetrahedral

#### AX₄ - Tetrahedral Molecular Geometry

**Structure**:
```
        X
        |
    X - A - X  (109.5° between all)
        |
        X
```

**Bond Angle**: 109.5°

**Examples**:
1. **CH₄** (Methane)
   - Perfect tetrahedral
   - All H-C-H angles = 109.5°
   
2. **CCl₄** (Carbon tetrachloride)
   - Non-polar (symmetrical)
   
3. **NH₄⁺** (Ammonium ion)
   - All N-H bonds equivalent
   - Positive charge doesn't affect geometry

**Derivation of 109.5°**:
```
cos(θ) = -1/3
θ = arccos(-1/3) = 109.47° ≈ 109.5°
```

#### AX₃E - Trigonal Pyramidal Molecular Geometry

**Structure**:
```
      X
     /|\
    X A X   <109.5°
     (E)
```

**Bond Angle**: ~107° (less than 109.5°)

**Examples**:
1. **NH₃** (Ammonia)
   - H-N-H angle: 107.3°
   - Lone pair compresses angles
   - Molecular dipole moment: 1.47 D
   
2. **PH₃** (Phosphine)
   - H-P-H angle: 93.5°
   - Larger deviation (P is larger, less s-character)
   
3. **PCl₃** (Phosphorus trichloride)
   - Cl-P-Cl angle: ~100°

**Why angle < 109.5°?**
- LP-BP repulsion > BP-BP repulsion
- Lone pair "pushes" bonding pairs closer together

#### AX₂E₂ - Bent (Angular) Molecular Geometry

**Structure**:
```
    X - A - X   <109.5°
       (E)
       (E)
```

**Bond Angle**: ~104.5° (significantly less than 109.5°)

**Examples**:
1. **H₂O** (Water)
   - H-O-H angle: 104.5°
   - Two lone pairs cause greater compression
   - Highly polar: μ = 1.85 D
   
2. **H₂S** (Hydrogen sulfide)
   - H-S-H angle: 92°
   - Even more compression (less s-character in S)
   
3. **SCl₂** (Sulfur dichloride)
   - Cl-S-Cl angle: ~103°

**Trend**:
```
CH₄ (109.5°) > NH₃ (107°) > H₂O (104.5°)
```
Each lone pair reduces angle by ~2-2.5°

## Steric Number 5: Trigonal Bipyramidal Base

### Electron-Pair Geometry: Trigonal Bipyramidal

**Key Feature**: Two types of positions!
- **Axial**: Top and bottom (2 positions) - 90° from equatorial
- **Equatorial**: Around middle (3 positions) - 120° apart

#### AX₅ - Trigonal Bipyramidal Molecular Geometry

**Structure**:
```
        X_ax
         |
    X_eq-A-X_eq  (120° equatorial, 90° ax-eq)
         |
        X_ax
```

**Bond Angles**: 
- Equatorial-Equatorial: 120°
- Axial-Equatorial: 90°
- Axial-Axial: 180°

**Examples**:
1. **PCl₅** (Phosphorus pentachloride)
   - 3 equatorial Cl, 2 axial Cl
   - Axial P-Cl bonds slightly longer (more repulsion)
   - Non-polar overall
   
2. **PF₅** (Phosphorus pentafluoride)
   - Gaseous state: trigonal bipyramidal
   - Solid state: ionic [PF₄⁺][PF₆⁻]
   
3. **AsF₅** (Arsenic pentafluoride)
   - Similar to PCl₅

**Important Rule**: 
> **Lone pairs preferentially occupy equatorial positions** (to minimize 90° LP-BP repulsions)

#### AX₄E - Seesaw (Disphenoidal) Molecular Geometry

**Structure**:
```
        X_ax
         |
    X_eq-A-X_eq
        (E)
```

**Bond Angles**: 
- Equatorial-Equatorial: <120° (~102°)
- Axial-Equatorial: <90° (~87°)

**Examples**:
1. **SF₄** (Sulfur tetrafluoride)
   - Lone pair in equatorial position
   - F(eq)-S-F(eq): ~102°
   - F(ax)-S-F(eq): ~87°
   - Polar molecule
   
2. **TeCl₄** (Tellurium tetrachloride)
   - Similar distortion
   
3. **XeO₂F₂** (Xenon dioxide difluoride)
   - Complex example with different atoms

**Why "Seesaw"?**
- Shape resembles playground seesaw
- Two atoms up/down (axial), two to sides (equatorial)

#### AX₃E₂ - T-Shaped Molecular Geometry

**Structure**:
```
        X
        |
    X - A - X
       (E)(E)
```

**Bond Angles**: <90° (~87°)

**Examples**:
1. **ClF₃** (Chlorine trifluoride)
   - 2 lone pairs in equatorial positions
   - F-Cl-F angles: ~87° (axial-equatorial)
   - Very reactive, powerful fluorinating agent
   
2. **BrF₃** (Bromine trifluoride)
   - Similar to ClF₃
   - F-Br-F: ~86°
   
3. **ICl₃** (Iodine trichloride)
   - Cl-I-Cl: ~90°

**Why T-shaped?**
- Two lone pairs occupy equatorial positions (minimize 90° repulsions)
- Three atoms form "T" shape

#### AX₂E₃ - Linear Molecular Geometry

**Structure**:
```
X --- A --- X
   (E)(E)(E)
```

**Bond Angle**: 180°

**Examples**:
1. **XeF₂** (Xenon difluoride)
   - 3 lone pairs in equatorial plane
   - 2 F atoms in axial positions
   - Non-polar (symmetrical)
   
2. **I₃⁻** (Triiodide ion)
   - Central I has 3 lone pairs
   - Linear arrangement
   
3. **ICl₂⁻** (Dichloroiodate ion)
   - Similar to XeF₂

**Why Linear?**
- 3 lone pairs occupy all equatorial positions
- Only axial positions available for bonding
- Maximizes distance between bonding pairs (180°)

## Steric Number 6: Octahedral Base

### Electron-Pair Geometry: Octahedral

**Key Feature**: All positions equivalent (unlike trigonal bipyramidal)

#### AX₆ - Octahedral Molecular Geometry

**Structure**:
```
        X
        |
    X - A - X  (90° angles)
        |
        X
    (X above and below plane)
```

**Bond Angles**: All 90° or 180°

**Examples**:
1. **SF₆** (Sulfur hexafluoride)
   - Perfect octahedral
   - All S-F bonds equivalent
   - Non-polar, very stable
   - Used as electrical insulator
   
2. **[Co(NH₃)₆]³⁺** (Hexaamminecobalt(III) ion)
   - Coordination complex
   - Oh symmetry
   
3. **[Fe(CN)₆]³⁻** (Hexacyanoferrate(III) ion)
   - All positions equivalent

#### AX₅E - Square Pyramidal Molecular Geometry

**Structure**:
```
        X
        |
    X - A - X  (square base)
        |
        X
       (E)
```

**Bond Angles**: 
- Base X-A-X: ~84° (slightly <90°)
- Apex-A-base: ~84°

**Examples**:
1. **BrF₅** (Bromine pentafluoride)
   - Lone pair causes compression
   - F-Br-F angles: ~84°
   
2. **IF₅** (Iodine pentafluoride)
   - Similar distortion
   
3. **[SbCl₅]²⁻** (Pentachloroantimonate ion)
   - Square pyramidal geometry

**Why angles < 90°?**
- Lone pair repulsion compresses bonding pairs
- All five atoms pushed slightly away from lone pair

#### AX₄E₂ - Square Planar Molecular Geometry

**Structure**:
```
    X
    |
X - A - X  (90° angles)
    |
    X
 (E above, E below)
```

**Bond Angles**: 90°, 180°

**Examples**:
1. **XeF₄** (Xenon tetrafluoride)
   - Lone pairs above and below plane
   - All F-Xe-F angles: 90° or 180°
   - Non-polar
   
2. **ICl₄⁻** (Tetrachloroiodate ion)
   - Square planar
   - Cl-I-Cl: 90°
   
3. **[PtCl₄]²⁻** (Tetrachloroplatinate ion)
   - Common coordination geometry
   - Used in cisplatin (anticancer drug)

**Why Square Planar?**
- Lone pairs occupy opposite positions (trans)
- Minimizes LP-LP repulsion (180° apart)
- Four atoms form square in plane

## Special Case: Steric Number 7

### AX₇ - Pentagonal Bipyramidal

**Structure**: 5 atoms in pentagonal plane, 2 axial

**Example**: **IF₇** (Iodine heptafluoride)
- 72° between equatorial F
- 90° axial-equatorial

|----------|-------|-----------|-----------|

| Trig planar | 120° | ~119° | - |
| Trig bipyramidal | 90°, 120° | <90°, <120° | - |
