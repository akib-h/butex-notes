## 🧪 Modern Concepts of Acids and Bases

### 1. Arrhenius Theory (1884)

> **Acid:** Produces H⁺ ions in aqueous solution  
> **Base:** Produces OH⁻ ions in aqueous solution

```
HCl  →  H⁺ + Cl⁻         (Acid)
NaOH →  Na⁺ + OH⁻        (Base)

Neutralization: H⁺ + OH⁻ → H₂O
```

**Limitations:**
- Only applicable in aqueous solutions
- Cannot explain bases like NH₃ (no OH⁻)
- Cannot explain acid-base reactions in non-aqueous solvents

---

### 2. Brønsted-Lowry Theory (1923)

> **Acid (Proton donor):** Any species that can **donate a proton (H⁺)**  
> **Base (Proton acceptor):** Any species that can **accept a proton (H⁺)**

```
General:   Acid + Base ⇌ Conjugate Base + Conjugate Acid

Example:
HCl  + H₂O  ⇌  Cl⁻  +  H₃O⁺
acid₁  base₂    conj base₁  conj acid₂

NH₃  + H₂O  ⇌  NH₄⁺  +  OH⁻
base₁  acid₂   conj acid₁  conj base₂
```

**Advantages over Arrhenius:**
- Applies in non-aqueous solvents
- Explains NH₃ as a base
- Introduces conjugate pairs

**Limitations:**
- Still requires proton transfer — cannot explain aprotic reactions
- Cannot explain BF₃ as Lewis acid

---

### 3. Lewis Theory (1923)

> **Acid (Electron pair acceptor):** Any species that can **accept an electron pair**  
> **Base (Electron pair donor):** Any species that can **donate an electron pair**

```
BF₃  +  :NH₃  →  F₃B←NH₃
Lewis    Lewis     Complex
acid     base

H⁺  +  :OH⁻  →  H:OH  (water)
Lewis     Lewis
acid      base
```

**Advantages:**
- Broadest definition — includes all Brønsted acids/bases
- Explains reactions without proton transfer (e.g., BF₃ + NH₃)
- Covers catalysis, coordination chemistry

**Limitations:**
- Difficult to quantify acid/base strength
- Strength varies with solvent and conditions

---

### 4. Lux-Flood Theory (1939)

> Proposed for **high-temperature reactions** (molten salts, oxides)

> **Acid (Oxide acceptor):** Species that **accepts oxide (O²⁻) ions**  
> **Base (Oxide donor):** Species that **donates oxide (O²⁻) ions**

```
SiO₂  +  CaO  →  CaSiO₃
oxide acceptor  oxide donor
(Lux-Flood acid) (Lux-Flood base)

SO₃  +  CaO  →  CaSO₄
(acid)   (base)
```

**Applications:** Metallurgy, high-temperature industrial chemistry

---

### Summary Comparison of Theories

| Theory | Acid | Base | Scope |
|--------|------|------|-------|
| Arrhenius | H⁺ producer | OH⁻ producer | Aqueous only |
| Brønsted-Lowry | Proton donor | Proton acceptor | Protic solvents |
| Lewis | e⁻ pair acceptor | e⁻ pair donor | Universal |
| Lux-Flood | O²⁻ acceptor | O²⁻ donor | High temperature |

## 🔄 Conjugate Acid-Base Pairs

> A **conjugate acid-base pair** differs by exactly one proton.

$$\text{Acid} \rightleftharpoons \text{Conjugate Base} + H^+$$

```
Pair 1:  HCl  ⇌  Cl⁻  +  H⁺
         acid₁     CB₁

Pair 2:  H₂O + H⁺  ⇌  H₃O⁺
         base₂       CA₂

Full equation:
HCl + H₂O ⇌ Cl⁻ + H₃O⁺
Acid₁ Base₂  CB₁   CA₂
  └──pair 1──┘  └──pair 2──┘
```

### Important Conjugate Pairs

| Acid | Conjugate Base | Comment |
|------|---------------|---------|
| HCl | Cl⁻ | Strong acid → very weak CB |
| H₂SO₄ | HSO₄⁻ | Strong acid |
| H₃O⁺ | H₂O | Hydronium ion |
| H₂O | OH⁻ | Water is amphoteric |
| NH₄⁺ | NH₃ | Weak acid → strong CB |
| HCO₃⁻ | CO₃²⁻ | Bicarbonate as acid |
| HCO₃⁻ | H₂CO₃ | Bicarbonate as base |

### Key Relationship
> **Stronger the acid → Weaker the conjugate base**  
> **Weaker the acid → Stronger the conjugate base**

## 🔁 Amphiprotic Species

> An **amphiprotic (amphoteric)** species can **act as both acid and base** depending on the reaction conditions.

```
H₂O acts as ACID:        H₂O + NH₃  ⇌  OH⁻  +  NH₄⁺
                          (donates H⁺ to NH₃)

H₂O acts as BASE:        H₂O + HCl  ⇌  H₃O⁺ +  Cl⁻
                          (accepts H⁺ from HCl)
```

### Other Amphiprotic Species

| Species | As acid | As base |
|---------|---------|---------|
| H₂O | → OH⁻ | → H₃O⁺ |
| HCO₃⁻ | → CO₃²⁻ | → H₂CO₃ |
| HSO₄⁻ | → SO₄²⁻ | → H₂SO₄ |
| H₂PO₄⁻ | → HPO₄²⁻ | → H₃PO₄ |
| NH₂⁻ | → NH₃ | → N²⁻ (rare) |
| Amino acids | → zwitterion | → zwitterion |

```
HCO₃⁻ + H⁺  →  H₂CO₃   (acting as base)
HCO₃⁻ + OH⁻ →  CO₃²⁻ + H₂O  (acting as acid)
```

## ⚖️ Lewis vs Brønsted-Lowry Comparison

| Feature | Brønsted-Lowry | Lewis |
|---------|----------------|-------|
| **Acid** | Proton donor | Electron pair acceptor |
| **Base** | Proton acceptor | Electron pair donor |
| **Transfer** | Proton (H⁺) | Electron pair |
| **Scope** | Protic reactions | Broader (includes non-protic) |
| **Examples of acids** | HCl, H₂SO₄, H₃O⁺ | BF₃, AlCl₃, BF₃, Fe³⁺ |
| **Examples of bases** | OH⁻, NH₃, H₂O | :NH₃, :OH⁻, :F⁻, CO |
| **Relation** | Subset of Lewis | Includes all B-L acids/bases |

```
All Brønsted acids are Lewis acids, but NOT vice versa:
  BF₃ → Lewis acid (no proton!) 
  BF₃ → NOT a Brønsted acid
```

## 🔷 HSAB Theory (Hard and Soft Acids and Bases)

### Proposed by R.G. Pearson (1963)

> **Hard species:** Small, non-polarizable, high charge density  
> **Soft species:** Large, polarizable, low charge density  
> **Principle:** "Hard acids prefer hard bases; soft acids prefer soft bases"

### Classification

#### Hard Acids (small, high charge, non-polarizable)
```
H⁺, Li⁺, Na⁺, K⁺, Mg²⁺, Ca²⁺, Al³⁺, Cr³⁺, Fe³⁺, Co³⁺, BF₃, BCl₃
```

#### Soft Acids (large, low charge, polarizable)
```
Cu⁺, Ag⁺, Au⁺, Hg²⁺, Pt²⁺, Pd²⁺, Pb²⁺, Tl⁺, I₂, Br₂
```

#### Intermediate Acids
```
Fe²⁺, Co²⁺, Ni²⁺, Cu²⁺, Zn²⁺, Pb²⁺, Sb³⁺, Bi³⁺
```

#### Hard Bases (small, electronegative, non-polarizable)
```
F⁻, OH⁻, H₂O, NH₃, RNH₂, Cl⁻, CO₃²⁻, SO₄²⁻, PO₄³⁻
```

#### Soft Bases (large, polarizable, low electronegativity)
```
I⁻, S²⁻, RS⁻, SCN⁻, CN⁻, CO, R₂S, R₃P, C₂H₄, H⁻
```

#### Intermediate Bases
```
Br⁻, NO₂⁻, SO₃²⁻, N₃⁻, C₆H₅NH₂
```

### Applications of HSAB

| Observation | Explanation |
|-------------|-------------|
| Fe³⁺ forms stable fluoride | Fe³⁺ is hard, F⁻ is hard → stable |
| Hg²⁺ prefers S²⁻ | Hg²⁺ is soft, S²⁻ is soft → stable |
| Pt²⁺ binds CN⁻ strongly | Both are soft → stable |
| CaF₂ (fluorite) is stable | Ca²⁺ hard + F⁻ hard |
| CsI is more stable than CsF | Cs⁺ is softer → prefers softer I⁻ |

## 📏 Strength of Acids

### Factors Affecting Acid Strength

#### 1. Bond Strength (H-X bond)
```
Weaker the H-X bond → stronger the acid

HI > HBr > HCl > HF   (in terms of bond strength)
(but in water, all except HF are strong — solvent effects matter)
```

#### 2. Electronegativity (same period)
```
More electronegative central atom → more acidic (better H⁺ release)

Across a period: HF > H₂O > NH₃ > CH₄
(F most electronegative → HF most acidic among these)
```

#### 3. Oxidation State (oxyacids)
```
Higher oxidation state → more electronegative center → more acidic

HClO < HClO₂ < HClO₃ < HClO₄
+1      +3      +5      +7
(increasing Cl oxidation state → increasing acid strength)
```

#### 4. Number of Oxygen Atoms (oxyacids)
```
More O atoms → more electron withdrawal from H-O bond → stronger acid

H₃PO₂ < H₃PO₃ < H₃PO₄
(1 OH)   (2 OH)   (3 OH)

Strength rules: +2 oxyacids: pKa ~ 1-2 (strong)
                +1 oxyacids: pKa ~ 2 (moderate)
```

### Arranging in Increasing Strength: H₂SO₄, HNO₃, HClO₄, HMnO₄

| Acid | Central atom | Oxidation state | No. of O | Ka |
|------|-------------|----------------|----------|-----|
| HNO₃ | N | +5 | 3 | ~27 |
| H₂SO₄ | S | +6 | 4 | very large |
| HClO₄ | Cl | +7 | 4 | ~10⁷ |
| HMnO₄ | Mn | +7 | 4 | ~2×10⁷ |

**Increasing order:**
$$\text{HNO}_3 < \text{H}_2\text{SO}_4 < \text{HClO}_4 < \text{HMnO}_4$$

> 💡 **Note:** All four are strong acids, but the order of **maximum strength** follows oxidation state and electronegativity of central atom.

## 🔥 Why is HClO₄ Stronger Acid than H₃PO₄?

This depends on:
1. **Oxidation state of central atom**
2. **Number of non-bridging oxygens**
3. **Electronegativity**

```
HClO₄ (Perchloric acid):           H₃PO₄ (Phosphoric acid):
  Cl oxidation state = +7             P oxidation state = +5
  4 oxygen atoms total                4 oxygen atoms total
  1 O-H bond, 3 P=O bonds            3 O-H bonds, 1 P=O bond
  
     O                                   OH
     ‖                                   |
HO─Cl═O         vs              HO─P═O
     ‖                                   |
     O                                   OH
  
  1 "ionizable" O-H                  3 "ionizable" O-H
  BUT Cl is more electronegative than P
  AND has higher oxidation state
  
  Cl(+7) withdraws more electron density from O-H → weaker O-H bond
  → H⁺ is released more readily → STRONGER ACID
```

**Inductive effect comparison:**
```
H─O─Cl(+7)  →  much electron withdrawal from O-H
H─O─P(+5)   →  less electron withdrawal

∴ HClO₄ is a much stronger acid than H₃PO₄
```

> HClO₄ is one of the **strongest acids known** (superacid class).  
> H₃PO₄ is a **weak/moderate acid** (Ka₁ = 7.5 × 10⁻³)

## 📐 Neutralization Theory — S.V₁ = S.V₂

### Derivation

For a neutralization reaction between an acid and a base:

$$\text{Acid} + \text{Base} \rightarrow \text{Salt} + \text{Water}$$

**At equivalence point:**

$$\text{moles of H}^+ = \text{moles of OH}^-$$

$$\text{Normality} \times \text{Volume} = \text{milliequivalents}$$

$$\boxed{S \cdot V_1 = S \cdot V_2}$$

Where:
- **S** = Strength (Normality, N) of the solution
- **V₁** = Volume of acid
- **V₂** = Volume of base

**Proof:**

Let acid of normality N₁, volume V₁ react with base of normality N₂, volume V₂.

```
Milliequivalents of acid = N₁ × V₁  (V in mL)
Milliequivalents of base = N₂ × V₂

At equivalence: N₁V₁ = N₂V₂

i.e., S₁V₁ = S₂V₂   where S = N = normality = strength
```

**Example:**
```
25 mL of 0.1 N HCl is neutralized by NaOH.
What volume of 0.05 N NaOH is needed?

S₁V₁ = S₂V₂
0.1 × 25 = 0.05 × V₂
V₂ = 2.5 / 0.05 = 50 mL
```

## 🔬 Analytical Chemistry Concepts

### Accuracy vs Precision

| Feature | Accuracy | Precision |
|---------|----------|-----------|
| **Definition** | How close a result is to the **true value** | How close **repeated measurements** are to each other |
| **Indicates** | Correctness | Reproducibility |
| **Can you have one without the other?** | Precise but not accurate (systematic error) | Accurate but not precise (random error) |

```
Target analogy:
  Accurate + Precise:    Accurate + Not Precise:   Not Accurate + Precise:
       ●                      ●                         ● ●
      ●●●                  ● ● ●                       ●●
       ●                      ●                         ●
   (clustered at        (spread but near           (clustered away
    bullseye)            center)                    from bullseye)
```

### Types of Errors

| Type | Cause | Examples | Elimination |
|------|-------|----------|-------------|
| **Systematic (Determinate)** | Instrument, method, or personal bias | Calibration error, impure reagent | Proper calibration, blanks |
| **Random (Indeterminate)** | Unpredictable fluctuations | Reading scale, vibration | Statistical averaging |
| **Gross (Blunders)** | Human mistakes | Misreading, wrong measurement | Repeating experiment |

### Instrumental Methods — Advantages

| Advantage | Explanation |
|-----------|-------------|
| **Speed** | Results in seconds vs hours for classical methods |
| **Sensitivity** | Detect ppb or ppm concentrations |
| **Accuracy** | Less operator-dependent |
| **Specificity** | Can analyze complex mixtures |
| **Automation** | Routine analysis with minimal human input |
| **Small sample size** | Requires µL to mL, not grams |

## 🧪 Practice Questions with Solutions

### Q1. Identify the conjugate acid-base pairs in: CH₃COOH + H₂O ⇌ CH₃COO⁻ + H₃O⁺

**Solution:**
```
CH₃COOH    +    H₂O    ⇌    CH₃COO⁻    +    H₃O⁺
  acid₁         base₂         CB₁              CA₂
  └──────────── pair 1 ───────┘   └──── pair 2 ───┘
```

- **Pair 1:** CH₃COOH / CH₃COO⁻ (differ by one H⁺)
- **Pair 2:** H₃O⁺ / H₂O (differ by one H⁺)

### Q2. Classify these as hard, soft, or intermediate: Ag⁺, Al³⁺, Fe²⁺, I⁻, F⁻, CO

**Solution:**

| Species | Classification | Reason |
|---------|---------------|--------|
| Ag⁺ | **Soft acid** | Large, low charge density, polarizable |
| Al³⁺ | **Hard acid** | Small, high charge, non-polarizable |
| Fe²⁺ | **Intermediate acid** | Medium size and charge |
| I⁻ | **Soft base** | Large, highly polarizable |
| F⁻ | **Hard base** | Small, non-polarizable, hard to polarize |
| CO | **Soft base** | Carbon donor, polarizable |

### Q3. Prove that a solution of 10 mL of 2N H₂SO₄ is exactly neutralized by 20 mL of 1N NaOH

**Solution:**
```
Acid: V₁ = 10 mL, S₁ = 2N → milliequivalents = 10 × 2 = 20 meq
Base: V₂ = 20 mL, S₂ = 1N → milliequivalents = 20 × 1 = 20 meq

S₁V₁ = 20 meq = S₂V₂  ✓ → Exactly neutralized
```

### Q4. Explain why HF is a weaker acid than HCl despite F being more electronegative than Cl

**Solution:**
The strength of an acid HX depends on the ease of breaking H−X bond and releasing H⁺.

```
H−F bond energy: 570 kJ/mol  (very strong, short bond)
H−Cl bond energy: 432 kJ/mol  (weaker)

Although F is more electronegative (better at stabilizing X⁻),
the very strong H−F bond makes it harder to donate H⁺.

Bond energy effect DOMINATES over electronegativity in this case.
∴ HF is a WEAK acid (Ka = 7.2 × 10⁻⁴)
   HCl is a STRONG acid (fully dissociates)
```

### Q5. A 20 mL solution of H₂SO₄ is titrated with 0.1 M NaOH. If 40 mL of NaOH is used, find the molarity of H₂SO₄.

**Solution:**
```
H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O

Moles NaOH = 0.1 M × 0.040 L = 0.004 mol
Moles H₂SO₄ = 0.004/2 = 0.002 mol  (2:1 ratio)
Molarity H₂SO₄ = 0.002/0.020 = 0.1 M
```
