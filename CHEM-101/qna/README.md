## 📋 Quick Formula Sheet

### Chapter 1 — Periodicity
```
Total orbitals = n²
Max electrons in shell = 2n²
Azimuthal: ℓ = 0 to (n-1)
Magnetic: mₗ = -ℓ to +ℓ
Heisenberg: Δx·Δp ≥ h/4π
```

### Chapter 2 — Chemical Bonding
```
Bond Order = (Nb - Na) / 2
Hybridization: sp(180°), sp²(120°), sp³(109.5°)
σ bonds: head-on overlap
π bonds: lateral overlap
```

### Chapter 3 — Complex Compounds
```
EAN = Z - n + 2L   (should equal 36, 54, or 86)
μ = √n(n+2) BM    (magnetic moment)
CFSE(oct): t₂g lower, eg higher
```

### Chapter 4 — Acid & Base
```
Neutralization: S₁V₁ = S₂V₂
HSAB: Hard-Hard and Soft-Soft preferred
Acid strength ∝ oxidation state of central atom
```

### Chapter 5 — Colligative Properties
```
Raoult's law: P = P⁰·X_A
ΔP/P⁰ = X_B
ΔTb = Kb·m
ΔTf = Kf·m
π = CRT = nRT/V
M = wRT/(πV)    [molar mass from osmotic pressure]
M = Kf × w₂ × 1000 / (ΔTf × w₁)   [from FP depression]
```
