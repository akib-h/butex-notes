# 1. Periodicity of the Elements

## Q1. Key Principles and Terms

### ① Mendeleev's Periodic Law
**Statement:** Properties of elements are a **periodic function of their atomic masses**.

**Steps to understand:**
1. Mendeleev arranged elements in order of increasing atomic mass.
2. Elements with similar properties were placed in the same vertical column (Group).
3. He left gaps for undiscovered elements (e.g., predicted Eka-silicon → Germanium).

**Limitation:** Could not explain isotopes; atomic mass order had exceptions (e.g., Ar before K).

**Modern Periodic Law:** Properties are a periodic function of **atomic number** (Moseley).

---

### ② Pauli Exclusion Principle
**Statement:** No two electrons in an atom can have the **same set of four quantum numbers**.

**Steps:**
1. Every electron is described by 4 quantum numbers: n, l, mₗ, mₛ.
2. Two electrons in the same orbital must have **opposite spins** (+½ and −½).
3. Therefore, each orbital holds a **maximum of 2 electrons**.

**Example:** In 1s orbital → one electron has (1, 0, 0, +½) and the other has (1, 0, 0, −½).

---

### ③ Aufbau Principle
**Statement:** Electrons fill orbitals in order of **increasing energy**.

**Steps:**
1. Use the (n + l) rule: lower (n + l) = lower energy.
2. Filling order: `1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p …`
3. Each orbital must be filled before moving to the next higher energy level.

**Example:** Carbon (Z=6): `1s² 2s² 2p²`

---

### ④ Heisenberg Uncertainty Principle
**Statement:** It is impossible to simultaneously determine the exact **position** and **momentum** of a particle.

**Mathematical Form:**
```
Δx · Δp ≥ h / 4π
```
Where:
- Δx = uncertainty in position
- Δp = uncertainty in momentum
- h = Planck's constant (6.626 × 10⁻³⁴ J·s)

**Steps to interpret:**
1. The more precisely we know position (small Δx), the less precisely we know momentum (large Δp), and vice versa.
2. This is significant for **subatomic particles** (electrons), not macroscopic objects.
3. It leads to the concept of **orbitals** (probability regions) instead of definite orbits.

---

### ⑤ Hund's Rule of Maximum Multiplicity
**Statement:** Electrons occupy **degenerate orbitals** (same energy) **singly** before pairing, and single electrons have the **same spin**.

**Steps:**
1. Fill each orbital in a subshell with one electron first.
2. All singly occupied orbitals have parallel spins (↑↑↑ before ↑↓).
3. This minimizes electron-electron repulsion.

**Example:** Nitrogen (Z=7), 2p subshell:
```
2p: ↑  ↑  ↑   (NOT ↑↓  ↑  _ )
```

---

### ⑥ Ionization Potential (IP)
**Definition:** The minimum energy required to **remove the outermost electron** from a gaseous atom.

```
M(g) → M⁺(g) + e⁻    ΔH = IP₁ (First Ionization Energy)
```

**Factors affecting IP:**
1. Nuclear charge (↑ Z → ↑ IP)
2. Atomic radius (↑ radius → ↓ IP)
3. Shielding effect (↑ shielding → ↓ IP)
4. Penetration effect (s > p > d > f penetration → higher IP for s electrons)

**Example:** He has highest IP (2372 kJ/mol); Cs has the lowest among metals.

---

### ⑦ Electron Affinity (EA)
**Definition:** The energy **released** when a neutral gaseous atom **gains an electron** to form a negative ion.

```
X(g) + e⁻ → X⁻(g)    ΔH = −EA (energy released)
```

**Steps:**
1. Higher EA = stronger tendency to gain electrons.
2. EA generally increases across a period (left → right) and decreases down a group.
3. Chlorine has the highest EA among all elements (349 kJ/mol).
4. Noble gases have near-zero or negative EA (electron goes to a new shell — energetically unfavorable).

## Q2. Quantum Numbers When n = 4

**Step 1:** Principal quantum number, **n = 4**

**Step 2:** Azimuthal (angular momentum) quantum number, **l = 0, 1, 2, 3**

| l value | Subshell |
|---------|----------|
| 0 | 4s |
| 1 | 4p |
| 2 | 4d |
| 3 | 4f |

**Step 3:** Magnetic quantum number **mₗ = −l to +l**

| l | mₗ values | No. of orbitals |
|---|-----------|-----------------|
| 0 | 0 | 1 |
| 1 | −1, 0, +1 | 3 |
| 2 | −2, −1, 0, +1, +2 | 5 |
| 3 | −3, −2, −1, 0, +1, +2, +3 | 7 |

**Step 4:** Spin quantum number **mₛ = +½ or −½** (for each orbital, 2 values always)

**Total orbitals in n=4 shell:** 1+3+5+7 = **16 orbitals**
**Total electrons:** 16 × 2 = **32 electrons**

## Q3. Trends in Atomic and Ionic Size

### Atomic Radius

**Across a Period (left → right):**
1. Nuclear charge (Z) increases.
2. Electrons are added to the **same shell** (shielding doesn't increase proportionally).
3. Effective nuclear charge (Zeff) increases → electrons pulled closer.
4. **Result: Atomic radius decreases** across a period.

**Down a Group (top → bottom):**
1. New electron shells are added.
2. Shielding by inner electrons increases.
3. Despite increasing Z, the outermost electrons are farther from nucleus.
4. **Result: Atomic radius increases** down a group.

### Ionic Radius

**Cations (M⁺, M²⁺):**
- Formed by **losing** electrons.
- Fewer electrons → less electron-electron repulsion → nuclear charge dominates.
- **Cation is always smaller than the parent atom.**

**Anions (X⁻, X²⁻):**
- Formed by **gaining** electrons.
- More electrons → greater repulsion → electron cloud expands.
- **Anion is always larger than the parent atom.**

**Isoelectronic series:** For species with same number of electrons, the one with higher Z has smaller radius.
- Example: N³⁻ > O²⁻ > F⁻ > Ne > Na⁺ > Mg²⁺ > Al³⁺

**Trend: IP decreases down the group**

| Element | 1st IP (kJ/mol) |

| Be | 900 |

| O | 1314 |

**General trend: IP increases left → right** (Zeff increases)

1. **Be > B:** B loses a 2p electron (higher energy than 2s of Be) → easier to remove.

## Q6. Shapes and Orientations of s, p, and d Orbitals

### s Orbital (l = 0)
- **Shape:** Spherical
- **Orientation:** Only 1 orbital (mₗ = 0), no directional preference
- **Examples:** 1s, 2s, 3s

### p Orbitals (l = 1)
- **Shape:** Dumbbell (two lobes on either side of nucleus)
- **Orientation:** 3 orbitals along x, y, z axes
  - pₓ: along x-axis
  - pᵧ: along y-axis
  - p_z: along z-axis

### d Orbitals (l = 2)
- **Shape:** Complex (mostly double-dumbbell or dumbbell with ring)
- **5 orientations (mₗ = −2 to +2):**

| Orbital | Shape | Orientation |
|---------|-------|-------------|
| d_z² | Dumbbell + ring | Along z-axis with toroidal ring in xy-plane |
| d_x²−y² | 4-lobed | Along x and y axes |
| d_xy | 4-lobed | Between x and y axes |
| d_xz | 4-lobed | Between x and z axes |
| d_yz | 4-lobed | Between y and z axes |

**Note:** The first two (d_z², d_x²−y²) are the **e_g** set; the last three are the **t₂g** set (important in crystal field theory).

## Q7. Quantum Numbers and Their Relationships

| Quantum Number | Symbol | Values | Physical Meaning |
|----------------|--------|--------|-----------------|
| Principal | n | 1, 2, 3, … | Shell/energy level; determines size and energy of orbital |
| Azimuthal | l | 0 to (n−1) | Subshell; determines shape of orbital |
| Magnetic | mₗ | −l to +l | Orientation of orbital in space |
| Spin | mₛ | +½ or −½ | Intrinsic spin of electron |

**Relationships:**
1. For a given n: number of subshells = n
2. For a given l: number of orbitals = (2l + 1)
3. For a given n: total orbitals = n²
4. For a given n: total electrons = 2n²
5. l depends on n: `l = 0, 1, 2, … (n−1)`
6. mₗ depends on l: `mₗ = −l, …, 0, …, +l`

# 2. Chemical Bonding

## Q1. Chemical Bond — Classification and Differentiation

### Definition
A **chemical bond** is the attractive force that holds two atoms or ions together in a compound, formed by the redistribution of electrons to achieve a stable (usually noble gas) configuration.

### Classification

| Type | Formed By | Example |
|------|-----------|---------|
| **Ionic Bond** | Transfer of electrons | NaCl, MgO |
| **Covalent Bond** | Sharing of electrons | H₂, H₂O, CH₄ |
| **Coordinate/Dative Bond** | Sharing by one atom only | NH₄⁺, [Cu(NH₃)₄]²⁺ |
| **Metallic Bond** | Sea of delocalized electrons | Fe, Cu, Na |
| **Hydrogen Bond** | Electrostatic attraction via H | H₂O, HF |
| **Van der Waals** | Weak induced dipole forces | Noble gases, CH₄ |

### Ionic vs Covalent vs Coordinate

| Property | Ionic | Covalent | Coordinate |
|----------|-------|----------|------------|
| Formation | Transfer of e⁻ | Sharing of e⁻ | One atom donates both e⁻ |
| Bond polarity | Fully polar | Non-polar to polar | Polar |
| Melting point | High | Low to moderate | Moderate |
| Conductivity | Good (in solution/melt) | Poor | Poor |
| Solubility | Polar solvents | Non-polar solvents | Variable |
| Example | NaCl | Cl₂, HCl | NH₄⁺, BF₄⁻ |

---

## Q2. MOT vs VBT

### Valence Bond Theory (VBT)
1. A covalent bond forms when half-filled orbitals of two atoms **overlap**.
2. Greater the overlap → stronger the bond.
3. Explains hybridization (sp, sp², sp³, dsp², d²sp³).
4. **Limitation:** Cannot explain the paramagnetic nature of O₂; fails for electron-deficient molecules.

### Molecular Orbital Theory (MOT)
1. Atomic orbitals combine to form **molecular orbitals** (MOs).
2. Two types: **Bonding MO** (σ, π) and **Antibonding MO** (σ*, π*).
3. Electrons fill MOs using Aufbau, Pauli, and Hund's rules.
4. **Advantage:** Correctly predicts paramagnetism of O₂.

### Differences

| Feature | VBT | MOT |
|---------|-----|-----|
| Electron location | Localized between atoms | Delocalized over molecule |
| Orbital type | Atomic orbitals | Molecular orbitals |
| Bond order | From Lewis structure | From (bonding − antibonding)/2 |
| Paramagnetism of O₂ | Cannot explain | Correctly explains |
| Resonance | Needed as workaround | Not needed |

---

## Q3. Bond Order

**Definition:** Bond order = (No. of electrons in bonding MOs − No. of electrons in antibonding MOs) / 2

**MO filling order:**
```
σ1s, σ*1s, σ2s, σ*2s, σ2p, π2p(x)=π2p(y), π*2p(x)=π*2p(y), σ*2p
```
*(For Z ≤ 7: π2p comes before σ2p)*

### N₂ (14 electrons)
```
σ1s²  σ*1s²  σ2s²  σ*2s²  π2px²  π2py²  σ2pz²
Bonding: 8,  Antibonding: 4
Bond Order = (8−4)/2 = 2 ... wait, recounting:
Bonding MOs: σ1s(2) + σ2s(2) + π2px(2) + π2py(2) + σ2pz(2) = 10
Antibonding MOs: σ*1s(2) + σ*2s(2) = 4
Bond Order = (10−4)/2 = 3 ✓
```
N₂ has a **triple bond** — very stable.

### O₂ (16 electrons)
```
σ1s²  σ*1s²  σ2s²  σ*2s²  σ2pz²  π2px²  π2py²  π*2px¹  π*2py¹
Bonding: 10,  Antibonding: 6
Bond Order = (10−6)/2 = 2
```
O₂ has a **double bond** and is **paramagnetic** (2 unpaired electrons in π*).

## Q4. Why PCl₅ Exists but NCl₅ Does Not

**Step 1:** PCl₅ requires 5 bonds → P needs 5 bond pairs.

**Step 2:** Phosphorus (P) — Electronic configuration: `[Ne] 3s² 3p³ 3d⁰`
- P has **vacant 3d orbitals** available for bonding.
- In PCl₅, P undergoes **sp³d hybridization** using 3s, 3p (×3), 3d orbitals.
- This accommodates 5 bond pairs → PCl₅ forms.

**Step 3:** Nitrogen (N) — Electronic configuration: `[He] 2s² 2p³`
- N is in Period 2 → **no d orbitals available** in the valence shell (n=2 has only s and p).
- Maximum covalency of N = 4 (using one s + three p orbitals).
- **NCl₅ cannot form** because N cannot expand its octet.

**Conclusion:** The presence of accessible d orbitals in P (but not N) allows PCl₅ to exist.

## Q5. Hybridization — sp³ and dsp²

### Definition
Hybridization is the **mixing of atomic orbitals of similar energy** to form a new set of equivalent hybrid orbitals with specific geometry.

### sp³ Hybridization
**Step 1:** One s + three p orbitals mix → 4 equivalent sp³ hybrid orbitals.

**Step 2:** Geometry = **Tetrahedral**, bond angle = **109.5°**

**Step 3:** Example — CH₄ (Methane):
```
C: 2s¹ 2p³ (after promotion) → 4 sp³ orbitals → bonds with 4 H atoms
```

| Molecule | Hybridization | Shape | Angle |
|----------|---------------|-------|-------|
| CH₄ | sp³ | Tetrahedral | 109.5° |
| NH₃ | sp³ | Trigonal pyramidal | 107° |
| H₂O | sp³ | Bent/V-shaped | 104.5° |

### dsp² Hybridization
**Step 1:** One d + one s + two p orbitals → 4 equivalent dsp² orbitals.

**Step 2:** Geometry = **Square planar**, bond angle = **90°**

**Step 3:** Example — [Ni(CN)₄]²⁻, [PtCl₄]²⁻:
- Ni²⁺ uses `3d, 4s, 4p, 4p` → dsp² → square planar complex.

## Q6. Fajan's Rules

**Purpose:** Predicts the degree of **covalent character** in an ionic compound.

A compound has more covalent character if:

| Rule | Condition | Reason |
|------|-----------|--------|
| 1 | **High charge on cation** (Z⁺ high) | Small, highly charged cation has high polarizing power |
| 2 | **Small size of cation** | Higher charge density → greater distortion of anion |
| 3 | **Large size of anion** | Larger anion is more polarizable (electron cloud easily distorted) |
| 4 | **High charge on anion** | More electrons → more polarizable |
| 5 | **Pseudo noble gas config. of cation** | 18e⁻ config. (e.g., Cu⁺, Zn²⁺) polarizes more than noble gas config. (e.g., Na⁺) |

**Examples:**
- AgCl has more covalent character than NaCl (Ag⁺ has pseudo-noble gas configuration and is more polarizing).
- AlCl₃ is more covalent than NaCl (Al³⁺ is small with high charge).

# 3. Complex Compounds

## Q1. Theories of Complex Compounds

### ① Werner's Theory (1893)
**Postulates:**
1. Metals in complex compounds exhibit **two types of valency**:
   - **Primary valency** (ionizable) = oxidation state of metal.
   - **Secondary valency** (non-ionizable) = coordination number.
2. Primary valency is satisfied by anions.
3. Secondary valency is satisfied by neutral molecules or anions (ligands) directed in space.

**Example:** [Co(NH₃)₆]Cl₃
- Primary valency = 3 (Co³⁺)
- Secondary valency = 6 (6 NH₃ ligands)

---

### ② Sidgwick's EAN Theory (Effective Atomic Number)
**Concept:** A metal atom in a complex gains electrons from ligands until its EAN equals that of the **nearest noble gas**.

```
EAN = Atomic number of metal − oxidation state + 2 × (number of ligands donating e⁻ pairs)
```

**Example:** [Fe(CO)₅]
- Fe (Z=26), oxidation state = 0
- EAN = 26 − 0 + 2×5 = 36 (= Kr) ✓

**Limitation:** Not universally applicable (many stable complexes don't satisfy EAN).

---

### ③ Pauling's Valence Bond Theory (VBT) for Complexes
**Steps:**
1. Metal provides empty hybrid orbitals.
2. Ligands donate lone pairs into these orbitals (**coordinate bonds**).
3. Type of hybridization determines geometry.

| Hybridization | Coordination No. | Geometry |
|---------------|-----------------|----------|
| sp³ | 4 | Tetrahedral |
| dsp² | 4 | Square planar |
| sp³d² | 6 | Octahedral |
| d²sp³ | 6 | Octahedral (inner orbital) |

**Inner orbital complex:** Uses (n−1)d orbitals → low spin, diamagnetic.
**Outer orbital complex:** Uses nd orbitals → high spin, paramagnetic.

---

### ④ Crystal Field Theory (CFT)

**Basic Idea:**
1. Ligands are treated as **point charges** (purely electrostatic model).
2. Approach of ligands causes **splitting** of degenerate d orbitals.

**Octahedral splitting:**
- Ligands approach along x, y, z axes.
- d_z² and d_x²−y² (e_g set) → higher energy (closer to ligands).
- d_xy, d_xz, d_yz (t₂g set) → lower energy.
- Energy gap = **Δₒ (crystal field splitting energy)**.

```
      ___  ___       ← e_g (+0.6Δₒ each)
         |
        Δₒ
         |
   ___  ___  ___    ← t₂g (−0.4Δₒ each)
```

**High spin vs Low spin:**
- **Strong field ligands** (CN⁻, CO, NO₂⁻): large Δₒ → low spin (electrons pair up in t₂g).
- **Weak field ligands** (F⁻, Cl⁻, H₂O): small Δₒ → high spin (electrons occupy e_g before pairing).

**Spectrochemical series (weak → strong):**
```
I⁻ < Br⁻ < Cl⁻ < F⁻ < OH⁻ < H₂O < NH₃ < en < CN⁻ < CO
```

**Merits of CFT:**
1. Explains color of complexes (d-d transitions).
2. Explains magnetic properties (paired vs unpaired electrons).
3. Explains stability differences.
4. Predicts geometry.

## Q2. Stability of Complexes

**Stability constant (β or K_f):**
```
[M] + n[L] ⇌ [MLn]
β = [MLn] / ([M][L]ⁿ)
```
Higher β = more stable complex.

**Factors affecting stability:**

| Factor | Effect | Example |
|--------|--------|---------|
| Charge on metal ion | Higher charge → more stable | Fe³⁺ complexes > Fe²⁺ complexes |
| Size of metal ion | Smaller metal ion → more stable | Al³⁺ > Tl³⁺ |
| Nature of ligand (basicity) | Stronger base → more stable | en > NH₃ |
| Chelate effect | Chelating ligands → much more stable | EDTA complexes very stable |
| Hard-soft matching | Hard acid+hard base or soft+soft → stable | Co³⁺ with NH₃ |

**Chelate effect:** A bidentate ligand (en) forms more stable complexes than a monodentate (NH₃) due to **entropic advantage** — one chelate releases more solvent molecules.

## Q3. Paramagnetic and Diamagnetic Complexes

### Definitions
- **Paramagnetic:** Has **unpaired electrons**; attracted by magnetic field. Magnetic moment μ = √[n(n+2)] BM (where n = unpaired electrons).
- **Diamagnetic:** All electrons **paired**; weakly repelled by magnetic field; μ ≈ 0.

### K₃[Fe(CN)₆] — Paramagnetic (Hexacyanoferrate(III))

**Step 1:** Central metal = Fe³⁺ (d⁵ configuration)
**Step 2:** CN⁻ is a **strong field ligand** (large Δₒ)
**Step 3:** d⁵ in strong field:
```
t₂g: ↑↓  ↑↓  ↑      e_g: _  _
       → 1 unpaired electron
```
**Step 4:** 1 unpaired electron → **paramagnetic**
μ = √[1(1+2)] = √3 ≈ 1.73 BM

### K₄[Fe(CN)₆] — Diamagnetic (Hexacyanoferrate(II))

**Step 1:** Central metal = Fe²⁺ (d⁶ configuration)
**Step 2:** CN⁻ is a **strong field ligand**
**Step 3:** d⁶ in strong field:
```
t₂g: ↑↓  ↑↓  ↑↓    e_g: _  _
       → 0 unpaired electrons
```
**Step 4:** All electrons paired → **diamagnetic**
μ = 0 BM

## Q4. Geometrical and Optical Isomerism

### Geometrical Isomerism
Arises due to **restricted rotation** around a double bond or in square planar/octahedral complexes.

**Square Planar Example — [Pt(NH₃)₂Cl₂]:**
- **cis isomer:** Both Cl on same side → polar
- **trans isomer:** Cl on opposite sides → non-polar
- Cis-platin (cis isomer) is an **anticancer drug**.

**Octahedral Example — [Co(NH₃)₄Cl₂]⁺:**
- cis: both Cl adjacent
- trans: both Cl opposite

### Optical Isomerism
Arises when a molecule has a **non-superimposable mirror image** (chiral molecule).

**Conditions:**
1. No plane of symmetry.
2. No center of inversion.

**Example — [Co(en)₃]³⁺:**
- The two isomers (d and l, or Δ and Λ) rotate plane-polarized light in opposite directions.
- Called **enantiomers**.
- They have identical physical properties but differ in optical rotation.

## Q5. Applications of Complex Compounds

### Textile Field
1. **Mordant dyeing:** Metal complexes (Al³⁺, Cr³⁺) act as mordants — they form coordination bonds with both the fiber and the dye, improving color fastness.
   - Example: Chrome mordant with alizarin dye.
2. **Vat dyes** involve reduction/oxidation processes where metal complexes catalyze color development.
3. **Direct dyes** sometimes incorporate metal complexes for brighter, more stable colors.
4. **Electroplating of textiles** — metallic finishes use complex salts (e.g., silver complexes).

### Non-Textile Field
1. **Medicine:** Cisplatin — anticancer drug; EDTA complexes treat heavy metal poisoning.
2. **Photography:** AgCl is dissolved in Na₂S₂O₃ (hypo) as [Ag(S₂O₃)₂]³⁻ during fixing.
3. **Extraction of metals:** Cyanide complexes extract gold: Au + CN⁻ → [Au(CN)₂]⁻.
4. **Electroplating:** Copper deposited from [Cu(CN)₄]²⁻ gives uniform coating.
5. **Catalysis:** [Co] and [Rh] complexes used in hydroformylation, Wacker process.
6. **Water treatment:** EDTA complexes remove Ca²⁺ and Mg²⁺ (softening hard water).

# 4. Acid, Bases and Analytical Chemistry

## Q1. Concepts of Acids and Bases

### ① Arrhenius Concept
- **Acid:** Substance that produces **H⁺ (proton)** in water.
- **Base:** Substance that produces **OH⁻** in water.
- **Limitation:** Only applies to aqueous solutions; cannot explain NH₃ or CO₂ as bases/acids in non-aqueous systems.

**Examples:**
```
HCl → H⁺ + Cl⁻       (acid)
NaOH → Na⁺ + OH⁻     (base)
```

### ② Brønsted-Lowry Concept (Protonic Theory)
- **Acid:** Proton (**H⁺**) donor.
- **Base:** Proton (**H⁺**) acceptor.
- Broader — applies to non-aqueous systems.

**Example:**
```
HCl + H₂O → H₃O⁺ + Cl⁻
Acid  Base    Conj.acid  Conj.base
```
- HCl and Cl⁻ are a **conjugate acid-base pair**.
- H₂O and H₃O⁺ are a **conjugate acid-base pair**.

### ③ Lewis Concept (Electronic Theory)
- **Acid:** Electron pair **acceptor** (has empty orbital).
- **Base:** Electron pair **donor** (has lone pair).
- Broadest concept — covers reactions with no proton transfer.

**Examples:**
```
BF₃ + :NH₃ → F₃B←NH₃
(Lewis acid)  (Lewis base)

Cu²⁺ + 4NH₃ → [Cu(NH₃)₄]²⁺
(Lewis acid)   (Lewis base)
```

---

## Q2. Measuring Strength of Acids and Bases

### Acid Strength

**Step 1:** Strong acids completely dissociate:
```
HCl → H⁺ + Cl⁻    (Ka → ∞)
```

**Step 2:** Weak acids partially dissociate:
```
CH₃COOH ⇌ H⁺ + CH₃COO⁻    Ka = [H⁺][CH₃COO⁻] / [CH₃COOH]
```
Larger Ka = stronger acid.

**Step 3:** pH scale:
```
pH = −log[H⁺]
```
Lower pH → stronger acid.

**Step 4:** Factors affecting acid strength:
1. Bond polarity (more polar H-A bond → stronger acid)
2. Bond strength (weaker H-A bond → stronger acid; down a group)
3. Electronegativity of central atom (across a period)
4. Inductive effects in organic acids

### Base Strength
- Measured by **Kb** or **pKb**.
- Strong base: completely dissociates → [OH⁻] high → high pH.
- pKa + pKb = 14 (for a conjugate pair at 25°C).

## Q3. Important Terms

### Amphiprotic Species
A species that can act as both **acid and base** (proton donor AND acceptor).

**Examples:**
- H₂O: accepts H⁺ → H₃O⁺ (base) / donates H⁺ → OH⁻ (acid)
- HCO₃⁻: accepts H⁺ → H₂CO₃ (base) / donates H⁺ → CO₃²⁻ (acid)
- H₂PO₄⁻, HPO₄²⁻, amino acids

---

### Electrophile and Nucleophile

| Term | Definition | Nature | Examples |
|------|-----------|--------|---------|
| **Electrophile** | Electron pair **acceptor** (Lewis acid) | Electron-deficient | H⁺, BF₃, AlCl₃, carbocations (R⁺) |
| **Nucleophile** | Electron pair **donor** (Lewis base) | Electron-rich | OH⁻, NH₃, Cl⁻, CN⁻, H₂O |

---

### Conjugate Acid-Base Pair
A pair that differs by **one proton (H⁺)**.

```
Acid  ⇌  Conjugate base + H⁺
HCl   →   Cl⁻
H₂SO₄ →   HSO₄⁻
NH₄⁺  →   NH₃
H₂O  →   OH⁻
```

**Rule:** Stronger the acid → weaker its conjugate base; and vice versa.

---

### Hard and Soft Acids and Bases (HSAB Theory — Pearson)

| Category | Characteristics | Examples |
|----------|----------------|---------|
| **Hard Acid** | Small, high charge, low polarizability | H⁺, Li⁺, Na⁺, Mg²⁺, Al³⁺, Fe³⁺ |
| **Soft Acid** | Large, low charge, high polarizability | Cu⁺, Ag⁺, Hg²⁺, Pt²⁺ |
| **Hard Base** | Small, high electronegativity, low polarizability | F⁻, OH⁻, H₂O, NH₃, Cl⁻ |
| **Soft Base** | Large, low electronegativity, high polarizability | I⁻, SCN⁻, CN⁻, S²⁻, CO |

**HSAB Principle:** Hard prefers Hard; Soft prefers Soft → more stable compounds form.

**Examples:**
- AgF (soft acid + hard base) — unstable
- AgI (soft acid + soft base) — more stable
- AlF₃ (hard acid + hard base) — very stable

## Q5. Precision, Accuracy, Errors, and Significant Figures

### Precision vs Accuracy

| Term | Definition | Analogy |
|------|-----------|---------|
| **Accuracy** | Closeness to the **true/accepted value** | Arrow hitting the bullseye |
| **Precision** | Closeness of **repeated measurements** to each other | Arrows clustered together (even if off-center) |

A measurement can be precise but not accurate (systematic error); or accurate but not precise (random error).

---

### Mean (Average)
```
x̄ = (x₁ + x₂ + x₃ + … + xₙ) / n
```

---

### Absolute Error
```
Absolute Error = |Measured Value − True Value|
```
Has units of the measurement.

---

### Relative Error
```
Relative Error = Absolute Error / True Value
```
Expressed as a fraction or percentage:
```
% Relative Error = (Absolute Error / True Value) × 100
```

---

### Significant Figures

**Rules for counting sig figs:**
1. All non-zero digits are significant. (e.g., 453 → 3 sig figs)
2. Zeros between non-zero digits are significant. (e.g., 1002 → 4 sig figs)
3. Leading zeros are NOT significant. (e.g., 0.0052 → 2 sig figs)
4. Trailing zeros after decimal point ARE significant. (e.g., 3.50 → 3 sig figs)
5. Trailing zeros without decimal point are ambiguous. (e.g., 1500 → 2, 3, or 4?)

**Operations:**
- **Addition/Subtraction:** Round to the least decimal places.
- **Multiplication/Division:** Round to the least number of significant figures.

**Insignificant figures:** Digits that are not reliable; beyond the precision of the measuring instrument.

# 5. Physical Chemistry

## Q1. Colligative Properties and Raoult's Law (Vapour Pressure Lowering)

### Colligative Properties
Properties that depend only on the **number of solute particles**, not their nature.

| Property | Symbol |
|----------|--------|
| Relative lowering of vapour pressure | ΔP/P° |
| Elevation of boiling point | ΔTb |
| Depression of freezing point | ΔTf |
| Osmotic pressure | π |

---

### Raoult's Law of Lowering of Vapour Pressure

**Statement:** The vapour pressure of a solvent over a solution is **directly proportional** to the mole fraction of the solvent.

```
P_solvent = x_solvent × P°_solvent
```

Where:
- P_solvent = vapour pressure of solvent in solution
- P°_solvent = vapour pressure of pure solvent
- x_solvent = mole fraction of solvent

**Derivation of lowering:**
```
ΔP = P° − P = P° × x_solute
ΔP/P° = x_solute = nB / (nA + nB)
```
For dilute solution (nB << nA):
```
ΔP/P° ≈ nB/nA = (wB/MB) / (wA/MA) = Relative lowering of vapour pressure
```

### Limitations of Raoult's Law:
1. Valid only for **ideal solutions** (no solute-solvent interaction different from solvent-solvent).
2. **Positive deviation:** If solute-solvent interaction < solvent-solvent interaction (e.g., ethanol + water → higher vapour pressure).
3. **Negative deviation:** If solute-solvent interaction > solvent-solvent interaction (e.g., HCl + water → lower vapour pressure than expected).
4. Does not apply to **electrolytes** directly (need van't Hoff factor, i).

---

## Q2. Elevation of Boiling Point (ΔTb ∝ m)

### Concept
Adding a non-volatile solute **raises the boiling point** (solute lowers vapour pressure → higher T needed to reach atmospheric pressure).

### Proof that ΔTb ∝ m

**Step 1:** At boiling point, vapour pressure = atmospheric pressure (P_atm).
Adding solute: vapour pressure decreases by ΔP → need to raise T to Tb' to restore vapour pressure.

**Step 2:** Using Clausius-Clapeyron equation:
```
ΔP/P° = ΔHvap × ΔTb / (RTb²)
```

**Step 3:** From Raoult's law:
```
ΔP/P° = x_B = nB/(nA+nB) ≈ nB/nA  (dilute solution)
```

**Step 4:** nB/nA in terms of molality (m):
```
nB/nA = (wB/MB) / (wA/MA) = (wB × MA) / (MB × wA)
molality m = (wB × 1000) / (MB × wA in g)
So: nB/nA = m × MA/1000
```

**Step 5:** Substituting:
```
m × MA/1000 = ΔHvap × ΔTb / (RTb²)
ΔTb = [R × Tb² × MA / (1000 × ΔHvap)] × m
ΔTb = Kb × m
```
Where **Kb** = ebullioscopic (boiling point elevation) constant.

**Therefore: ΔTb ∝ m** ✓

---

## Q3. Depression of Freezing Point (ΔTf ∝ m)

### Concept
A non-volatile solute **lowers the freezing point** (solute lowers vapour pressure of solution below that of solid solvent → must cool more for equilibrium).

### Proof that ΔTf ∝ m

**Step 1:** At freezing point Tf, vapour pressure of liquid = vapour pressure of solid.
Adding solute lowers liquid vapour pressure → now need lower temperature Tf' to match solid vapour pressure.

**Step 2:** Using Clausius-Clapeyron:
```
ΔP/P° = ΔHfus × ΔTf / (RTf²)
```

**Step 3:** From Raoult's law (as before):
```
ΔP/P° ≈ nB/nA = m × MA/1000
```

**Step 4:** Equating:
```
m × MA/1000 = ΔHfus × ΔTf / (RTf²)
ΔTf = [R × Tf² × MA / (1000 × ΔHfus)] × m
ΔTf = Kf × m
```
Where **Kf** = cryoscopic (freezing point depression) constant.

**Therefore: ΔTf ∝ m** ✓

---

## Q4. Osmotic Pressure

**Definition:** The minimum pressure required to prevent the flow of solvent through a semi-permeable membrane from pure solvent to solution; or equivalently, the pressure needed to stop osmosis.

### Laws of Osmotic Pressure (van't Hoff)

**1. Effect of Concentration:**
```
π ∝ C    (at constant T)
```

**2. Effect of Temperature:**
```
π ∝ T    (at constant C)
```

**3. Combined (van't Hoff equation):**
```
πV = nRT
π = CRT = (n/V)RT
```
Where:
- π = osmotic pressure (atm or Pa)
- C = molar concentration (mol/L)
- R = gas constant
- T = absolute temperature (K)

**This is analogous to the ideal gas law!**

---

## Q5. Prove: Osmotic Pressure ∝ Lowering of Vapour Pressure

**Step 1:** Van't Hoff equation for dilute solution:
```
π = CRT = (nB/V) × RT
```

**Step 2:** Raoult's law (lowering of vapour pressure):
```
ΔP = P° × xB = P° × nB/(nA+nB) ≈ P° × nB/nA
```

**Step 3:** Nernst equation relates vapour pressure to osmotic work:
```
π × V_solvent = RT × ln(P°/P)
For small ΔP: ln(P°/P) ≈ ΔP/P°
π × V̄A = RT × (ΔP/P°)
```

**Step 4:** Therefore:
```
π = RT × ΔP / (P° × V̄A)
π ∝ ΔP    (at constant T, for a given solvent)
```
**Therefore: Osmotic pressure ∝ Lowering of vapour pressure** ✓

---

## Q6. Prove: Osmotic Pressure ∝ Relative Lowering of Vapour Pressure (ΔP/P°)

**Step 1:** From van't Hoff: π = CRT

**Step 2:** Relative lowering = ΔP/P° = xB ≈ nB/nA (dilute)

**Step 3:** Concentration: C = nB/V

**Step 4:** nA = (mass of solvent/MA) = (1000/MA) per 1 kg solvent

In dilute solution: nB/nA ≈ nB × MA/1000

**Step 5:** π = (nB/V)RT and ΔP/P° = nB/nA

Both proportional to nB (number of solute moles) in dilute solution at constant T.

```
π = CRT = (nB/V) × RT
ΔP/P° = nB/nA = nB × MA / (1000 × V × ρ_A)
```

**Ratio π / (ΔP/P°) = constant** (for given T and solvent)

**Therefore: π ∝ ΔP/P°** ✓

# 6. Chemical Equilibrium

## Q1. Chemical Equilibrium, Law of Mass Action, Kc and Kp

### Chemical Equilibrium
A state where **forward and reverse reaction rates are equal**, and concentrations of reactants and products remain constant over time.

**Characteristics:**
1. Dynamic (both reactions still occurring)
2. Concentrations are constant (not equal)
3. Can be approached from either direction
4. Only reached in **closed systems**

---

### Law of Mass Action (Guldberg and Waage, 1864)
**Statement:** The rate of a chemical reaction is proportional to the product of the **molar concentrations** of reactants, each raised to the power of its **stoichiometric coefficient**.

For: `aA + bB ⇌ cC + dD`

**Equilibrium constant in terms of concentration:**
```
Kc = [C]^c [D]^d / [A]^a [B]^b
```

**Equilibrium constant in terms of partial pressure:**
```
Kp = (P_C)^c (P_D)^d / (P_A)^a (P_B)^b
```

---

### Relationship Between Kc and Kp

**Step 1:** Using ideal gas law: PV = nRT → P = CRT (C = n/V = concentration)

**Step 2:** Substituting into Kp:
```
Kp = Kc × (RT)^Δn
```
Where **Δn = (c + d) − (a + b)** = change in moles of gas.

**Cases:**
- Δn = 0: Kp = Kc
- Δn > 0: Kp > Kc
- Δn < 0: Kp < Kc

---

## Q2. Prove that Kc = Kp = Kx (Special Conditions)

**When Δn = 0:** (e.g., H₂ + I₂ ⇌ 2HI)
```
Δn = 2 − (1+1) = 0
Kp = Kc(RT)⁰ = Kc
```

**Kx** (in terms of mole fractions):
```
Kx = Kp / P^Δn
If Δn = 0: Kx = Kp / P⁰ = Kp
```

**Therefore when Δn = 0:** Kc = Kp = Kx ✓

---

## Q3. Calculate Kc, Kp, Kx

### Example: N₂ + 3H₂ ⇌ 2NH₃

Suppose at equilibrium:
- [N₂] = 0.5 M, [H₂] = 1.5 M, [NH₃] = 0.4 M

```
Kc = [NH₃]² / ([N₂][H₂]³)
Kc = (0.4)² / (0.5 × (1.5)³)
Kc = 0.16 / (0.5 × 3.375)
Kc = 0.16 / 1.6875 = 0.0948
```

```
Δn = 2 − (1+3) = −2
Kp = Kc(RT)^Δn = Kc / (RT)²
```
At T = 500 K, RT = 0.0821 × 500 = 41.05
```
Kp = 0.0948 / (41.05)² = 0.0948 / 1685 = 5.63 × 10⁻⁵
```

---

## Q4. Le Chatelier's Principle

**Statement (Le Chatelier and Braun):** If a system at equilibrium is subjected to a change (stress), the equilibrium will **shift in the direction that partially counteracts the change** and restores a new equilibrium.

### Effect of Temperature

| Reaction type | Increase T | Decrease T |
|---------------|------------|------------|
| Exothermic (ΔH < 0) | Shifts backward | Shifts forward |
| Endothermic (ΔH > 0) | Shifts forward | Shifts backward |

**Effect on K:** Temperature is the only factor that changes K itself.

### Effect of Pressure (for gaseous reactions)

| Change | Effect |
|--------|--------|
| Increase pressure | Shifts toward side with **fewer** moles of gas |
| Decrease pressure | Shifts toward side with **more** moles of gas |
| Δn = 0 | No shift |

**Example:** N₂ + 3H₂ ⇌ 2NH₃ (Δn = −2): Increase P → more NH₃ formed.

### Effect of Concentration

| Change | Effect |
|--------|--------|
| Add reactant | Shifts forward |
| Remove reactant | Shifts backward |
| Add product | Shifts backward |
| Remove product | Shifts forward |

---

## Q5. Activity and Activity Coefficient

### Activity (a)
**Definition:** The **effective concentration** of a species in a non-ideal system, correcting for interactions between particles.

For ideal solutions: **a = concentration (or mole fraction)**
For non-ideal solutions: **a = γ × C** (or γ × x)

For gases: **a = fugacity / standard pressure**

---

### Activity Coefficient (γ)
- A dimensionless correction factor that accounts for **non-ideal behavior**.
- For ideal behavior: γ = 1
- For positive deviations from Raoult's law: γ > 1
- For negative deviations: γ < 1

**Thermodynamic equilibrium constant** uses activities:
```
K = product of activities of products / product of activities of reactants
```

# 7. Chemical Kinetics

## Q1. Velocity, Order, and Molecularity

### Rate (Velocity) of Reaction
```
Rate = −d[A]/dt = +d[P]/dt
```
Units: mol L⁻¹ s⁻¹

### Order of Reaction
- Experimentally determined.
- Sum of exponents in the **rate law**: `Rate = k[A]^m [B]^n`
- Order = m + n
- Can be 0, 1, 2, 3 or even fractional.

### Molecularity
- Theoretical concept.
- Number of molecules/ions that **collide simultaneously** in an elementary step.
- Always a whole number: 1 (unimolecular), 2 (bimolecular), 3 (termolecular).

| Feature | Order | Molecularity |
|---------|-------|-------------|
| Determined by | Experiment | Mechanism |
| Can be fractional | Yes | No |
| Applies to | Overall reaction | Elementary step only |

---

## Q2. Reaction Orders

### Zero Order Reaction
**Rate = k[A]⁰ = k** (rate is constant)

**Integrated rate law:**
```
[A]t = [A]₀ − kt
```

**Half-life:**
```
t₁/₂ = [A]₀ / 2k
```
(depends on initial concentration)

**Graph:** [A] vs t → straight line with slope = −k

**Example:** Decomposition of HI on gold surface; certain enzyme reactions at high substrate concentration.

---

### First Order Reaction
**Rate = k[A]**

**Integrated rate law:**
```
ln[A]t = ln[A]₀ − kt
or
[A]t = [A]₀ × e^(−kt)
```

**Half-life:**
```
t₁/₂ = ln2 / k = 0.693 / k
```
(independent of initial concentration)

**Graph:** ln[A] vs t → straight line with slope = −k

**Examples:** Radioactive decay, first-order decomposition (N₂O₅ → 4NO₂ + O₂), hydrolysis.

---

### Second Order Reaction
**Rate = k[A]²** or **Rate = k[A][B]**

**Integrated rate law:**
```
1/[A]t = 1/[A]₀ + kt
```

**Half-life:**
```
t₁/₂ = 1 / (k[A]₀)
```
(depends on initial concentration)

**Graph:** 1/[A] vs t → straight line with slope = +k

**Example:** NO₂ → NO + ½O₂; saponification of esters.

---

### Third Order Reaction
**Rate = k[A]³** or **Rate = k[A]²[B]**

**Half-life:**
```
t₁/₂ = 1 / (2k[A]₀²)
```

**Example:** 2NO + O₂ → 2NO₂ (overall third order)

---

## Q3. Calculate Rate Constant

### For First Order (given concentration and time):
```
k = (1/t) × ln([A]₀/[A]t)
  = (2.303/t) × log([A]₀/[A]t)
```

**Example:** If [A]₀ = 1.0 M, [A]t = 0.5 M at t = 100 s:
```
k = (2.303/100) × log(1.0/0.5)
k = (2.303/100) × log(2)
k = (2.303/100) × 0.301
k = 6.93 × 10⁻³ s⁻¹
```

### For Zero Order:
```
k = ([A]₀ − [A]t) / t
```

### For Second Order:
```
k = (1/t) × (1/[A]t − 1/[A]₀)
```

---

## Q4. Determining Order of Reaction

### Method 1: Initial Rate Method
1. Run reactions with different initial concentrations.
2. Compare initial rates.
3. If doubling [A] doubles rate → 1st order in A.
4. If doubling [A] quadruples rate → 2nd order in A.

### Method 2: Integrated Rate Law (Graphical)
Test which graph gives a straight line:

| If straight line from | Order |
|----------------------|-------|
| [A] vs t | 0 |
| ln[A] vs t | 1 |
| 1/[A] vs t | 2 |

### Method 3: Half-Life Method
1. Measure successive half-lives.
2. If t₁/₂ is constant → 1st order.
3. If t₁/₂ increases (proportional to [A]₀) → 2nd order.
4. If t₁/₂ decreases → zero order.

### Method 4: Isolation Method
Keep all reactants in large excess except one; measure rate dependence on that one.

## Q5. Catalysis

### Definition
A **catalyst** increases reaction rate by providing an alternative pathway with **lower activation energy (Ea)**, without being consumed.

### Types

#### Homogeneous Catalysis
Catalyst and reactants in same phase.
- Example: H₂SO₄ catalyst in esterification (all liquid)
- SO₂ + ½O₂ → SO₃ (with NO catalyst, gas phase)

#### Heterogeneous Catalysis
Catalyst in different phase from reactants.
- Example: Fe(s) catalyst in N₂ + H₂ → NH₃ (Haber process)
- Steps: adsorption → activation → reaction → desorption

---

#### Acid-Base Catalysis
- **Acid catalysis:** H⁺ donates proton to reactant, activating it. Example: hydrolysis of esters by HCl.
- **Base catalysis:** OH⁻ accepts proton, forming reactive intermediate. Example: Claisen condensation.
- **General acid-base catalysis:** Any proton donor/acceptor acts as catalyst.

---

#### Enzyme Catalysis
Biological catalysts (proteins) that catalyze reactions with **extreme specificity**.

**Steps (Lock and Key mechanism / Induced Fit):**
1. Substrate (S) binds to enzyme active site → **Enzyme-Substrate complex (ES)**.
2. ES undergoes reaction at low activation energy.
3. Products (P) released; enzyme is regenerated.

```
E + S ⇌ ES → E + P
```

**Michaelis-Menten equation:**
```
Rate = Vmax[S] / (Km + [S])
```

**Characteristics:**
1. Highly specific (one enzyme, one substrate type)
2. Very efficient (rate up to 10¹⁷ times faster)
3. Work at mild conditions (body temperature, neutral pH)
4. Can be inhibited

---

#### Catalysis Retardation (Inhibition)
Some substances **decrease the rate** of a catalyzed reaction → **inhibitors** or **negative catalysts**.

**Types:**
1. **Competitive inhibitor:** Blocks active site of enzyme (similar structure to substrate).
2. **Non-competitive inhibitor:** Binds elsewhere, changing enzyme shape.
3. **Auto-inhibition:** Product slows down reaction.

---

#### Catalyst Poisoning
Complete or severe **deactivation of a heterogeneous catalyst** by trace amounts of a substance.

**Mechanism:** Poison strongly adsorbs onto the catalyst surface, blocking active sites.

**Examples:**
- CO poisons Fe catalyst in Haber process.
- As₂O₃ poisons Pt catalyst in H₂SO₄ production (Contact process) → H₂S is also a poison.
- Pb in gasoline poisons catalytic converters in vehicles.

---

# 8. Colloids and Solutions

## Q1. Colloids and Their Types

### Definition
A **colloid** (or colloidal dispersion) is a heterogeneous system where particles of one substance (**dispersed phase**, size 1–1000 nm) are uniformly distributed in another substance (**dispersion medium**).

### Classification by Dispersed Phase and Medium

| Dispersed Phase | Dispersion Medium | Type | Example |
|----------------|------------------|------|---------|
| Gas | Liquid | Foam | Whipped cream, fire extinguisher foam |
| Gas | Solid | Solid foam | Pumice, foam rubber |
| Liquid | Gas | Aerosol | Fog, clouds, mist |
| Liquid | Liquid | Emulsion | Milk, mayonnaise |
| Liquid | Solid | Gel | Butter, jelly, cheese |
| Solid | Gas | Aerosol | Smoke, dust |
| Solid | Liquid | Sol | Paint, starch solution, muddy water |
| Solid | Solid | Solid sol | Alloys, ruby glass |

---

### Lyophilic Colloids
- **Definition:** Colloids where dispersed phase has **high affinity** (love) for the dispersion medium (solvent).
- Also called **solvent-loving** colloids.
- **Steps of formation:** Dispersed phase spontaneously disperses in the medium.
- **Examples:** Gelatin in water, starch in water, rubber in benzene.

### Lyophobic Colloids
- **Definition:** Colloids where dispersed phase has **low affinity** (fear/avoidance) for the dispersion medium.
- Also called **solvent-hating** colloids.
- **Steps of formation:** Require special methods (e.g., electrical dispersion, Bredig's arc method, peptization).
- **Examples:** Gold sol, silver sol, Fe(OH)₃ sol in water.

---

## Q2. Lyophilic vs Lyophobic Colloids

| Property | Lyophilic | Lyophobic |
|----------|-----------|-----------|
| Affinity for medium | High | Low |
| Stability | High (self-stabilizing) | Low (require stabilizer/charge) |
| Preparation | Easy (direct mixing) | Requires special methods |
| Reversibility | Reversible | Irreversible (generally) |
| Surface tension | Markedly lower than medium | Slightly lower |
| Viscosity | Much higher than medium | Similar to medium |
| Effect of electrolytes | Require large amounts to coagulate | Coagulate with small amounts |
| Tyndall effect | Weak | Strong |
| Examples | Gelatin, starch, rubber | Gold sol, Fe(OH)₃ sol |

---

## Q3. Gold Number

### Definition
The **minimum amount of a protective colloid (in milligrams)** required to just prevent the coagulation of **10 mL of standard gold sol** (red gold sol) by the addition of **1 mL of 10% NaCl solution**.

**Key points:**
1. Introduced by **Zsigmondy**.
2. Lower gold number = better protective power (less colloid needed).
3. Measured in mg of colloid per 10 mL gold sol.

| Protective Colloid | Gold Number |
|-------------------|-------------|
| Gelatin | 0.005–0.01 |
| Haemoglobin | 0.03–0.07 |
| Egg albumin | 0.08–0.10 |
| Gum arabic | 0.15–0.25 |
| Potato starch | 25 |

**Interpretation:** Gelatin is the best protective colloid (lowest gold number).

---

### Applications of Colloids

1. **In medicine:** Colloidal medicines (colloidal gold, colloidal sulfur for skin diseases); blood is a colloid; X-ray dye (BaSO₄ colloid).
2. **In food industry:** Milk (emulsion), ice cream (foam+emulsion), butter (gel), mayonnaise (emulsion).
3. **In industry:** Paints and inks (solid in liquid sols); rubber (latex is colloidal); cement setting.
4. **Electrical precipitation (Cottrell precipitator):** Smoke/dust particles (colloids) removed from industrial gas by electrostatic precipitation.
5. **Water purification:** Alum (Al₂(SO₄)₃) coagulates colloidal clay/mud in water.
6. **Tanning of leather:** Chrome tanning uses Cr³⁺ to coagulate colloidal proteins in hide.
7. **Photography:** AgBr in gelatin forms a photographic emulsion.
8. **Textile industry:** Sizing of textiles uses colloidal starch.

---

## Q4. Preparation and Purification of Colloids

### Preparation

**A. Dispersion Methods** (breaking down bulk material):

1. **Mechanical dispersion:** Colloid mill grinds particles to colloidal size.
2. **Electrical (Bredig's arc) method:** Electric arc between metal electrodes in water → metal colloid (Au, Ag).
3. **Peptization:** Addition of a suitable electrolyte (peptizing agent) disperses a freshly formed precipitate into colloid.
   - Example: Fe(OH)₃ precipitate + small FeCl₃ → red Fe(OH)₃ sol (Fe³⁺ adsorbs, gives charge).

**B. Condensation Methods** (building up from smaller particles):

1. **Chemical methods:**
   - **Double decomposition:** As₂S₃ sol from As₂O₃ + H₂S.
   - **Reduction:** Gold sol from AuCl₃ + formaldehyde.
   - **Oxidation:** Sulfur sol from H₂S + SO₂.
   - **Hydrolysis:** Fe(OH)₃ sol from FeCl₃ + hot water.

2. **Physical methods:**
   - **Excessive cooling:** Sulfur in alcohol → poured in water.
   - **Exchange of solvent:** Resin in alcohol → poured in water.

---

### Purification

**1. Dialysis:**
- Colloidal solution placed in a semipermeable membrane bag in water.
- Small ions/molecules diffuse out through membrane; colloid particles remain inside.
- **Electrodialysis:** Apply electric field to speed up removal of electrolytes.

**2. Ultrafiltration:**
- Use ultrafilter paper (graded pore size) — smaller than colloidal particles.
- Colloidal particles retained; crystalloidal impurities pass through.

**3. Ultacentrifugation:**
- Very high-speed centrifugation sediments colloidal particles.
- Used when dialysis is insufficient.

# 9. Photochemistry

## Q1. Key Terms in Photochemistry

### Quantum Yield (Φ)
```
Φ = Number of molecules reacting (or product formed) / Number of photons absorbed
```

- **High quantum yield (Φ >> 1):** One photon triggers a chain reaction.
  - Example: H₂ + Cl₂ → 2HCl (Φ = 10⁶ — each photon initiates chain).
- **Low quantum yield (Φ << 1):** Many photons absorbed, few molecules react (energy lost via fluorescence, heat, etc.).
  - Example: HBr formation from H₂ + Br₂ (Φ ≈ 0.01).
- **Φ = 1:** Every absorbed photon causes one molecule to react.

---

### Luminescence
General term for **emission of light** from a substance NOT due to incandescence (heat).

Subtypes: Fluorescence and Phosphorescence.

---

### Fluorescence
1. Absorption of light → excites electron to higher energy state.
2. Electron rapidly loses some energy (non-radiative, vibrational relaxation) — stays in lowest excited singlet state (S₁).
3. Falls back to ground state → emits photon of **longer wavelength** (lower energy) than absorbed.
4. **Occurs very rapidly (~10⁻⁸ s); stops immediately when illumination stops.**
5. Examples: Fluorescein, rhodamine B, quinine sulfate, chlorophyll.

---

### Phosphorescence
1. Similar to fluorescence but involves **intersystem crossing** from excited singlet (S₁) → triplet state (T₁).
2. T₁ → ground state is spin-forbidden → much slower emission.
3. **Glow persists after light source is removed (milliseconds to hours).**
4. Emits at even longer wavelengths than fluorescence.
5. Examples: ZnS (glow-in-the-dark material), organic dyes at low temperature.

**Key Difference:**
| Feature | Fluorescence | Phosphorescence |
|---------|-------------|----------------|
| State | Singlet excited → Ground | Triplet excited → Ground |
| Duration | Very fast (nanoseconds) | Slow (ms to hours) |
| After illumination | Stops immediately | Continues glowing |

---

### Chemiluminescence
- Emission of light as a result of a **chemical reaction** (no heat involved).
- Chemical energy → electronic excitation → light.
- **Examples:**
  - Firefly bioluminescence (luciferin + O₂, enzyme luciferase → light)
  - Glow sticks (reaction of oxalic esters with H₂O₂)
  - White phosphorus oxidation in air → faint glow

---

## Q2. Laws of Photochemistry

### Law I — Grotthuss-Draper Law
**Statement:** Only the light that is **absorbed** by a substance can cause a photochemical reaction. Light that is transmitted or reflected has no photochemical effect.

**Implication:** A substance must have chromophore groups that absorb the incident radiation.

---

### Law II — Stark-Einstein Law (Law of Photochemical Equivalence)
**Statement:** Each molecule (or atom) that takes part in a photochemical reaction absorbs **one photon** of the radiation causing the reaction.

```
Energy of one photon = hν = hc/λ
One mole of photons (1 Einstein) = Nhν = Nhc/λ
```
Where N = Avogadro's number.

**Implication:** This is the basis of quantum yield calculation. In practice, secondary processes (chain reactions, energy transfer) cause Φ to deviate from 1.

---

### Beer-Lambert Law

**Statement:** When monochromatic light passes through a homogeneous absorbing medium, the **absorbance is proportional** to both the concentration of the absorbing species and the path length.

```
A = log(I₀/I) = ε × c × l
```

Where:
- A = Absorbance (no units)
- I₀ = Incident light intensity
- I = Transmitted light intensity
- ε = Molar extinction coefficient (L mol⁻¹ cm⁻¹)
- c = Molar concentration (mol/L)
- l = Path length (cm)

**Transmittance (T) = I/I₀**
```
A = −log T = 2 − log(%T)
```

**Limitations:**
1. Valid only for monochromatic light.
2. Fails at high concentrations (solute-solute interactions).
3. Fails if solution fluoresces or scatters light.
4. Depends on refractive index at high concentrations.

---

## Q3. Photosensitization and Photosensitized Reactions

### Photosensitization
A process where a **substance (photosensitizer)** absorbs radiation and transfers energy to another molecule (acceptor) that **cannot absorb** that radiation directly, thereby initiating photochemical change in the acceptor.

**Steps:**
```
Sensitizer (S) + hν → S* (excited sensitizer)
S* + Acceptor (A) → S + A* (energy transfer)
A* → products (photochemical reaction)
```

**Examples:**
1. **Hg photosensitization:** Hg absorbs 253.7 nm UV → Hg* → transfers energy to H₂ → H₂ dissociates to 2H• → then reacts with other molecules.
2. **Chlorophyll** photosensitizes water oxidation in photosynthesis.
3. **Rose Bengal** photosensitizes singlet oxygen production.

---

### Photosensitized Reactions

**1. Decomposition of HI:**
```
HI + hν (photosensitized by Hg) → H• + I•
2H• → H₂
2I• → I₂
```

**2. Cis-trans isomerization:** Sensitized by dyes like rose bengal, erythrosine.

**3. Photosynthesis:** Chlorophyll absorbs light → excites electrons → used to reduce CO₂.

---

## Q4. Mechanisms of Photochemical Reactions

### Mechanism of Methane (CH₄) Photochemical Reactions

Methane reacts with Cl₂ under UV light (free radical chain mechanism):

**Initiation:**
```
Cl₂ + hν → 2Cl•
```

**Propagation:**
```
Cl• + CH₄ → HCl + CH₃•
CH₃• + Cl₂ → CH₃Cl + Cl•
(cycle continues)
```

**Termination:**
```
Cl• + Cl• → Cl₂
CH₃• + Cl• → CH₃Cl
CH₃• + CH₃• → C₂H₆
```

---

### Mechanism of Ammonia (NH₃) Photochemical Decomposition

Under UV light (~200 nm):

**Primary step:**
```
NH₃ + hν → NH₂• + H•
```

**Secondary reactions:**
```
H• + NH₃ → NH₂• + H₂
2NH₂• → N₂H₄ (hydrazine)
N₂H₄ → N₂ + 2H₂
```

**Overall:**
```
2NH₃ → N₂ + 3H₂    (photochemical decomposition)
```

---

### Mechanism of Ozone (O₃) Formation and Decomposition

**Formation (Chapman mechanism in stratosphere):**
```
Step 1:  O₂ + hν (λ < 242 nm) → 2O•      (photodissociation)
Step 2:  O• + O₂ + M → O₃ + M           (M = third body, N₂ or O₂)
```

**Decomposition:**
```
Step 3:  O₃ + hν (λ = 200–300 nm) → O₂ + O•
Step 4:  O• + O₃ → 2O₂
```

**Net:**
```
2O₃ → 3O₂    (ozone destruction cycle)
```

**Catalytic destruction by NOₓ (from pollutants):**
```
NO + O₃ → NO₂ + O₂
NO₂ + O• → NO + O₂
Net: O₃ + O• → 2O₂
```
This is why NOₓ emissions from aircraft damage the ozone layer.
