## 🔑 Key Formulas at a Glance

### Derivative Definition
$$f'(x) = \lim_{h \to 0}\frac{f(x+h)-f(x)}{h}$$

### Chain Rule
$$\frac{dy}{dx} = \frac{dy}{du}\cdot\frac{du}{dx}$$

### Taylor's Series (about $x = a$)
$$f(x) = \sum_{n=0}^{\infty}\frac{(x-a)^n}{n!}f^{(n)}(a)$$

### Maclaurin's Series (about $x = 0$)
$$f(x) = \sum_{n=0}^{\infty}\frac{x^n}{n!}f^{(n)}(0)$$

### L'Hôpital's Rule
$$\lim_{x\to a}\frac{f(x)}{g(x)} = \lim_{x\to a}\frac{f'(x)}{g'(x)} \quad \left[\frac{0}{0} \text{ or } \frac{\infty}{\infty}\right]$$

### Mean Value Theorem
$$f'(c) = \frac{f(b)-f(a)}{b-a}, \quad c\in(a,b)$$

### Euler's Theorem
$$x\frac{\partial f}{\partial x} + y\frac{\partial f}{\partial y} = nf \quad (f \text{ homogeneous of degree } n)$$
