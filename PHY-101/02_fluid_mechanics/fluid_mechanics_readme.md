## 🔑 Key Equations

### Fundamental Relations

| Equation | Formula | Description |
|----------|---------|-------------|
| **Density** | $\rho = \frac{m}{V}$ | Mass per unit volume |
| **Pressure** | $P = \frac{F}{A}$ | Force per unit area |
| **Hydrostatic Pressure** | $P = P_0 + \rho gh$ | Pressure at depth h |

### Flow Relations

| Equation | Formula | Description |
|----------|---------|-------------|
| **Volume Flow Rate** | $Q = Av$ | Volume per time |
| **Mass Flow Rate** | $\dot{m} = \rho Q = \rho Av$ | Mass per time |
| **Continuity** | $A_1v_1 = A_2v_2$ | Conservation of mass |

### Energy Relations

| Equation | Formula | Description |
|----------|---------|-------------|
| **Bernoulli's Equation** | $P + \frac{1}{2}\rho v^2 + \rho gh = \text{const}$ | Energy conservation |
| **Head Form** | $\frac{P}{\rho g} + \frac{v^2}{2g} + z = \text{const}$ | Energy per weight |
| **Torricelli's Theorem** | $v = \sqrt{2gh}$ | Speed of efflux |

### Flow Measurement

| Equation | Formula | Description |
|----------|---------|-------------|
| **Venturimeter** | $Q = C_d \frac{A_1A_2}{\sqrt{A_1^2-A_2^2}}\sqrt{\frac{2\Delta P}{\rho}}$ | Flow measurement |
| **Discharge Coefficient** | $C_d = \frac{Q_{actual}}{Q_{theoretical}}$ | Real vs ideal flow |

## 💡 Study Tips

### For Understanding Concepts

1. **Start with basics**: Understand fluid properties before moving to dynamics
2. **Visualize flow**: Draw streamlines and control volumes
3. **Use conservation laws**: Always identify what is conserved
4. **Check units**: Dimensional analysis catches many errors
5. **Physical intuition**: Think about what makes physical sense

### For Problem Solving

1. **Define system clearly**: Draw diagram, label knowns/unknowns
2. **State assumptions**: Incompressible? Steady? Inviscid?
3. **Apply equations systematically**: Continuity first, then Bernoulli
4. **Check reasonableness**: Does the answer make sense?
5. **Compare methods**: Try different approaches to verify

### For Exams

1. **Practice problems**: Work through all examples in notes
2. **Derive equations**: Don't just memorize, understand derivations
3. **Know limitations**: When does Bernoulli's equation apply?
4. **Master units**: Convert between different unit systems
5. **Review applications**: Real-world problems test understanding

## 🛠️ Applications in Textile Engineering

Fluid mechanics is fundamental to many textile processes:

1. **Dyeing Processes**
   - Flow through fabric pores
   - Dye bath circulation (continuity)
   - Jet dyeing machines (Bernoulli)
   - Drainage systems (Torricelli)

2. **Washing and Rinsing**
   - Water distribution systems
   - Flow through textiles
   - Drainage design
   - Flow rate optimization

3. **Coating Applications**
   - Coating fluid flow
   - Film thickness control (continuity)
   - Nozzle design (Bernoulli)
   - Distribution uniformity

4. **Air Flow Systems**
   - Ventilation design
   - Dust collection (pressure differences)
   - Pneumatic transport
   - Drying processes

5. **Hydraulic Systems**
   - Pressure-based machinery
   - Pumping systems
   - Flow measurement (Venturimeter)
   - Process control
