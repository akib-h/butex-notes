## 🔗 Quick Reference: Key Formulas

| Concept | Formula |
|---------|---------|
| Path difference (Young's) | `Δ = xd/D` |
| Fringe width | `β = λD/d` |
| Bright fringe position | `x_n = nλD/d` |
| Dark fringe position | `x_n = (2n+1)λD/2d` |
| Intensity (interference) | `I = 4a²cos²(δ/2)` |
| Phase–path relation | `δ = (2π/λ) × path diff` |
| Newton's ring radius (dark) | `r_m = √(mλR)` |
| Thin film (reflected, bright) | `2μt·cos r = (2n−1)λ/2` |
| Thin film (reflected, dark) | `2μt·cos r = nλ` |
