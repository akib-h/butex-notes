## Key Formulas at a Glance

$$T = \frac{F}{l} = \frac{W}{\Delta A} \qquad \text{(Surface Tension)}$$

$$P_{\text{liquid drop}} = \frac{2T}{r} \qquad P_{\text{soap bubble}} = \frac{4T}{r}$$

$$P = T\left(\frac{1}{r_1} + \frac{1}{r_2}\right) \qquad \text{(Young-Laplace)}$$

$$h = \frac{2T\cos\theta}{\rho g r} \qquad \text{(Capillary rise)}$$

$$T = \frac{r\rho g}{2}\left(h + \frac{r}{3}\right) \qquad \text{(Surface tension of water, } \theta=0°\text{)}$$
