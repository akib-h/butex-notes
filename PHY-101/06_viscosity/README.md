## Key Formulas Quick Reference

| Formula | Name |
|---------|------|
| $\tau = \eta \frac{du}{dy}$ | Newton's Law of Viscosity |
| $Re = \frac{\rho v D}{\eta}$ | Reynolds Number |
| $v_c = \frac{Re_c \cdot \eta}{\rho D}$ | Critical Velocity |
| $Q = \frac{\pi R^4 \Delta P}{8\eta L}$ | Hagen-Poiseuille Equation |
| $u(r) = \frac{\Delta P}{4\eta L}(R^2 - r^2)$ | Parabolic Velocity Profile |
| $\eta = \frac{\pi R^4 \rho g h t}{8VL}$ | Viscosity by Capillary Method |
| $\frac{\eta_1}{\eta_2} = \frac{\rho_1 t_1}{\rho_2 t_2}$ | Ostwald Viscometer Formula |
