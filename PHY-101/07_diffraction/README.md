## 🔑 Key Formulas at a Glance

| Formula | Meaning |
|---|---|
| $I(\theta) = I_0\left(\frac{\sin\beta}{\beta}\right)^2$ | Single-slit diffraction intensity |
| $N_F = a^2/(L\lambda)$ | Fresnel number (near vs. far field) |
| $d\sin\theta = m\lambda$ | Grating equation (normal incidence) |
| $R = mN$ | Resolving power of a grating |
| $D = m/(d\cos\theta)$ | Dispersive power (angular dispersion) |
| $\rho_m = \sqrt{m\lambda ab/(a+b)}$ | Radius of $m$-th Fresnel zone |
| $f = r_1^2/\lambda$ | Focal length of a zone plate |
| $2d\sin\theta = n\lambda$ | Bragg's Law (X-ray diffraction) |
| $D = K\lambda/(\beta\cos\theta)$ | Scherrer equation (crystallite size) |
