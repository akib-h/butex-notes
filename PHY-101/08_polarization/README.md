## Key Formulae Quick Reference

| Formula | Description |
|---------|-------------|
| $I = I_0\cos^2\theta$ | Malus's Law |
| $\tan\theta_B = n_2/n_1$ | Brewster's Law |
| $n_e(\theta) = \frac{n_o n_e}{\sqrt{n_e^2\cos^2\theta + n_o^2\sin^2\theta}}$ | E-ray refractive index |
| $\delta = \frac{2\pi t}{\lambda}\|n_e - n_o\|$ | Phase difference in birefringent plate |
| $t_{QWP} = \frac{\lambda}{4\|n_e - n_o\|}$ | Quarter-wave plate thickness |
| $[\alpha]_\lambda^T = \frac{\theta}{l \cdot C}$ | Specific rotation |
| $\theta = \frac{\pi l}{\lambda}(n_R - n_L)$ | Optical rotation (Fresnel theory) |
| $P = \frac{I_{max} - I_{min}}{I_{max} + I_{min}}$ | Degree of polarization |
| $R = \left(\frac{n_1 - n_2}{n_1 + n_2}\right)^2$ | Reflectance at normal incidence |
| $\sin\theta_c = n_2/n_1$ | Critical angle (TIR) |
