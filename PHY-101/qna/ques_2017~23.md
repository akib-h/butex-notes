# 1. Fluid Mechanics

## 1.1 Types of Flow

### Definition
**Fluid flow** is the motion of a fluid (liquid or gas) from one point to another. Based on the nature of motion, fluid flow is classified as:

---

### A) Streamline (Laminar) Flow

> **Definition:** Flow in which every fluid particle passing a given point follows the same path and has the same velocity as its predecessor. The flow pattern does not change with time.

**Key characteristics:**
- Fluid layers slide smoothly past one another
- No mixing between adjacent layers
- Velocity at every point is constant with time
- Occurs at **low velocities**
- Represented by smooth, parallel **streamlines**

**Streamlines** are imaginary curves drawn in a fluid such that the tangent at any point gives the direction of the fluid velocity at that point. In streamline flow, streamlines never cross each other.

```
Streamline Flow:
→ → → → → → →
→ → → → → → →   (parallel, smooth)
→ → → → → → →
```

---

### B) Turbulent Flow

> **Definition:** Flow in which the velocity at any point changes randomly and irregularly with time. Fluid particles move in zigzag, chaotic paths.

**Key characteristics:**
- Irregular, erratic motion
- Mixing between fluid layers
- High energy dissipation
- Occurs at **high velocities**
- Formation of eddies and vortices

```
Turbulent Flow:
→ ↗ ↙ → ↘ ↗ →
↘ → ↗ ↙ → ↗ ↘   (irregular, chaotic)
→ ↘ → ↗ ↙ → →
```

---

### C) Uniform vs Non-Uniform Flow

| Property | Uniform Flow | Non-Uniform Flow |
|----------|-------------|-----------------|
| Velocity | Same at all cross-sections | Varies along length |
| Example | Flow through constant-diameter pipe | Flow through tapering pipe |

---

### D) Steady vs Unsteady Flow

| Property | Steady Flow | Unsteady Flow |
|----------|-------------|---------------|
| Velocity at a point | Does not change with time | Changes with time |
| Streamlines | Fixed in space | Change with time |

## 1.2 Equation of Continuity

### Statement
> For an incompressible fluid in steady flow, the product of the cross-sectional area and the fluid velocity remains constant at every point along a streamline.

### Derivation

Consider a fluid flowing through a pipe of varying cross-section:

```
       A₁              A₂
  _____|_____      _____|_____
 |           \  /             |
 |    v₁      \/      v₂      |
 |           /  \             |
 |_________/    \_____________|
      P₁                P₂
```

Let:
- `A₁`, `A₂` = cross-sectional areas at points 1 and 2
- `v₁`, `v₂` = fluid velocities at points 1 and 2
- `ρ` = density of fluid (constant for incompressible fluid)

**Mass of fluid entering at P₁ per second:**
```
m₁ = ρ × A₁ × v₁
```

**Mass of fluid leaving at P₂ per second:**
```
m₂ = ρ × A₂ × v₂
```

By conservation of mass (no fluid created or destroyed):
```
m₁ = m₂
ρ A₁ v₁ = ρ A₂ v₂
```

**∴ A₁v₁ = A₂v₂ = constant**

> **Physical meaning:** Where the pipe is **narrower (smaller A)**, the fluid flows **faster (larger v)**, and vice versa. This explains why water flows faster through a narrow nozzle.

## 1.3 Bernoulli's Theorem

### Statement
> For a steady, irrotational, incompressible, non-viscous fluid in streamline flow, the total mechanical energy per unit volume (sum of pressure energy, kinetic energy, and potential energy) remains constant throughout the flow.

**Mathematical Form:**

```
P + ½ρv² + ρgh = constant
```

Or between two points 1 and 2:

```
P₁ + ½ρv₁² + ρgh₁ = P₂ + ½ρv₂² + ρgh₂
```

Where:
- `P` = pressure at the point (Pa)
- `ρ` = density of fluid (kg/m³)
- `v` = velocity of fluid (m/s)
- `g` = acceleration due to gravity (9.8 m/s²)
- `h` = height of the point above reference level (m)

---

### Derivation (Work-Energy Theorem Approach)

Consider a fluid flowing through a pipe from section AB to section CD:

```
        A                    C
   _____|____           _____|____
  |          \         /          |
  |  P₁,v₁   \       /  P₂,v₂   |
  |    h₁      \_____/    h₂      |
  |____________|     |____________|
        B                    D
```

**Assumptions:**
1. Fluid is ideal (non-viscous, incompressible)
2. Flow is steady and irrotational
3. No energy losses due to friction

**Work done on fluid at section AB** (pushing fluid in):
```
W₁ = P₁ × A₁ × dx₁ = P₁ × dV
```

**Work done by fluid at section CD** (fluid pushes outward):
```
W₂ = P₂ × A₂ × dx₂ = P₂ × dV
```

**Net work done by pressure forces:**
```
W_net = W₁ - W₂ = (P₁ - P₂) dV
```

**Change in kinetic energy (for mass dm = ρ dV):**
```
ΔKE = ½(ρ dV)v₂² - ½(ρ dV)v₁²
```

**Change in potential energy:**
```
ΔPE = (ρ dV)g h₂ - (ρ dV)g h₁
```

By **work-energy theorem**:
```
W_net = ΔKE + ΔPE

(P₁ - P₂)dV = ½ρ dV(v₂² - v₁²) + ρ dV g(h₂ - h₁)
```

Dividing throughout by `dV`:
```
P₁ - P₂ = ½ρ(v₂² - v₁²) + ρg(h₂ - h₁)
```

Rearranging:
```
P₁ + ½ρv₁² + ρgh₁ = P₂ + ½ρv₂² + ρgh₂
```

**∴ P + ½ρv² + ρgh = constant** ✓

---

### Applications of Bernoulli's Theorem

| Application | Principle Used |
|-------------|----------------|
| Venturi meter | Pressure drops where velocity increases |
| Airplane lift (Aerofoil) | Higher speed above wing → lower pressure |
| Spray gun / Atomizer | Fast air stream → low pressure → liquid rises |
| Pitot tube | Measures fluid velocity from pressure difference |
| Magnus effect (spinning ball) | Asymmetric velocity → asymmetric pressure |

## 1.4 Poiseuille's Equation

### Derivation: Rate of Flow through a Capillary Tube

**Poiseuille's equation** gives the volume of viscous liquid flowing per second through a narrow cylindrical tube (capillary).

**Setup:**

```
          r
    ←————→
    ___________________________________________
   |  · · · · · · · · · · · · · · · · · · ·  |  ← cylindrical shell
   |  · · · · · · · · · · · · · · · · · · ·  |     at radius x
   |←——————————— L ————————————————————————→|
   |                                           |
   P₁ (high pressure)               P₂ (low pressure)
```

**Assumptions:**
1. Liquid is incompressible and Newtonian
2. Flow is steady and laminar (streamline)
3. No slip at the tube walls (velocity = 0 at r = R)
4. Tube is horizontal and rigid

**Step 1: Velocity Profile**

Consider a cylindrical shell of radius `x` and thickness `dx`. The driving pressure force on this shell:
```
F_pressure = (P₁ - P₂) × π x²
```

Viscous (retarding) force on the shell surface area `2πxL`:
```
F_viscous = -η × (2πxL) × (dv/dx)
```

At equilibrium (steady flow):
```
(P₁ - P₂)πx² = -η(2πxL)(dv/dx)

dv/dx = -(P₁ - P₂)x / (2ηL)
```

Integrating:
```
v = -(P₁ - P₂)x² / (4ηL) + C
```

Applying boundary condition: at `x = R` (tube wall), `v = 0`:
```
0 = -(P₁ - P₂)R² / (4ηL) + C
C = (P₁ - P₂)R² / (4ηL)
```

**Velocity profile (parabolic):**
```
v(x) = (P₁ - P₂)(R² - x²) / (4ηL)
```

This is a **parabolic velocity profile** — maximum at center, zero at walls.

**Step 2: Volume Flow Rate**

Volume flowing per second through the shell of radius `x` and thickness `dx`:
```
dQ = v(x) × 2πx dx
dQ = [(P₁ - P₂)(R² - x²) / (4ηL)] × 2πx dx
```

Integrating from 0 to R:
```
Q = π(P₁ - P₂) / (2ηL) × ∫₀ᴿ (R² - x²)x dx

= π(P₁ - P₂) / (2ηL) × [R²x²/2 - x⁴/4]₀ᴿ

= π(P₁ - P₂) / (2ηL) × [R⁴/2 - R⁴/4]

= π(P₁ - P₂) / (2ηL) × R⁴/4
```

**∴ Poiseuille's Equation:**

```
        π(P₁ - P₂)R⁴
Q = ———————————————————
           8ηL
```

Where:
- `Q` = volume flow rate (m³/s)
- `P₁ - P₂` = pressure difference (Pa)
- `R` = radius of tube (m)
- `η` = coefficient of viscosity (N·s/m²)
- `L` = length of tube (m)

> **Key insight:** Flow rate is proportional to **R⁴** — doubling the radius increases flow by **16 times**!

---

### Necessary Corrections to Poiseuille's Equation

The ideal Poiseuille equation needs corrections for real-world conditions:

**1. End Correction (Entrance Length Correction)**
- Fluid entering the tube needs a certain length to develop the parabolic velocity profile
- **Correction:** Replace `L` with `(L + 1.64R)` or `(L + nR)` where n ≈ 0.82 to 1.64
```
Q = πPR⁴ / [8η(L + 1.64R)]
```

**2. Kinetic Energy Correction**
- Fluid at the outlet has kinetic energy that must be subtracted from driving pressure
- **Correction:** Effective pressure = `P - ρv̄²m/8`  where `v̄` is mean velocity
```
P_eff = P - ρQ² / (8π²R⁴)
```

**3. Correction for Turbulence**
- If Reynolds number exceeds ~2000, flow becomes turbulent and Poiseuille's law fails
- Must ensure `Re < 2000` for the equation to be valid

**4. Temperature Correction**
- Viscosity changes with temperature; must measure `η` at the exact temperature of the experiment

**5. Compressibility Correction**
- For gases, density varies with pressure; a correction factor is needed

**6. Wall Slip Correction**
- At very small radii (nanoscale), no-slip condition may fail

## 1.5 Venturi Meter

### Theory and Derivation

A **Venturi meter** is a device used to measure the rate of flow of a fluid through a pipe. It works on Bernoulli's principle.

**Construction:**

```
    Wide section              Throat              Wide section
        A₁                    A₂                     A₁
  ________|_____         _____|_____         _________|______
 |              \       /           \       /                |
 |   v₁, P₁     \_____/   v₂, P₂   \_____/    v₁, P₁       |
 |_______________|     |_____________|     |__________________|
        |                   |
        |___________________|
              Manometer (h)
```

**Derivation:**

For a horizontal Venturi meter (h₁ = h₂), Bernoulli's equation gives:
```
P₁ + ½ρv₁² = P₂ + ½ρv₂²

(P₁ - P₂) = ½ρ(v₂² - v₁²)   ... (1)
```

From continuity equation:
```
A₁v₁ = A₂v₂
v₁ = (A₂/A₁)v₂   ... (2)
```

Substituting (2) into (1):
```
P₁ - P₂ = ½ρv₂²[1 - (A₂/A₁)²]

v₂² = 2(P₁ - P₂) / [ρ(1 - A₂²/A₁²)]

v₂ = √[2(P₁-P₂)A₁² / ρ(A₁²-A₂²)]
```

**Volume flow rate:**
```
Q = A₂v₂ = A₁A₂ × √[2(P₁-P₂) / ρ(A₁²-A₂²)]
```

If manometer shows height difference `h`:
```
P₁ - P₂ = ρgh  (for same liquid)
```

**∴ Rate of flow:**
```
         A₁A₂
Q = ————————————— × √(2gh)
     √(A₁² - A₂²)
```

With discharge coefficient `Cd` (accounts for real fluid losses, typically 0.95–0.99):
```
           Cd × A₁A₂
Q = ———————————————————— × √(2gh)
         √(A₁² - A₂²)
```

## 1.6 Torricelli's Theorem

### Statement
> The velocity of efflux of a liquid through a small orifice at depth `h` below the free surface is equal to the velocity that a body would acquire falling freely from height `h`.

```
v = √(2gh)
```

### Proof

```
        Free surface
   ___________________
  |                   |
  |       Liquid      | h
  |                   |
  |___________________⊙—→ v (efflux velocity)
          orifice
```

Apply Bernoulli's theorem between the free surface (point 1) and the orifice (point 2):
- Point 1: `P₁ = P_atm`, `v₁ ≈ 0` (large tank), height = `h`
- Point 2: `P₂ = P_atm`, `v₂ = v`, height = `0`

```
P_atm + 0 + ρgh = P_atm + ½ρv² + 0

ρgh = ½ρv²

v² = 2gh

∴ v = √(2gh)
```

This is identical to the velocity of free fall from height `h` (from `v² = 2gh`), hence the theorem.

**Range of the jet** (horizontal distance):
```
x = v × t = √(2gh) × √(2(H-h)/g) = 2√[h(H-h)]
```
where `H` = total height of liquid.

## 1.7 Critical Velocity & Reynolds Number

### Critical Velocity

> **Definition:** The velocity of fluid flow below which flow is streamline (laminar) and above which it becomes turbulent.

**Formula (from Reynolds):**
```
v_c = Rη / (ρD)   or   v_c = Kη / (ρr)
```
Where:
- `η` = coefficient of viscosity
- `ρ` = density
- `D` = diameter of tube
- `K` = Reynolds number (dimensionless constant)

### Reynolds Number

The **Reynolds number (Re)** is a dimensionless quantity that predicts the nature of flow:

```
Re = ρvD / η = ρvL / η
```

| Reynolds Number | Type of Flow |
|-----------------|-------------|
| Re < 2000 | Laminar (streamline) |
| 2000 < Re < 4000 | Transitional |
| Re > 4000 | Turbulent |

**Physical meaning:** Re = (inertial forces) / (viscous forces)

## 1.8 Solved Numerical Problems - Fluid Mechanics

---

### Problem F1 *(2022, 2020, 2017)*
**Water flows through a horizontal pipeline of varying cross-section at the rate of 0.2 m³/s. Calculate the velocity of water at a point where the area of cross-section is 0.02 m².**

**Given:**
- Volume flow rate: Q = 0.2 m³/s
- Cross-sectional area: A = 0.02 m²

**Solution:**

From the definition of flow rate:
```
Q = A × v

v = Q/A = 0.2 / 0.02

∴ v = 10 m/s
```

**Answer: The velocity of water = 10 m/s**

---

### Problem F2 *(2021)*
**Water flows through a horizontal pipeline of varying cross-section. At a point where the pressure of water is 0.05 m of Hg, the velocity of flow is 0.25 m/s. Calculate the pressure at another point where the velocity of flow is 0.4 m/s. (Density of water = 1000 kg/m³)**

**Given:**
- P₁ = 0.05 m of Hg = 0.05 × 13600 × 9.8 = 6664 Pa
- v₁ = 0.25 m/s
- v₂ = 0.40 m/s
- ρ_water = 1000 kg/m³
- Horizontal pipe: h₁ = h₂

**Solution:**

Applying Bernoulli's equation (horizontal pipe):
```
P₁ + ½ρv₁² = P₂ + ½ρv₂²

P₂ = P₁ + ½ρ(v₁² - v₂²)

P₂ = 6664 + ½ × 1000 × (0.25² - 0.40²)

P₂ = 6664 + 500 × (0.0625 - 0.16)

P₂ = 6664 + 500 × (-0.0975)

P₂ = 6664 - 48.75

P₂ = 6615.25 Pa
```

Converting to m of Hg:
```
P₂ = 6615.25 / (13600 × 9.8) = 0.04964 m of Hg ≈ 0.0496 m of Hg
```

**Answer: P₂ ≈ 6615 Pa ≈ 0.0496 m of Hg**

---

### Problem F3 *(2023)*
**Under pressure of a column of water 30 cm in height, find the rate of flow of water through the capillary tube of radius 0.5 mm and length 20 cm. (Viscosity of water = 10⁻³ N·s/m²)**

**Given:**
- Pressure head: h = 30 cm = 0.30 m
- Radius: R = 0.5 mm = 0.5 × 10⁻³ m
- Length: L = 20 cm = 0.20 m
- η = 10⁻³ N·s/m²
- ρ_water = 1000 kg/m³

**Solution:**

Pressure difference:
```
P = ρgh = 1000 × 9.8 × 0.30 = 2940 Pa
```

Using Poiseuille's equation:
```
Q = πPR⁴ / (8ηL)

Q = π × 2940 × (0.5×10⁻³)⁴ / (8 × 10⁻³ × 0.20)

Q = π × 2940 × 6.25×10⁻¹⁴ / (1.6×10⁻³)

Q = π × 1.8375×10⁻¹⁰ / 1.6×10⁻³

Q = 3.14159 × 1.148×10⁻⁷

Q ≈ 3.61 × 10⁻⁷ m³/s
```

**Answer: Q ≈ 3.61 × 10⁻⁷ m³/s**

---

### Problem F4 *(2019)*
**Calculate the speed at which the velocity head of a stream of water is equal to 0.50 m of Hg.**

**Given:**
- Velocity head = h = 0.50 m of Hg
- Density of Hg = 13600 kg/m³
- Density of water = 1000 kg/m³

**Solution:**

Velocity head in terms of water:
```
h_water = (ρ_Hg / ρ_water) × h_Hg = (13600/1000) × 0.50 = 6.8 m of water
```

Using velocity head formula:
```
h = v²/(2g)

v² = 2gh = 2 × 9.8 × 6.8 = 133.28

v = √133.28 ≈ 11.54 m/s
```

**Answer: v ≈ 11.54 m/s**

---

### Problem F5 *(2018)*
**Water flows through a horizontal capillary tube of 1 mm internal diameter and length 70 cm under a pressure head of 20 cm of water. Find the rate of flow. (η_water = 10⁻³ Pa·s)**

**Given:**
- Diameter = 1 mm → R = 0.5 mm = 5 × 10⁻⁴ m
- Length L = 70 cm = 0.70 m
- Pressure head = 20 cm of water = 0.20 m
- η = 10⁻³ Pa·s

**Solution:**
```
P = ρgh = 1000 × 9.8 × 0.20 = 1960 Pa

Q = πPR⁴ / (8ηL)
  = π × 1960 × (5×10⁻⁴)⁴ / (8 × 10⁻³ × 0.70)
  = π × 1960 × 6.25×10⁻¹⁴ / 5.6×10⁻³
  = π × 1.225×10⁻¹⁰ / 5.6×10⁻³
  = π × 2.1875×10⁻⁸
  ≈ 6.87 × 10⁻⁸ m³/s
```

**Answer: Q ≈ 6.87 × 10⁻⁸ m³/s**

# 2. Viscosity

## 2.1 Newton's Law of Viscous Flow

### Definition of Viscosity
> **Viscosity** is the property of a fluid by virtue of which it resists relative motion between adjacent layers. It is the internal friction of fluids.

### Newton's Law of Viscous Flow

Consider two parallel layers of fluid at heights `y` and `y + dy`, with velocities `v` and `v + dv`:

```
  ————————————————————————  v + dv  (faster layer)
          ↑ dy
  ————————————————————————  v       (slower layer)
```

**Newton's law states:** The tangential (shearing) stress `F/A` between adjacent fluid layers is directly proportional to the velocity gradient `dv/dy` perpendicular to the layers:

```
F/A ∝ dv/dy

F/A = η × (dv/dy)
```

Or:
```
τ = η (dv/dy)
```

Where:
- `τ = F/A` = shear stress (Pa)
- `η` = coefficient of viscosity (dynamic viscosity)
- `dv/dy` = velocity gradient (s⁻¹)

The negative sign indicates that the viscous force opposes relative motion.

---

## 2.2 Coefficient of Viscosity

### Definition
> The **coefficient of viscosity** (η) is defined as the tangential force per unit area required to maintain a unit velocity gradient between adjacent layers of the fluid.

**From Newton's law:**
```
η = (F/A) / (dv/dy) = τ / (dv/dy)
```

**SI Unit:** Pascal-second (Pa·s) = N·s/m² = kg/(m·s)

**CGS Unit:** Poise (P) = g/(cm·s) = dyn·s/cm²

**Conversion:** 1 Pa·s = 10 Poise

### Velocity Gradient

> **Velocity gradient** = rate of change of velocity with distance perpendicular to the direction of flow.

```
Velocity gradient = dv/dy    [units: s⁻¹ or m/s per m]
```

### Temperature Dependence of Viscosity

| Fluid Type | Effect of Temperature on η |
|-----------|--------------------------|
| Liquids | η **decreases** with increasing T (molecules move apart) |
| Gases | η **increases** with increasing T (more inter-molecular collisions) |

---

## 2.3 Poiseuille's Method for Viscosity

### Determination of Viscosity by Capillary Flow Method

**Principle:** Poiseuille's formula is rearranged to find η:

```
η = πPR⁴ / (8QL)
```

**Experimental Setup:**

```
    Constant head
    tank (pressure P)
         |
         | (tube of length L, radius R)
         ↓
    Measuring
    cylinder (measure Q)
```

**Procedure:**
1. Set up a capillary tube of known radius `R` and length `L`
2. Maintain a constant pressure head `P` (or `ρgh`)
3. Collect the liquid that flows out in a measuring cylinder over time `t`
4. Calculate flow rate: `Q = Volume / time`
5. Compute: `η = πPR⁴ / (8QL)`

**Precautions:**
- Ensure laminar flow (Re < 2000)
- Apply end corrections
- Maintain constant temperature
- Clean tube to avoid contamination

## 2.4 Solved Numerical Problems - Viscosity

---

### Problem V1 *(2022, 2020, 2017)*
**A plate of metal (area = 10⁻² m²) rests on a layer of castor oil of thickness 2×10⁻³ m, coefficient of viscosity = 1.55 N·s/m². Calculate the horizontal force required to move the plate with uniform speed of 3×10⁻² m/s.**

**Given:**
- Area: A = 10⁻² m²
- Oil thickness (separation): dy = 2×10⁻³ m
- Coefficient of viscosity: η = 1.55 N·s/m²
- Velocity: v = 3×10⁻² m/s (uniform, so lower surface is stationary)

**Solution:**

Velocity gradient:
```
dv/dy = v/dy = (3×10⁻²) / (2×10⁻³) = 15 s⁻¹
```

Using Newton's viscosity law:
```
F = η × A × (dv/dy)

F = 1.55 × 10⁻² × 15

F = 1.55 × 0.15

F = 0.2325 N
```

**Answer: F = 0.2325 N ≈ 0.23 N**

---

### Problem V2 (Stokes' Law Application)
**A steel ball of radius 2 mm falls through glycerine. If terminal velocity = 5 cm/s and density of steel = 7800 kg/m³, density of glycerine = 1260 kg/m³, find η of glycerine.**

**Solution:**

At terminal velocity, using Stokes' law:
```
mg = F_buoyancy + F_viscous

(4/3)πr³ρ_s × g = (4/3)πr³ρ_l × g + 6πηrv_t

6πηrv_t = (4/3)πr³g(ρ_s - ρ_l)

η = 2r²g(ρ_s - ρ_l) / (9v_t)

η = 2 × (2×10⁻³)² × 9.8 × (7800 - 1260) / (9 × 0.05)

η = 2 × 4×10⁻⁶ × 9.8 × 6540 / 0.45

η = 2 × 4×10⁻⁶ × 64092 / 0.45

η ≈ 1.139 Pa·s
```

**Answer: η ≈ 1.14 Pa·s**

# 3. Surface Tension

## 3.1 Surface Energy

### Definition
> **Surface energy** is the potential energy stored per unit area of the liquid surface. It arises because molecules at the surface have fewer neighbors than those in the bulk, resulting in higher potential energy.

**Units:** J/m² (joules per square meter)

**Why does surface energy exist?**

```
Bulk molecule:        Surface molecule:
    ○ ○ ○               ○ ○ ○
    ○ ● ○   ← full      ○ ● ○   ← fewer
    ○ ○ ○    attraction  · · ·    neighbors
             all sides            (air above)
```

Molecules at the surface are pulled inward — to expand the surface, work must be done against this inward pull. This work is stored as **surface energy**.

---

## 3.2 Proof: Surface Tension = Surface Energy per Unit Area

### Definition of Surface Tension
> **Surface tension** (T or γ) is the force acting per unit length along the surface of a liquid in a direction perpendicular to the line and tangential to the surface.

```
T = F / l    [Unit: N/m]
```

### Proof (Work-Energy Method)

Consider a rectangular wire frame with a movable wire of length `l`. A soap film is stretched across it.

```
        Fixed wire
        ___________
       |           |
  F →  |  soap     | ← width l
       |  film     |
       |___________|
        Movable wire (length l)
           ↑ dx (displacement)
```

**Force on movable wire:** The soap film has **two surfaces**, each pulling with tension T:
```
F = 2Tl
```

**Work done to extend film by `dx`:**
```
dW = F × dx = 2Tl × dx
```

**Increase in surface area** (both surfaces):
```
dA = 2 × l × dx
```

**Work done per unit area of new surface created:**
```
dW/dA = (2Tl dx) / (2l dx) = T
```

But work done per unit area = surface energy per unit area (E):
```
E = T
```

**∴ Surface tension = Surface energy per unit area** ✓

```
T [N/m] = E [J/m²]
```

(Both have the same SI unit since 1 N/m = 1 J/m²)

## 3.3 Capillary Rise

### Derivation of Height of Capillary Rise

**Setup:**

```
        ┌───────────────────┐
        │                   │    ←  Wide container
        │      liquid       │
        │                   │
        │  ┌─────────┐      │
        │  │ ┌─────┐ │      │
        │  │ │  ↑h │ │      │
        │  │ │     │ │      │
        │  │ └─────┘ │      │
        │  │  tube   │      │
        │  │ radius r│      │
        └──┴─────────┴──────┘
```

Let:
- `r` = radius of capillary tube
- `h` = height of liquid rise
- `T` = surface tension of liquid
- `θ` = contact angle
- `ρ` = density of liquid
- `g` = acceleration due to gravity

**Upward force** (surface tension along contact line, resolved vertically):
```
F_up = T cos θ × 2πr = 2πrT cos θ
```

**Weight of liquid column** (downward):
```
W = mass × g = (Volume × ρ) × g = (πr²h × ρ) × g = πr²hρg
```

*Note: We neglect the small meniscus volume for derivation simplicity.*

At equilibrium:
```
F_up = W

2πrT cos θ = πr²hρg

h = 2T cos θ / (rρg)
```

**∴ Capillary Rise Formula:**

```
        2T cos θ
h = ————————————————
          rρg
```

### Key Observations

| Condition | Result |
|-----------|--------|
| θ < 90° (water in glass) | h > 0 → liquid **rises** |
| θ > 90° (mercury in glass) | h < 0 → liquid **depresses** |
| θ = 0° (complete wetting) | h = 2T/(rρg) — maximum rise |
| Smaller radius r | Greater height h |

**Jurin's Law:** `h × r = constant` (for given liquid and tube)

---

## 3.4 Angle of Contact

### Definition
> The **angle of contact** (θ) is the angle between the tangent to the liquid surface at the point of contact and the solid surface, measured through the liquid.

```
Glass-Water (θ ≈ 0°)      Glass-Mercury (θ ≈ 135°)

       |                        |
       | ╲ θ≈0°                 |╱ θ≈135°
    ~~~╯                    ~~~~╲
    concave meniscus         convex meniscus
    (liquid rises)           (liquid depresses)
```

| Liquid-Solid Pair | θ | Behavior |
|-------------------|---|----------|
| Water-Glass | ~0° | Strong adhesion, concave meniscus |
| Mercury-Glass | ~135° | Strong cohesion, convex meniscus |
| Water-Silver | ~90° | Flat meniscus |

### Problem S1 *(2023)*
**A capillary tube has radius 0.5×10⁻³ m. Density of water = 0.9×10³ kg/m³. Surface tension of water = 7.2×10⁻² N/m. Find the height of water column when dipped vertically. (Assume θ = 0°)**

**Given:**
- r = 0.5×10⁻³ m
- ρ = 0.9×10³ kg/m³ = 900 kg/m³
- T = 7.2×10⁻² N/m
- θ = 0° → cos 0° = 1

**Solution:**
```
h = 2T cos θ / (rρg)

h = 2 × 7.2×10⁻² × 1 / (0.5×10⁻³ × 900 × 9.8)

h = 0.144 / (0.5×10⁻³ × 8820)

h = 0.144 / 4.41

h ≈ 0.03265 m ≈ 3.27 cm
```

**Answer: h ≈ 3.27 cm**

---

### Problem S2 *(2021)*
**Calculate the height to which a liquid will rise in a capillary tube of radius 0.22 mm when surface tension = 26×10⁻³ N/m and density = 1000 kg/m³. Assume angle of contact = 0°.**

**Given:**
- r = 0.22 mm = 0.22×10⁻³ m
- T = 26×10⁻³ N/m
- ρ = 1000 kg/m³
- θ = 0°

**Solution:**
```
h = 2T cos θ / (rρg)

h = 2 × 26×10⁻³ × 1 / (0.22×10⁻³ × 1000 × 9.8)

h = 52×10⁻³ / (0.22×10⁻³ × 9800)

h = 52×10⁻³ / 2.156

h ≈ 0.02412 m ≈ 2.41 cm
```

**Answer: h ≈ 2.41 cm**

### Practice Problem 1 (Continuity Equation)
**A pipe has a circular cross-section of diameter 10 cm. It branches into two pipes of diameters 6 cm and 8 cm. If the velocity in the 10 cm pipe is 2 m/s, find the velocity in the 6 cm pipe given that the velocity in the 8 cm pipe is 1.5 m/s.**

<details>
<summary>📋 Click to see Solution</summary>

**Given:**
- D₁ = 10 cm → r₁ = 5 cm = 0.05 m → A₁ = π(0.05)² = 7.854×10⁻³ m²
- D₂ = 6 cm → r₂ = 3 cm → A₂ = π(0.03)² = 2.827×10⁻³ m²
- D₃ = 8 cm → r₃ = 4 cm → A₃ = π(0.04)² = 5.027×10⁻³ m²
- v₁ = 2 m/s, v₃ = 1.5 m/s

By continuity (flow in = sum of flows out):
```
A₁v₁ = A₂v₂ + A₃v₃
7.854×10⁻³ × 2 = 2.827×10⁻³ × v₂ + 5.027×10⁻³ × 1.5
15.708×10⁻³ = 2.827×10⁻³ × v₂ + 7.54×10⁻³
2.827×10⁻³ × v₂ = 8.168×10⁻³
v₂ = 2.89 m/s
```

**Answer: v₂ ≈ 2.89 m/s**

</details>

### Practice Problem 2 (Bernoulli's Theorem)
**In a horizontal pipe, water flows from a section of area 100 cm² to a narrower section of area 25 cm². If the gauge pressure at the wider section is 3×10⁴ Pa and the velocity there is 1 m/s, find the pressure at the narrow section.**

<details>
<summary>📋 Click to see Solution</summary>

**Given:**
- A₁ = 100 cm² = 10⁻² m², v₁ = 1 m/s, P₁ = 3×10⁴ Pa
- A₂ = 25 cm² = 25×10⁻⁴ m²

From continuity:
```
v₂ = A₁v₁/A₂ = (10⁻² × 1) / (25×10⁻⁴) = 4 m/s
```

Bernoulli's (horizontal pipe):
```
P₂ = P₁ + ½ρ(v₁² - v₂²)
P₂ = 3×10⁴ + ½ × 1000 × (1² - 4²)
P₂ = 30000 + 500 × (1 - 16)
P₂ = 30000 - 7500
P₂ = 22500 Pa = 2.25×10⁴ Pa
```

**Answer: P₂ = 2.25×10⁴ Pa**

</details>

### Practice Problem 3 (Poiseuille's Flow)
**Two capillary tubes A and B have lengths 20 cm and 40 cm and radii 0.5 mm and 1 mm respectively. They are connected in series. Find the ratio of pressure drops across A and B.**

<details>
<summary>📋 Click to see Solution</summary>

In series connection, same flow rate Q passes through both:

From Poiseuille's equation: `Q = πPR⁴/(8ηL)` → `P = 8ηLQ/(πR⁴)`

```
P_A/P_B = (L_A × R_B⁴) / (L_B × R_A⁴)
= (20 × (1×10⁻³)⁴) / (40 × (0.5×10⁻³)⁴)
= (20 × 1×10⁻¹²) / (40 × 0.0625×10⁻¹²)
= 20 / (40 × 0.0625)
= 20 / 2.5
= 8
```

**Answer: P_A : P_B = 8 : 1**

</details>

### Practice Problem 4 (Torricelli's Theorem)
**A large tank contains water to a height of 5 m. A small hole is made at the side of the tank at height 2 m from the bottom. Find: (a) the velocity of efflux, and (b) the horizontal range of the jet.**

<details>
<summary>📋 Click to see Solution</summary>

**Given:** H = 5 m (total height), h_hole = 2 m from bottom
Depth of hole below surface: h = 5 - 2 = 3 m

**(a) Velocity of efflux:**
```
v = √(2gh) = √(2 × 9.8 × 3) = √58.8 ≈ 7.67 m/s
```

**(b) Horizontal range:**
```
x = 2√[h_hole × (H - h_hole)]
= 2√[2 × 3] = 2√6 ≈ 4.90 m
```

**Answer: v ≈ 7.67 m/s, Range ≈ 4.90 m**

</details>

### Practice Problem 5 (Newton's Law)
**Two large horizontal plates are separated by 1 mm. The lower plate is fixed, and the upper plate moves with velocity 2 m/s. The fluid between them has η = 0.5 Pa·s. Find the shear stress and force required if the plate area is 0.5 m².**

<details>
<summary>📋 Click to see Solution</summary>

```
dv/dy = v/d = 2/1×10⁻³ = 2000 s⁻¹

Shear stress τ = η × (dv/dy) = 0.5 × 2000 = 1000 Pa

Force F = τ × A = 1000 × 0.5 = 500 N
```

**Answer: τ = 1000 Pa, F = 500 N**

</details>

### Practice Problem 6 (Coefficient of Viscosity)
**A viscometer consists of two concentric cylinders of radii 5 cm and 5.5 cm. The inner cylinder is fixed. The outer cylinder rotates at 2 rev/s. If the torque required is 0.9 N·m and the length of cylinders is 10 cm, find η.**

<details>
<summary>📋 Click to see Solution</summary>

```
v = 2π × n × r_outer = 2π × 2 × 0.055 = 0.6912 m/s
Gap d = 0.055 - 0.05 = 0.005 m
dv/dy = v/d = 0.6912/0.005 = 138.23 s⁻¹

Area of outer cylinder = 2πr_outer × L = 2π × 0.055 × 0.10 = 0.03456 m²

F = Torque / r_outer = 0.9/0.055 = 16.36 N

τ = F/A = 16.36/0.03456 = 473.4 Pa

η = τ/(dv/dy) = 473.4/138.23 ≈ 3.42 Pa·s
```

**Answer: η ≈ 3.42 Pa·s**

</details>

### Practice Problem 7 (Surface Energy)
**Calculate the work done in blowing a soap bubble of radius 5 cm. Surface tension of soap solution = 0.025 N/m.**

<details>
<summary>📋 Click to see Solution</summary>

```
Total surface area = 2 × 4πr²
= 2 × 4π × (0.05)²
= 2 × 4π × 0.0025
= 0.0628 m²

Work done = T × Total area = 0.025 × 0.0628 ≈ 1.57 × 10⁻³ J
```

**Answer: W ≈ 1.57 × 10⁻³ J = 1.57 mJ**

### Practice Problem 8 (Capillary Rise)
**A capillary tube of radius 0.3 mm is dipped in mercury (density = 13600 kg/m³, surface tension = 0.465 N/m, angle of contact = 135°). Find the depression of mercury in the tube.**

<details>
<summary>📋 Click to see Solution</summary>

**Given:**
- r = 0.3 mm = 3×10⁻⁴ m
- ρ = 13600 kg/m³
- T = 0.465 N/m
- θ = 135° → cos 135° = -0.707

```
h = 2T cosθ / (rρg)
= 2 × 0.465 × (-0.707) / (3×10⁻⁴ × 13600 × 9.8)
= -0.6580 / 39.97
= -0.01646 m ≈ -1.65 cm
```

The negative sign means mercury **depresses** by 1.65 cm.

**Answer: Mercury depresses by ≈ 1.65 cm**

### Practice Problem 9 (Surface Tension — Drops)
**Find the work done to split a water drop of radius 1 mm into 1000 equal droplets. Surface tension of water = 0.072 N/m.**

<details>
<summary>📋 Click to see Solution</summary>

Volume conservation:
```
(4/3)πR³ = 1000 × (4/3)πr³
R³ = 1000r³
r = R/10 = 1mm/10 = 0.1 mm = 10⁻⁴ m
R = 10⁻³ m
```

Increase in surface area:
```
ΔA = 1000 × 4πr² - 4πR²
= 4π[1000 × (10⁻⁴)² - (10⁻³)²]
= 4π[1000 × 10⁻⁸ - 10⁻⁶]
= 4π[10⁻⁵ - 10⁻⁶]
= 4π × 9×10⁻⁶
= 1.131×10⁻⁴ m²
```

Work done:
```
W = T × ΔA = 0.072 × 1.131×10⁻⁴
= 8.14×10⁻⁶ J ≈ 8.14 μJ
```

**Answer: W ≈ 8.14 × 10⁻⁶ J**

</details>

### Practice Problem 10 (Combined — Venturi Meter)
**A Venturi meter has pipe diameter 20 cm and throat diameter 10 cm. The pressure difference between inlet and throat is 4×10⁴ Pa. Find the rate of flow of water through the meter. (Cd = 0.98)**

<details>
<summary>📋 Click to see Solution</summary>

**Given:**
- D₁ = 20 cm → A₁ = π(0.1)² = 0.03142 m²
- D₂ = 10 cm → A₂ = π(0.05)² = 7.854×10⁻³ m²
- ΔP = 4×10⁴ Pa → h = ΔP/(ρg) = 40000/9800 ≈ 4.082 m
- Cd = 0.98

```
Q = Cd × A₁A₂/√(A₁²-A₂²) × √(2gh)

= 0.98 × (0.03142 × 7.854×10⁻³)/√(0.03142² - 7.854×10⁻³²) × √(2×9.8×4.082)

Numerator of area term: 0.03142 × 7.854×10⁻³ = 2.468×10⁻⁴

A₁² = 9.87×10⁻⁴, A₂² = 6.17×10⁻⁵
A₁²-A₂² = 9.25×10⁻⁴
√(A₁²-A₂²) = 0.03042

√(2gh) = √(2×9.8×4.082) = √79.99 ≈ 8.944

Q = 0.98 × (2.468×10⁻⁴/0.03042) × 8.944
= 0.98 × 8.113×10⁻³ × 8.944
= 0.98 × 0.07256
≈ 0.0711 m³/s
```

**Answer: Q ≈ 0.071 m³/s**

</details>

## 📐 Fluid Mechanics

| Formula | Name | Variables |
|---------|------|-----------|
| `A₁v₁ = A₂v₂` | Continuity Equation | A = area, v = velocity |
| `P + ½ρv² + ρgh = const` | Bernoulli's Theorem | P = pressure, ρ = density |
| `Q = πPR⁴/(8ηL)` | Poiseuille's Equation | Q = flow rate, R = radius |
| `v = √(2gh)` | Torricelli's Theorem | h = depth below surface |
| `Re = ρvD/η` | Reynolds Number | D = diameter, η = viscosity |
| `Re < 2000` | Laminar flow criterion | — |
| `v_c = Kη/(ρD)` | Critical Velocity | K ≈ 2000 |

## 📐 Viscosity

| Formula | Name | Variables |
|---------|------|-----------|
| `τ = η(dv/dy)` | Newton's Viscosity Law | τ = shear stress |
| `η = τ/(dv/dy)` | Coefficient of Viscosity | units: Pa·s |
| `F = 6πηrv` | Stokes' Law | r = sphere radius |
| `v_t = 2r²(ρ-σ)g/(9η)` | Terminal Velocity | ρ = sphere, σ = fluid density |
