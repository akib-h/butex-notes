# 7. Defects of Jute Fibre

> **Date:** 2026-02-26  
> **Course:** YE-101 — Natural Textile Fibres  
> **References:** Toufiqua Siddiqua (BUTEX), Fardin AE-50, Engr. Ahshanul Hoq (SARSTEC)

---

## 7.1 Overview

Defects in jute fibre arise primarily from:

- Improper retting (under-retting or over-retting)
- Careless stripping, washing, and drying
- Insect damage / disease in the growing plant
- Poor storage conditions
- Immature or over-mature plants at harvest

---

## 7.2 Defects — Detailed Description

### 1. Rooty Fibres

| | |
|---|---|
| **Description** | Lower parts of the jute fibres from which **barks and gums have NOT been completely removed**. Root ends are cut off and used for low grade yarn. |
| **Causes** | Improper decomposition (retting) of jute; under-retting of the root ends; plants allowed to stay longer after water receded from field; root ends not completely covered for rotting; incorrect rolling of root ends |
| **Effect on processing** | Hard root ends — difficult to process |
| **Remedy** | Cut off root ends; use for low grade yarn |

### 2. Specky / Spotted Fibres

| | |
|---|---|
| **Description** | Uneven retting or insufficient washing causes the **outer bark to adhere in places** on the fibres, showing specks (spots). |
| **Causes** | Jute fibres not retted and washed properly; bark adheres to the fibres — then comes "speck"; also called "Specky jute" |
| **Effect** | Poor appearance; requires lying longer in batch and increased carding for use |
| **Remedy** | Proper retting and thorough washing; increased carding |

### 3. Croppy Fibres

| | |
|---|---|
| **Description** | The **top end of the fibres are rough, black, and hard** due to careless steeping/retting. The fibres at the top of the plant become sticky and rough. |
| **Causes** | Plants don't immerse properly in water while retting (top ends not covered); careless steeping — top ends not properly submerged |
| **Effect** | Hard wiry top end; unsuitable for spinning |
| **Remedy** | Should be cut off and used for low grade yarn |

### 4. Knotty Fibres

| | |
|---|---|
| **Description** | Having **knots in the fibres** or fibres sticky together. |
| **Causes** | Punctures in the growing plants from insect bites; insects cause difficulty in fibre collection |
| **Effect** | Difficult to separate fibres from the stem; resist separation |
| **Remedy** | Should be cut off and used for low grade yarn |

### 5. Weak Fibres

| | |
|---|---|
| **Description** | Fibres with **reduced tensile strength** — dull, poor in strength. |
| **Causes** | Over-retting; bundling in wet condition; storing for a long time reduces fibre strength |
| **Effect** | Cannot be used for standard spinning |
| **Remedy** | Should be mixed up with better fibres for spinning |

### 6. Sticky Fibre

| | |
|---|---|
| **Description** | Fibre collected with **sticks of the plant** attached. |
| **Causes** | Fibre collected from **immature plants** — the stick of the plant comes with the fibre during collection |
| **Effect** | Requires additional cleaning/carding |
| **Remedy** | Harvest at proper maturity stage |

### 7. Dead / Dazed Fibres

| | |
|---|---|
| **Description** | The jute fibre has **lost its lustre, strength, and colour** — dull, poor in strength, inferior for spinning. |
| **Causes** | Over-retting; storing jute fibres in moist condition for a long time; over ageing |
| **Effect** | Makes them dull, poor in strength, inferior for spinning |
| **Remedy** | Should be mixed with other fibres and used for low grade yarn |

### 8. Hunka Fibre

| | |
|---|---|
| **Description** | Defect caused by **non-removal of dried up base and hard bark** from the fibres. |
| **Causes** | Hard and strong cuticle or outer layer of the plant is not removed during stripping |
| **Effect** | Hard, harsh segments in the fibre |
| **Remedy** | Should be cut off and used for low grade yarn |

### 9. Runners

| | |
|---|---|
| **Description** | **Long and hard barky ribbon of fibres** — long hard bark strips running along the fibre. |
| **Causes** | Uneven steeping and careless washing during retting process |
| **Effect** | Resist separation; makes yarn uneven |
| **Remedy** | Should be cut off and used for low grade yarn |

### 10. Mossy Fibre

| | |
|---|---|
| **Description** | Fibres covered with **mosses** that adhere to the jute plant. |
| **Causes** | Jute plants grow in standing water; mosses grow side by side and adhere to jute plants, causing them to become mossy |
| **Effect** | Contamination; poor appearance |
| **Remedy** | Mossy portion should be cut off and used for low grade yarn |

### 11. Heart Damage

| | |
|---|---|
| **Description** | The centre of the bale becomes **badly tendered** and in some cases fibres are reduced to powder. |
| **Causes** | Jute fibre contains **excess moisture when baled** — causes internal heat damage |
| **Effect** | Fibres reduced to powder; complete degradation |
| **Remedy** | Fibre should be sorted out; infected fibres discarded |

### 12. Sticky, Woody and Shive

| | |
|---|---|
| **Description** | Fibres contain **broken pieces of jute sticks**, etc. |
| **Causes** | Short plants that cannot be properly stripped and cleaned |
| **Effect** | These fibres require increased carding for use |

## 7.3 Summary Table

| # | Defect Name | Main Cause | Location | Remedy |
|---|-------------|-----------|----------|--------|
| 1 | Rooty Fibres | Under-retting at roots | Root end | Cut off; low grade |
| 2 | Specky/Spotted | Uneven retting + poor washing | Whole fibre | Better retting; carding |
| 3 | Croppy Fibres | Careless steeping (top not submerged) | Top end | Cut off; low grade |
| 4 | Knotty Fibres | Insect bites / punctures | Any location | Cut off; low grade |
| 5 | Weak Fibres | Over-retting or wet storage | Whole fibre | Mix with better fibres |
| 6 | Sticky Fibre | Immature plant; stick with fibre | Whole fibre | Harvest at maturity |
| 7 | Dead/Dazed | Over-retting or moist storage | Whole fibre | Mix; low grade |
| 8 | Hunka | Hard cuticle not removed | Base area | Cut off; low grade |
| 9 | Runners | Uneven steeping + careless washing | Whole fibre | Cut off; low grade |

| 11 | Heart Damage | Excess moisture when baled | Centre of bale | Discard infected fibres |
| 12 | Sticky/Woody | Short plants; improper stripping | Whole fibre | Increased carding |

## 7.4 How to Prevent Defects — Best Practices

| Stage | Best Practice |
|-------|---------------|
| **Retting** | Ensure all parts (especially top and root ends) are fully submerged; monitor retting completion |
| **Stripping** | Strip promptly when retting is complete; avoid delay |
| **Washing** | Wash thoroughly to remove all bark and gum residues |
| **Drying** | Dry completely before storage; use bamboo railings, not bare ground |
| **Storage** | Store in dry conditions; avoid excess moisture in bales |
| **Harvesting** | Harvest at proper maturity (120–150 days); avoid over-mature or immature plants |
| **Pest Control** | Use appropriate pesticides to prevent insect damage |
