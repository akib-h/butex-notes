## Quick Reference — Key Facts

| Property | Value |
|----------|-------|
| Scientific name | *Corchorus olitorius* (Tossa), *Corchorus capsularis* (White) |
| Fibre type | Natural, Bast, Ligno-cellulosic, Composite |
| Nickname | Golden Fibre of Bangladesh |
| Cellulose content | 65.2% |
| Lignin content | 10.8% |
| Hemicellulose | 22.2% |
| Moisture Regain | 13.75% |
| Tenacity (dry) | 3–5 gm/denier |
| Elongation at break | 1.7–1.8% |
| Specific gravity | 1.48–1.5 |
| Resiliency | Bad |
| Dye affinity | Basic dyes |
| Fibre length | 5–12 feet |
| Ultimate cell length | 0.8–6 mm (avg 2.5 mm) |
