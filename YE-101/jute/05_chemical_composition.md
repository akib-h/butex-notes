## 5.1 Overview

Jute is a **cellulosic fibre**. Its repeating unit is **cellubiose**. Jute is primarily composed of three major polymeric components:

| Component | Percentage |
|-----------|-----------|
| **Cellulose** | 65.2% |
| **Hemicelluloses** | 22.2% |
| **Lignin** | 10.8% |
| Water soluble | 1.5% |
| Fat and wax | 0.3% |

> **Note from class notes (Fardin AE-50):** Some sources give the range as Cellulose 55–65%, Hemicellulose 14–20%, Lignin 12–13%, Moisture 13%, Fat/Wax 0.5%.

## 5.2 Cellulose

### 5.2.1 Definition and Structure

Cellulose is the main structural component of the jute fibre cell wall. It is an **inert carbohydrate** which is the main constituent of plant cell walls.

- **Building unit:** The simple sugar **β-D-Glucose** (C₆H₁₂O₆)
- **Polymer type:** Long-chain, linear (unbranched) polymer of β-1,4-linked glucose units
- **Repeating unit:** **Cellobiose** (a disaccharide of two glucose units)

### 5.2.2 Chemical Structure of Cellulose

```
Chemical Formula: (C₆H₁₀O₅)ₙ

Molecular structure (cellobiose repeating unit):

CH₂OH          CH₂OH
  |               |
  C       O       C
 / \     / \     / \
H   C   C   C   C   C—OH
    |   |   |   |
   OH   H   OH  H
    \   |   /
     O—C—C
         |
        OH

β-1,4-glucosidic linkage between glucose units
```

- **Degree of Polymerization (DP) of jute cellulose:** ~**500–2000**
- The chain has many identical glucose units strung **head-to-tail** (β-1,4 linkage)
- Long, linear chains → high crystallinity → strength

### 5.2.3 Properties of Cellulose in Jute

| Property | Value/Description |
|----------|-----------------|
| DP | 500–2000 |
| Nature | Crystalline and strong polymer |
| Structure | Long chain (7,000–15,000 sugar units in general cellulose) |
| Hydrolysis | Resistant to hydrolysis (unlike hemicellulose) |
| Polymer type | Unbranched |
| Synthesis | By rosette terminal complexes (RTCs) at the plasma membrane |
| Digestibility | Only digestible by ruminants |

## 5.3 Hemicellulose

### 5.3.1 Definition

Hemicellulose (polyose) is one of a number of **heteropolymers (matrix polysaccharides)** present along with cellulose in all plant cell walls. It has a simpler structure than cellulose.

### 5.3.2 Chemical Structure of Hemicellulose

```
Chemical Formula: Variable (branched heteropolymer)

Main sugar units:
- Xylose (most common in jute)
- Galactose
- Mannose
- Rhamnose
- Arabinose

Structure: Short backbone chain with side branches (acidic)
DP = 50–200
```

### 5.3.3 Key Differences: Cellulose vs Hemicellulose

| Property | Cellulose | Hemicellulose |
|----------|-----------|---------------|
| **Monomer** | β-D-Glucose only | Xylose, galactose, mannose, etc. |
| **Chain type** | Unbranched (linear) | **Branched** |
| **DP** | 500–2000 (jute); 7,000–15,000 (wood) | **50–200** |
| **Structure** | Crystalline, strong | Amorphous, little strength |
| **Hydrolysis** | Resistant | Easily hydrolyzed by dilute acid or base |
| **Chain length** | Long (7,000–15,000 sugar units general) | Short (500–3,000 sugar units) |
| **Synthesis** | Rosette terminal complexes (RTCs) | Sugar nucleotides in Golgi apparatus |
| **Digestibility** | Only ruminants | Humans and ruminants |
| **Uses** | Paper, textiles, pharmaceuticals | Paper, furfural, ethanol |
| **Role in cell wall** | Main structural component | Cross-linker between microfibrils |

### 5.3.4 Role of Hemicellulose in Jute

1. Hemicellulose molecular chains are **very much shorter** than cellulose chains
2. Hemicellulose molecule has **short side chains sticking out at intervals** along its length (branched)
3. These side chains are **acidic in nature** — this gives jute its **slightly acidic reaction** and its **affinity for basic dyestuffs**
4. **Microfibrils are cross-linked together** by hemicellulose homopolymers
5. Lignin assists and strengthens the attachment of hemicelluloses to microfibrils
6. Two main types in jute: **Xylans** and **Glucomannans**

## 5.4 Lignin

### 5.4.1 Definition and Structure

Lignin is an **aromatic (phenolic) polymer** — it is the cementing element, which attaches ultimate fibres together and makes a composite long fibre.

```
Basic phenylpropane unit of Lignin:

         OH
          |
    [Benzene ring]—C₃H₇
          |
         OH

Contains: Methoxy groups (–OCH₃) and Phenolic Hydroxyl groups (–OH)
```

### 5.4.2 Properties and Role of Lignin in Jute

| Property/Role | Description |
|---------------|-------------|
| **Content in jute** | ~10.8% (some sources: 12–13%) |
| **Chemical nature** | Aromatic (phenolic) polymer; contains methoxy and phenolic hydroxyl groups |
| **Cementing function** | Attaches ultimate fibres together → composite long fibre |
| **Effect on strength** | **Increases** strength, hardness, and stiffness |
| **Effect on flexibility** | **Reduces** flexibility and extensibility of the fibre |
| **Bacterial resistance** | Provides good resistance to bacterial degradation |
| **Colour** | Responsible for the **wide colour range** and **yellowing of jute on exposure to sunlight** |
| **Location** | Most concentrated in the **middle lamella** |
| **Chemical structure** | Its exact nature has not yet been fully established despite >100 years of study |

### 5.4.3 Why Lignin Causes Problems

1. **Colouring/yellowing:** Lignin undergoes photo-oxidation when exposed to light (UV), leading to yellowing/discolouration
2. **Cannot bleach to white:** Because of lignin content, jute is difficult to bleach and can **never be made pure white**
3. **Reduces dyeability:** Limits penetration of certain dyes
4. **Brittleness:** High lignin content makes jute brittle

> **Jute is never available in pure white colour:**  
> Because of its lignin content and lack of strength, jute is difficult to bleach and can never be made pure white. It can be converted into a **wool-like fibre** by treatment with **strong caustic soda (NaOH)**.

## 5.5 Chemical Composition — Mathematical Analysis

### 5.5.1 Cellulose Content Verification

Given composition:
- Cellulose: 65.2%
- Hemicelluloses: 22.2%
- Lignin: 10.8%
- Water soluble: 1.5%
- Fat and wax: 0.3%

**Check (Total):**
```
65.2 + 22.2 + 10.8 + 1.5 + 0.3 = 100.0%  ✓
```

### 5.5.2 Ratio Analysis

**Cellulose to Non-Cellulose ratio:**
```
Cellulosic content = 65.2 + 22.2 = 87.4%
Non-cellulosic (lignin + water sol. + fat) = 10.8 + 1.5 + 0.3 = 12.6%

Ratio = 87.4 : 12.6 ≈ 6.94 : 1
```

**Lignin to Hemicellulose ratio:**
```
Lignin : Hemicellulose = 10.8 : 22.2 ≈ 1 : 2.06
```

This ratio explains why hemicellulose has more influence on hydrophilicity and dyeability than lignin.

### 5.5.3 Degree of Polymerization (DP) — Formula

The DP of a polymer relates to its molecular weight:

```
Molecular Weight (Mw) = DP × Mw of repeating unit

For cellulose, repeating unit (cellobiose) = C₁₂H₂₀O₁₀
Mw of cellobiose = 12(12) + 20(1) + 10(16) = 144 + 20 + 160 = 324 g/mol

If DP = 500:  Mw = 500 × 162 = 81,000 g/mol
              (using glucose monomer Mw = 162 for anhydrous unit)

If DP = 2000: Mw = 2000 × 162 = 324,000 g/mol
```

Higher DP → longer chains → stronger intermolecular bonding → higher tensile strength.

## 5.6 Effect of Composition on Properties

| Composition Factor | Effect on Jute |
|-------------------|----------------|
| High cellulose (65.2%) | Gives tensile strength, moisture absorption |
| High hemicellulose (22.2%) | Contributes to moisture regain, gives affinity for basic dyes |
| High lignin (10.8%) | Makes fibre stiff, brittle, hard; responsible for yellowing |
| Low fat/wax (0.3%) | Less lubrication → rough feel |
| Acidic hemicellulose side chains | Affinity for **basic (cationic) dyes** |
