# Ramie Fibre  
**Date:** April 5, 2026  

## 1. Botanical Source
- Extracted from *Boehmeria nivea* (China grass).

## 2. Properties
1. High cellulose content (~70%).  
2. Excellent tensile strength.  
3. Lustrous, silky appearance.  

![Ramie fibre](https://upload.wikimedia.org/wikipedia/commons/3/3a/Ramie_fibre.jpg)

## 3. Applications
- Apparel, upholstery, industrial fabrics.  
- Blended with cotton for wrinkle-resistant fabrics.  

## 4. Example
- **Ramie-cotton blends** used in summer clothing for breathability.
