# Pineapple Fibre (PALF)  
**Date:** April 5, 2026  

## 1. Source
- Extracted from pineapple leaves.

## 2. Properties
1. High tensile strength.  
2. Lightweight and biodegradable.  
3. Glossy, silk-like texture.  

![Pineapple fibre](https://upload.wikimedia.org/wikipedia/commons/7/7f/Pineapple_leaf_fibre.jpg)

## 3. Applications
- Biocomposites, eco-textiles.  
- PALF reinforced polyester composites.  

## 4. Example
- **Piñatex** – pineapple fibre leather alternative.
