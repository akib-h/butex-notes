## 📖 Overview
This repository contains a detailed study of **Mineral and Plant-based Fibres** for textile engineering students. Each fibre type is documented in a separate Markdown file, following the syllabus requirements. The content is designed for **university-level lessons**, enriched with examples, diagrams, and references to authoritative sources.

The repo is intended to:
- Provide **structured, syllabus-aligned notes** for textile engineering courses.  
- Serve as a **knowledge base** for fibres used in textiles and composites.  
- Offer **examples and case studies** to connect theory with practice.  
- Include **visual references** (images, diagrams) for better understanding.  
- Maintain **academic integrity** with citations and references.

## 📌 Topics Covered

1. **Mineral Fibre**
   - Asbestos Fibre: definition, types, properties, health hazards, industrial uses.  

2. **Plant-Based Fibres**
   - Hemp Fibre: eco-friendly textile applications.  
   - Ramie Fibre: high cellulose content, blends with cotton.  
   - Banana Fibre: extraction from pseudostem, silk-like properties.  
   - Pineapple Fibre (PALF): biocomposites and eco-textiles.  
   - Kapok Fibre: buoyant, antibacterial, used in pillows and insulation.  
   - Sisal Fibre: ropes, mats, durable composites.  
   - Coir Fibre: coconut husk fibre, geotextiles, mats, brushes.
