## 🔑 Key Facts Summary

| Property | Silk |
|----------|------|
| **Source** | Silkworm (*Bombyx mori*) |
| **Protein** | Fibroin (75–80%) + Sericin (20–25%) |
| **Structure** | Antiparallel β-pleated sheet (Silk II) |
| **Key amino acids** | Gly (43%), Ala (30%), Ser (12%) |
| **Sulfur** | Absent |
| **Cross-section** | Triangular (rounded corners) |
| **Longitudinal** | Smooth, structureless rod (degummed) |
| **Filament length** | 600–1,500 m per cocoon |
| **Diameter** | 10–14 μm (brin) |
| **Density** | 1.30–1.38 g/cm³ |
| **Moisture regain** | 11% |
| **Tenacity** | 2.4–5.1 g/denier |
| **Thermal stability** | Tg ~175°C |
| **Top producers** | China (63%), India (18%), Uzbekistan (4%) |

## 💡 Study Tips

1. **For microscopy questions:** Remember the key rule — *Smooth + Triangular = Mulberry Silk*; *Scaly + Circular = Wool/Hair*.
2. **For chemistry questions:** Focus on the absence of sulfur in silk (no disulfide bonds in main chain) vs wool (many disulfide bonds) — this difference explains elasticity, crimp, and chemical resistance differences.
3. **For structure questions:** Silk = **β-pleated sheet**; Wool = **α-helix**. This is the single most important structural distinction.
4. **For production questions:** Know all 9 steps from egg to degummed silk — each step (stifling, reeling, throwing, degumming) has a specific purpose and method.
5. **For property questions:** The triangular cross-section explains lustre; the hydrogen bonds between β-sheets explain tensile strength; the amorphous regions explain moisture absorption and dye uptake.
