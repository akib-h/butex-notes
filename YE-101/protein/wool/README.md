## 🔑 Quick Reference — Most Important Facts

### Core Identity
```
Wool = Natural + Animal + Protein (Keratin) + Staple + Crimped fibre
Polymer = α-Keratin (polypeptide made of α-amino acids linked by peptide bonds)
Repeating unit = α-amino acid (20 types)
DP = Unknown
```

### Key Dimensions
```
Length:    5–12 cm (up to 35 cm coarse)
Diameter:  14 µm (fine) → 45 µm (coarse)
Crimp:     100/10cm (fine) → 4/10cm (coarse)
Sp. Gr.:   1.32
MR%:       16%
Tenacity:  8.8–15 cN/tex (dry)
Elong.:    25–35% (dry)
Crystal.:  25–30%
```

### Key Bonds (Strongest → Weakest)
```
1. Peptide bond (–CO–NH–)      [Covalent, backbone]
2. Cystine/Disulphide (–S–S–)  [Covalent, cross-link — UNIQUE to wool]
3. Salt/Ionic linkage           [Electrostatic]
4. Hydrogen bond               [H-bond]
5. Van der Waals forces        [Weakest]
```

### Manufacturing Steps
```
Shearing → Skirting → Sorting/Grading → Scouring → Oiling → 
Carbonizing → Drying → Bailing → Garnetting → Carding → Spinning
```

### Yarn Count Formulas
```
Ne (English)   = L(yards) / (840 × w(lbs))
Woollen count  = L(yards) / (256 × w(lbs))
Worsted count  = L(yards) / (560 × w(lbs))
```

### Identification
```
Burning:    Smell of burning hair; self-extinguishing
Chemical:   Dissolves in NaOH, NaOCl; slowly in 70% HNO₃
Microscope: Oval cross-section, serrated/scaly longitudinal surface
```
