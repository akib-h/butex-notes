# 🧪 Acids

## **Organic Acids (contain carbon, often carboxylic acids)**
- Citric Acid → C6H8O7 (found in lemon juice)  
- Oxalic Acid → HOOC–COOH (in spinach, rhubarb)  
- Tartaric Acid → C4H6O6 (in tamarind, grapes)  
- Lactic Acid → CH3–CH(OH)–COOH (in sour milk, muscles)  
- Acetic Acid → CH3COOH (vinegar)  
- Formic Acid → HCOOH (ant sting)  
- Pyruvic Acid → CH3–CO–COOH  
- Carbolic Acid (Phenol) → C6H6O  
- Oleic Acid → C17H33COOH (unsaturated fatty acid)  
- Stearic Acid → C17H35COOH (saturated fatty acid)  
- Palmitic Acid → C15H31COOH (fatty acid)  
- Gluconic Acid → C6H12O7  

---

## **Inorganic Acids (mineral acids, no carbon backbone)**
- Hydrochloric Acid → HCl  
- Sulfuric Acid → H2SO4  
- Sulfurous Acid → H2SO3  
- Nitric Acid → HNO3  
- Nitrous Acid → HNO2  
- Phosphoric Acid → H3PO4  
- Phosphorous Acid → H3PO3  
- Carbonic Acid → H2CO3  
- Boric Acid → H3BO3  
- Silicic Acid → H2SiO3  
- Chloric Acid → HClO3  
- Perchloric Acid → HClO4  
- Hypochlorous Acid → HClO  
- Hydrobromic Acid → HBr  
- Hydroiodic Acid → HI  
- Permanganic Acid → HMnO4  
- Pyrosulfuric Acid → H2S2O7  
- Pyroboric Acid → H2B4O7  
- Cyanic Acid → HCNO  
- Thiocyanic Acid → HCNS  
- Thiosulfuric Acid → H2S2O3  
- Hydrazoic Acid → HN3 (special, unstable acid)  

---

# ⚗️ Bases (Alkalis)
- Sodium Oxide → Na2O  
- Sodium Hydroxide → NaOH  
- Ammonium Hydroxide → NH4OH  
- Calcium Oxide → CaO  
- Potassium Hydroxide → KOH  
- Calcium Hydroxide → Ca(OH)2  

---

# 🧂 Salts
- Sodium Chloride (table salt) → NaCl  
- Ammonium Chloride (sal ammoniac) → NH4Cl  
- Potassium Chloride → KCl  
- Calcium Carbonate (limestone) → CaCO3  
- Sodium Sulfate → Na2SO4  
- Potassium Ferrocyanide → K4[Fe(CN)6]  
- Ammonium Phosphate → (NH4)3PO4  
- Gypsum → CaSO4·2H2O  
- Glauber’s Salt → Na2SO4·10H2O  
- Zinc Sulfate → ZnSO4  
- Gold Chloride → AuCl3  

---

✅ **Key Tip for Exams:**  
- **Organic acids** are usually weak acids found in plants/animals (vinegar, lemon, milk).  
- **Inorganic acids** are strong mineral acids (HCl, H2SO4, HNO3).  
- Bases are mostly hydroxides/oxides of metals.  
- Salts are neutralization products of acids + bases.
